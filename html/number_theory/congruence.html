congruence
Fix an integer `m` called the modulus, then say `a -= b (mod m)` if and only if `m | b - a`
congruence iff same remainder
`a, b in ZZ` are congruent modulo `m` if and only if `a % m` ` = b % m`

note: this means that `a` and `b` have the same remainder upon division by `m`

Suppose that `a -= b (mod m)`, so we know `m | b - a`, then we can write `b = k_b m + r_b` and `a = k_a m + r_a` so that `b - a = m *(k_b + k_a) + k_b - k_a ` thus `m | k_b - k_a`.

Notice that `0 le k_a, k_b lt abs(m)` also we know that `k_a ne k_b` therefore `k_b - k_a ne 0` so then `-abs(m) lt k_b - k_a lt abs(m)`

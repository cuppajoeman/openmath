function
A function from `X` to `Y` denoted by `f: X to Y` is a binary relation`R` on `X xx Y` such that if `(x, y), (x, z) in R`, then `z = y`, which says given an element `x in X` it is only ever related to a single `y in Y`.
dom f
Given a function `f : X to Y`, then we denote `text(dom)(f) := X` which we is shorthand for the word domain
ran f
Given a function `f : X to Y`, then we denote `text(ran)(f) := Y` which we is shorthand for the word range
image of a set
Let `f: X to Y` be a function and `S sube X`, then `f(S) := {y in Y: y = f(s), text( for some ) s in S}`
inverse image of a function
Suppose `f: X to Y` and `U sube Y`, then `f^(-1)(U) := {x in X: x in U}`
injective function
a function `f: X to Y` is injective iff `AA x, y in X, f(x) = f(y) implies x = y`
surjective function
a function `f: X to Y` is surjective iff `AA y in Y, EE x in X text( such that ) f(x) = y`
bijective function
a function is bijective iff it is surjective and injective
function composition
Suppose that `A, B, C` are sets, and that `f: A to B, g: B to C` are functions, then we define `g circ f (x) := g(f(x))` for any `x in A`, so that `g circ f: A to C`
composition of two bijective functions is bijective
Suppose `A, B, C` are non-empty sets and `f: A to B, g: B to C` are functions, and that `g circ f : A to C` is a function. Then if `f` and `g` are bijective then `g circ f` is also bijective.
show

We'll show that `g circ f` is surjective first, so let `c in C`, since `g` is bijective we get some `b in B` such that `g (b) = c`, since `b in B`, then since `f` is surjective we get some `a in A` such that `f(a) = b`, thus `g circ f (a) = g(f(a)) = g(b) = c` so then `g circ f` is surjective

Now let's show that `g circ f` is injective, so suppose that `a_1, a_2 in A` and assume that `g circ f (a_1) = g @ f (a_2)`, note that this means `g(f(a_1)) = g(f(a_2))`, therefore since `g` is injective we know that `f(a_1) = f(a_2)` and since `f` is injective then we know that `a_1 = a_2` as needed.

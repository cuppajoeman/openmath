Category
A category \(\mathscr{C}\) consists of
  • a collection ob \((\mathscr{C})\) of objects
  • for each \(A, B \in \mathrm{ob}(\mathscr{C})\), a collection \( \operatorname{ mor } (A, B) \) of what we call morphisms / arrows from \(A\) to \(B\)
  • for each \( A, B, C \in \operatorname{ob}(\mathscr{C})\), a function which we call composition \( \circ _ \mathscr{C} : \operatorname{ mor } (B, C) \times \operatorname{ mor } (A, B) \rightarrow \operatorname{ mor } (A, C) \)
    • \[ \circ _ \mathscr{C} \left( g, f \right) : = g \circ f \]
  • for each \(A \in \mathrm{ob}(\mathscr{C})\), an element \(1_A\) of \( \operatorname{ mor } (A, A)\), called the identity on \(A\), satisfying the following axioms:
    • associativity: for each \(f \in \operatorname{ mor } (A, B), g \in \operatorname{ mor } (B, C)\) and \(h \in \operatorname{ mor } (C, D)\), we have \((h \circ g) \circ f=h \circ(g \circ f)\);
    • identity laws: for each \(f \in \operatorname{ mor } (A, B)\), we have \(f \circ 1_A = f = 1_B \circ f\).
Domain and Codomain
Each morphism has associated domain and codomain, denoted \[f \,:\, A \to B\]
  • The domain of \(f\), denoted \(\operatorname{dom}(f)\), is \(A\)
  • The codomain of \(f\), denoted \(\operatorname{cod}(f)\), is \(B\)
Some Notations on Morphism
  • \(f, g \,:\, A \rightrightarrows B \) denotes two morphisms with the same domain and codomain
  • \(f \,:\, A \rightleftarrows B \,:\, g\) denotes two morphisms, \(f \,:\, A \to B\) and \(g \,:\, B \to A\)
Isomorphism / Isomorphic
Let \(\mathcal{C}\) be a category, \(A, B \in \operatorname{ob}(\mathcal{C})\). \(f \,:\, A \to B\) is an isomorphism if and only if \[ \begin{align*} \exists \, g \,:\, B \to A \,.\, & g \circ f = 1_A \land\\ & f \circ g = 1_B \end{align*} \] In which case \(A, B\) are isomorphic and denoted as \(A \cong B\)
Endomorphism
Let \(\mathcal{C}\) be a category, \(A \in \operatorname{ob}(\mathcal{C})\). \(f \,:\, A \to A\) is an endomorphism if and only if \(f\) has the same domain and codomain
Automorphism
A morphism \(f\) is an automorphism if and only if \[ f \text{ is an endomorphism} \land f \text{ is an isomorphism} \]
Opposite Category, Duality
Let \(\mathcal{C}\) be a category. Then the opposite category \(\mathcal{C}^\mathrm{op}\) is defined as follow:
  • \(\operatorname{ob}(\mathcal{C}^{\mathrm{op}}) = \operatorname{ob}(\mathcal{C}) \)
  • For each \(f \in \operatorname{mor}_{\mathcal{C}}(B, A) \), a morphism \(f^{\mathrm{op}} \in \operatorname{mor}_{\mathcal{C}^{\mathrm{op}}}(A, B)\)
  • \(g^{\mathrm{op}} \circ_{\mathcal{C}^{\mathrm{op}}} f^{\mathrm{op}} = (f \circ_{\mathcal{C}} g)^\mathrm{op}\)
The opposite category is a category
The construction \(\mathcal{C}^{\mathrm{op}}\) indeed form a new category.

The definition already gives us the mathematical objects of a category. We only need to show composition satisfies associativity and identity law.

We first show associativity. Let \(f^\mathrm{op} \,:\, A \to B, g^\mathrm{op} \,:\, B \to C, h^\mathrm{op} \,:\, C \to D\) be morphisms in \(\mathcal{C}^\mathrm{op}\). Correspondingly, we have morphisms \( f \,:\, B \to A, g \,:\, C \to B, h \,:\, D \to C\) in \(\mathcal{C}\). We then have the following reasoning (composition subscript omitted): \[ \begin{align*} & (h^\mathrm{op} \circ g^\mathrm{op}) \circ f^\mathrm{op} \\ =& (g \circ h)^\mathrm{op} \circ f^\mathrm{op} \\ =& (f \circ (g \circ h))^\mathrm{op} \\ =& ((f \circ g) \circ h)^\mathrm{op} \\ =& h^\mathrm{op} \circ (f \circ g)^\mathrm{op} \\ =& h^\mathrm{op} \circ (g^\mathrm{op} \circ f^\mathrm{op}) \end{align*} \]

Lastly we show the existence of identity morphism along with their desired property. Let \(A,B \in \mathcal{C}^\mathrm{op}\), we choose the identity morphism of this object to be \(1_A^\mathrm{op}, 1_B^\mathrm{op}\) respectively.

Let \(f^\mathrm{op} \,:\, A \to B \) be a morphism in \(\mathcal{C}\). We can show the following: \[ \begin{align*} f^\mathrm{op} \circ 1_A^\mathrm{op} = & (1_A \circ f)^\mathrm{op} \\ = & (f)^\mathrm{op} \\ 1_B^\mathrm{op} \circ f^\mathrm{op} = & (f \circ 1_B)^\mathrm{op} \\ = & (f)^\mathrm{op} \end{align*} \]

Thus all desired properties are shown, we conclude \(\mathcal{C}^\mathrm{op}\) is a category.

Functor
Let \(\mathscr{A}\) and \(\mathscr{B}\) be categories. A functor \(F: \mathscr{A} \rightarrow \mathscr{B}\) consists of
  • a function \( \mathrm{ob}(\mathscr{A}) \rightarrow \mathrm{ob}(\mathscr{B}) \), written as \(A \mapsto F(A)\)
  • - for each \(A, A^{\prime} \in \mathscr{A}\), a function \[ \operatorname{ mor } _ \mathscr{A}\left(A, A^{\prime}\right) \rightarrow \operatorname{ mor } _ \mathscr{B}\left(F(A), F\left(A^{\prime}\right)\right) \] written as \(f \mapsto F(f)\), satisfying the following axioms:
    • \(F\left(f^{\prime} \circ f\right)=F\left(f^{\prime}\right) \circ F(f)\) whenever \(A \xrightarrow{f} A^{\prime} \xrightarrow{f^{\prime}} A^{\prime \prime}\) in \(\mathscr{A}\);
    • \(F\left(1_A\right)=1_{F(A)}\) whenever \(A \in \mathscr{A}\).
Pointed Topological Spaces Category
The category of pointed topological spaces denoted by \( \operatorname{ Top } _ 0 \) is one where objects are pairs \(\left(X, x_0\right)\), where \(X\) is a topological space, and \(x_0 \in X\) is a point of \(X\). Morphisms between \(\left(X, x_0\right)\) and \(\left(Y, y_0\right)\) are continuous functions \(f: X \rightarrow Y\) such that \(f\left(x_0\right)=y_0\).
Group Category
The category \( \operatorname{ Grp } \) has the class of all groups for objects and group homomorphisms for morphisms
The Fundamental Group Is a Functor
\( \pi _ 1 : \operatorname{ Top } _ 0 \to \operatorname{ Grp } \) can be completed to a functor

For each \( \left( X, x_0 \right) \in \operatorname{ ob } \left( \operatorname{ Top } _ 0 \right) \) we map it to \( \pi _ 1 \left( X , x _ 0 \right) \) which is the object functor. We call this function \( \pi _ 1 ^ \operatorname{ ob } : \operatorname{ ob } \left( \operatorname{ Top } _ 0 \right) \to \operatorname{ ob } \left( \operatorname{ Grp } \right) \)

Now we will construct our morphism functor, that is given \( f \in \operatorname{ hom } \left( \left( X, x _ 0 \right), \left( Y, y _ 0 \right) \right) \) we map it to a function , defined by \( \left[ \gamma \right] \mapsto \left[ f \circ \gamma \right] \) we call this mapping \( h \), that is: \( F _ { \left( X, x_0 \right), \left( Y, y _ 0 \right) } \) where \( F _ { \left( X, x_0 \right), \left( Y, y _ 0 \right) } \left( f \right) = h \)

We now show that \( h \in \operatorname{ hom } \left( \pi _ 1 \left( X , x _ 0 \right), \pi _ 1 \left( Y , y _ 0 \right) \right) \), to do this, we must prove that it is a group homomorphism, and also that it is well defined. For the well definedness we note that if \( \left[ \alpha \right] = \left[ \beta \right] \) then we have that \( \left[ f \circ \alpha \right] = \left[ f \circ \beta \right] \). For ease of notation recall that \( f _ * \left( \left[ \gamma \right] \right) : = \left[ f \circ \gamma \right] \), and we prove that our function \( h \) is a group homomorphism, that is we show that \[ f _ * \left( \left[ \alpha \right] \left[ \beta \right] \right) = f _ * \left( \left[ \alpha \right] \right) f _ * \left( \left[ \beta \right] \right) \] but on the left we have \[ \begin{align*} f _ * \left( \left[ \alpha \right] \cdot \left[ \beta \right] \right) &= f _ * \left( \left[ \alpha \cdot \beta \right] \right)\\ &= \left[ f \circ \left( \alpha \cdot \beta \right) \right] \\ &= \left[ \left( f \circ \alpha \right) \cdot \left( f \circ \beta \right) \right] \\ &= \left[ f \circ \alpha \right] \cdot \left[ f \circ \beta \right] \\ &= f _ * \left( \alpha \right) \cdot f _ * \left( \beta \right) \end{align*} \] which is the right, so we are done.

We now verify the functor properties: \[ F _ { \left( X, x _ 0 \right), \left( X, x _ 0 \right) } \left( \operatorname{ id } _ { \left( X, x _ 0 \right) } \right) = \operatorname{ id } _ { \pi _ 1 ^ { \operatorname{ ob } } \left( X, x_0 \right) } = \operatorname{ id } _ { \pi _ 1 } \left( X, x_0 \right) } \] We have that \( F _ { \left( X, x _ 0 \right), \left( X, x _ 0 \right) } \left( \operatorname{ id } _ { \left( X, x _ 0 \right) } \right) = h \) where \( h \left( \left[ \gamma \right] \right) = \left[ \operatorname{ id } _ { \left( X, x _ 0 \right) } \circ \gamma \right] = \left[ \gamma \right] \), in other words \( h \) takes an element from \( \pi _ 1 \left( X, x_0 \right) \) and returns that same element, so that \( h = \operatorname{ id } _ { \pi _ 1 \left( X, x _ 0 \right) } \) as needed.

We now verify that \[ F _ { \left( X , x _ 0 \right), \left( Y, y _ 0 \right) } \left( g \circ f \right) = F _ { \left( Y , y _ 0 \right), \left( Z, z _ 0 \right) } \left( g \right) \circ F _ { \left( X , x _ 0 \right), \left( Y, y _ 0 \right) } \left( f \right) \] If we focus on the right hand side and give the two functions simpler names ; \( h _ g : \pi _ 1 \left( Y, y _ 0 \right) \to \pi _ 1 \left( Z, z _ 0 \right) \) and \( h _ f : \pi _ 1 \left( X, x _ 0 \right) \to \pi _ 1 \left( Y, y _ 0 \right) \) respectively, and thus on the right hand side we have: \[ \begin{align*} \left( h _ g \circ h _ f \right) \left( \left[ \gamma \right] \right) &= h _ g \left( h _ f \left( \left[ \gamma \right] \right) \right) \\ &= h _ g \left( \left[ f \circ \gamma \right] \right) \\ &= \left[ g \circ \left( f \circ \gamma \right) \right] \\ &= \left[ \left( g \circ f \right) \circ \gamma \right] \\ &= F _ { \left( X , x _ 0 \right), \left( Y, y _ 0 \right) } \left( g \circ f \right) \end{align*} \] so our original equation holds true.

Thus since we've verified all the conditions, we conclude that \( \pi _ 1 \) is a functor.

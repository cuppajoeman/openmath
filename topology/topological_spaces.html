topology

topology on a set `X`

is a collection `cc T` of subset of `X` with the following properties

  1. `O/` and `X` are in `cc T`
  2. the union of the elements of any subcollection of `cc T` is in `cc T`
  3. the intersection of the elements of any finite subcollection of `cc T` is in `cc T`
The set `X` along with `cc T` satsifying the above conditions is called a topological space and is denoted by `(X, cc T)`
open set
Suppose `(X, cc T)` is a topological space, if `U in cc T` then we say that `U` is open with respect to `X`.
finer and coarser topologies

suppose that `cc T` and `cc T'` are two topologies on a given set `X`. If `cc T sube cc T'`, then `cc T'` is finer than `cc T`. If the reverse inclusion is true, then we say that `cc T'` is coarser than `cc T`, there are also strict variations of these definitions for the strict inclusions.

comparable topologies

given two topologies, they are comparable if at least one is finer than the other

basis

basis for a set

a basis for a set `X` is a collection `cc B` of subsets of `X` called basis elements such that

  1. for each `x in X`, there is at least one basis element `B` containing `x`
  2. if `x` belongs in the intersection of two basis elements `B_1` and `B_2`, then there is a basis element `B_3` containing `x` such that `B_3 sube B_1 nn B_2`
collection generated by a basis
Given a basis `cc B` we define the set `G_{cc B}` by `U in G_{cc B}` iff for each `x in U` there is a `B in cc B` such that `x in B` and `B sube U`
basis elements are open
In the above definition we can see that for any `B in cc B`, we can use `B` itself such that for each `x in B`, we have `B` so that `B sube B`.
`G_{cc B}` is a topology
The collection `G_{cc B}` generated by a basis is a topology
show

We will verify it's a topology from the definition

`O/` is in `G_{cc B}` because it vacuously satisfies the condition. `X` is in `G_{cc B}` since given any point `x in X`, we have a `B in cc B` such that `x in B sube X` by the definition of a basis

Now suppose we have an arbitrary union of elements from `G_{cc B}`, say `S sube G_{cc B}, M = bigcup_{U in S} U`. We need to verify that `M in G_{cc B}`, so let `x in M`, therefore `x in bar U` for some `bar U in S` and since `bar U` was assumed to be in `G_{cc B}` we know that there is some `B in cc B` such that `x in B sube bar U` but also `bar U sube M` since `M` includes `bar U` in the union so then `x in B sube M` meaning by the definition that `M in G_{cc B}`

Now if we are given a finite intersection of elements from `G_{cc B}` we will prove by induction that it is also an element of `G_{cc B}`

  • base case

    • if `n = 1` then the union of one element from `G_{cc B}` (namely just itself) is also an element of `G_{cc B}` trivially.

    Before moving to the inductive step, notice that it also holds for `n = 2`, because given `U_1, U_2 in G_{cc B}`, we can see that for any `x in U_1 nn U_2`, we know that we have a basis elements `B_1, B_2 in cc B` such that `x in B_1 sube U_1` and `x in B_2 sube U_2` and thus `x` also belongs to the intersection of `B_1` and `B_2`, so by the definition of a basis, we get a `B_3 sube B_1 nn B_2` that contains `x` since `B_1 nn B_2 sube U_1 nn U_2`, we have `x in B_3 sube U_1 nn U_2`, which means that `U_1 nn U_2 in G_{cc B}`

  • inductive step

    • Let `k in NN_1` and assume that the intersection of `k` elements from `G_{cc B}` is also in `G_{cc B}` now we'll show it's true for `k + 1`. So consider the intersection `U_1 nn ... nn U_(k+1)`, then we can re-write it as `(U_1 nn ... nn U_k) nn U_(k+1)` so by our inductive hypothesis `(U_1 nn ... nn U_k) in G_{cc B}` and therefore this `k+1` union is a union of two elements already in `G_{cc B}` so by our `n = 2` argument `U_1 nn ... nn U_{k+1} in G_{cc B}`, as needed.

topology generated by a basis
Since a collection generated by a basis turns out to be a topology we define `cc T_{cc B} := G_{cc B}` and call it the topology generated by `cc B`

theorem: `cc T_{cc B}` equals the collection of all unions of elements of `cc B`

theorem: basis criterion

Let `X` be a topological space. Suppose that `cc C` is a collection of open sets of `X` such that for each open set `U in X` and each `x in U`, there is an element C of `cc C` such that `x in C sube U`. Then `cc C` is a basis for the topology of `X`
basis finer equivalence
Let `cc B` and `cc B'` be bases for the topologies `cc T` and `cc T'`, respectively on `X`. Then the following are equivalent:
  1. `cc T'` is finer than `cc T`
  2. for each `x in X` and each basis element `B in cc B` containing `x` there is a basis element `B' in cc B'` such that `x in B' sube B`
topology equality with basis
Given two bases `cc B` and `cc B^'` for the toplogies `cc T` and `cc T^'` on `X`, then `cc T = cc T^'` if and only if all of the following hold:
  • for each `x in X` and each basis element `B in cc B` containing `x` there is a basis element `B' in cc B'` such that `x in B' sube B`
  • for each `x in X` and each basis element `B^' in cc B^'` containing `x` there is a basis element `B in cc B` such that `x in B sube B^'`
standard topology on `RR^n`
Let `cc B` be the collection of all sets of the form `{y in RR^n : norm(x - y) < epsilon}` for any `x in RR^n` and `epsilon in RR^(gt 0)`, then the topology generated by `cc B` is called the standard topology on the `RR^n`. Whenever we consider `RR^n`, we shall suppose it is given this topology unless we specifically state otherwise

subbasis
A subbasis S for a topology on X is a collection of subsets of X whose union equals X. The topology generated by the subbasis S is defined to be the collection T of all unions of finite intersections of elements of S
Let `X` be a topological space, and `A sube X` . Suppose that for each `x in A` there is an open set `U` containing `x` such that `U sube A`. Show that `A` is open in `X`.
show

For each `x in A` we have the set `U_x sube A`, then the union of elements the collection `cc C = {U_x : x in A}` in the collection is also a subset of `A` due to the union of subsets is still a subset. Thus we've just shown that `bigcup_{x in A} U_x sube A`

But we also know given an element `x in A` that `x in bigcup_{x in A} U_x` since each `U_x` contains `x` as well. Thus we have `A sube bigcup_{x in A} U_x` thus by equality by subsets we conclude that `A = bigcup_{x in A} U_x`

Since each `U_x` was assumed to be open with respect to `X`, then an arbitrary union of them is also open with respect to `X`, in other words `A` must be open.

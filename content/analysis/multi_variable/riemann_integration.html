Partition
A partition of \( \left[ a, b \right] \) is a finite tuple sorted in ascending order \[ P = \left( x _ 1, x _ 2, \ldots x _ { n + 1 } \right) \] such that \( a = x _ 1 \) and \( b = x _ { n + 1 } \)
Refinement of a Partition
A partition \( R \) is a refinement of a partition \( P \) if \( P \subseteq R \)
Common Refinement
Suppose that \( P, Q \) are partitions, then \( R \) is a common refinement of \( P \) and \( Q \) if \( R \) is a refinement of \( P \cup Q \)
i-th Section of a Partition
Suppose that \( P = \left( x _ 1, \ldots x _ { n + 1 } \right) \) is a partition, then for each \( i \in \left[ 1, \ldots , n \right] \) we define \[ s _ i := \left[ x _ i, x _ { i + 1 } \right] \]
Supremum of a Bounded Function on a Section
Suppose that \( f: \left[ a, b \right] \to \mathbb{ R } \) and that \( P = \left( x _ 1, \ldots , x _ { n + 1 } \right) \) is a partition of \( \left[ a, b \right] \) then for each \( i \in \left[ 1, \ldots , n \right] \) we have \[ M _ i \left( f, P \right) = \sup \left( \left\{ f \left( x \right): x \in \left[ x _ i, x _ { i + 1 } \right] \right\} \right) \]
i-th Delta of a Partition
Suppose that \( P = \left( x _ 1, \ldots , x _ { n + 1 } \right) \) is a partition, then for each \( i \in \left[ 1, \ldots n \right] \) we define: \[ \Delta _ i = x _ { i + 1 } - x _ { i } \]

Note that if a partition has \( k \) elements then there will be \( k - 1 \) deltas.

A Delta Sum Telescopes
Suppose that \( P = \left( x _ 1, \ldots , x _ { n + 1 } \right) \) is a partition of \( \left[ a, b \right] \) then \[ \sum _ { i = 1 } ^ { n } \Delta _ i = b - a \]
Recall that \[ \sum _ { i = 1 } ^ { n } \Delta _ i = \sum _ { i = 1 } ^ { n } \left( x _ { i + 1 } - x _ i \right) \stackrel{1}{=} x _ { n + 1 } - x _ 1 = b - a \] where we've used fact 1.
Mesh of a Partition
Suppose that \( P \) is a partition of \( \left[ a, b \right] \) such that \( \left\lvert P \right\rvert = n + 1 \) for some \( n \in \mathbb{ N } _ 0 \) then \[ \operatorname{ mesh } \left( P \right) = \max \left( \left\{ \Delta _ i: i \in \left[ 1, \ldots , n \right] \right\} \right) \]
Upper Sum of a Bounded Function over a Partition
Suppose that \( f: \left[ a, b \right] \to \mathbb{ R } \) is bounded, and that \( P \) is a partition of \( \left[ a, b \right] \) then we define \[ U \left( f, P \right) := \sum _ { j = 1 } ^ { n } M _ j \left( f, P \right) \Delta _ j \]
Lower Sum of a Bounded Function over a Partition
Suppose that \( f: \left[ a, b \right] \to \mathbb{ R } \) is bounded, and that \( P \) is a partition of \( \left[ a, b \right] \) then we define \[ L \left( f, P \right) := \sum _ { j = 1 } ^ { n } m _ j \left( f, P \right) \Delta _ j \]
Upper Sum Decreases over Refinements
Suppose that \( P \supseteq R \) then \[ U \left( f, P\right) \ge U \left( f, R \right) \]
Lower Sum Decreases over Refinements
Suppose that \( P \supseteq R \) then \[ L \left( f, P\right) \le L \left( f, R \right) \]
Riemann Integrable
Suppose that \( f: \left[ a, b \right] \to \mathbb{ R } \) is bounded, then we say that it is Riemann integrable if \[ \sup _ { P } \left( L \left( f, P \right) \right) = \inf _ { Q } \left( U \left( f, Q \right) \right) \] In that case we write \[ \int _ { a } ^ { b } f \left( x \right) dx \] as their common value
Riemann Integrable Characterizations
Let \( f \left( x \right) \) be bounded on \( \left[ a, b \right] \) then the following are equivalent
  • \( f \) is Riemann integrable
  • For each \( \epsilon \in \mathbb{ R } ^ + \) there is a partition \( P \) such that
    • \[ U \left( f, P \right) - L \left( f, P \right) \lt \epsilon \]
  • For every \( \epsilon \in \mathbb{ R } ^ + \) there is some \( \delta \in \mathbb{ R } ^ + \) such that for every partition \( Q \) such that \( \operatorname{ mesh } \left( Q \right) \lt \delta \) wherein
    • \[ U \left( f, Q \right) - L \left( f, Q \right) \lt \epsilon \]
  • There exists an \( I \in \mathbb{ R } \) such that for every \( \epsilon \in \mathbb{ R } ^ + \) there is a \( \delta \in \mathbb{ R } ^ + \) such that for every partition \( Q \) with \( \operatorname{ mesh } \left( Q \right) \lt \delta \) and every evaluation sequence \( X \) of \( Q \) we have
    • \[ \left\lvert I \left( f, Q, X \right) - I \right\rvert \lt \epsilon \] and in that case \( I = \int _ a ^ b f \left( x \right) dx \)

Note that bullet point 2 is a the most tractable for explicit functions, as one can construct a partition that works for a given function.

Jump Discontinuity
Suppose \( f: \left[ a, b \right] \to \mathbb{ R } \), and \( c \in \left( a, b \right) \) then if \[ \lim _{ x \to c ^ + } f \left( x \right) \neq \lim _ { x \to c ^ - } f \left( x \right) \] then we say that \( f \) has a jump discontinuity at \( c \)
Function Difference Set is the Same as its Negative
Suppose that \( f: A \to \mathbb{ R } \) and define \( F = \left\{ f \left( x \right) - f \left( y \right) : x, y \in A \right\} \), then \[ F = -F \]
Note that \[ F = \left\{ f \left( x \right) - f \left( y \right) : x, y \in A \right\} = \left\{ - \left( f \left( y \right) - f \left( x \right) \right) : y, x \in A \right\} = -F \] as needed.
Supremum of the Function Difference Set is the Same with Absolute Values

Piecewise Continuous
A function \( f: \left[ a, b \right] \to \mathbb{ R } \) is called piecewise continuous if for every \( \left[ c, d \right] \subseteq \left[ a, b \right] \) if it only has a finite number of discontinuities all of which are jump discontinuities
Riemann Integrable on Two Parts of the Interval Means Riemann Integrable on the Whole Interval
Suppose that \( f: \left[ a, b \right] \to \mathbb{ R } \) and let \( x \in \left[ a, b \right] \) then if \( f \) restricted to \( \left[ a, x \right] \) and \( f \) restricted to \( \left[ x, b \right] \) (which we denote as \( g _ 1, g _ 2 \)) are both riemann integrable then so is \( f \) on \( \left[ a, b \right] \) and then \[ \int _ { a } ^ { b } f \left( x \right) dx = \int _ { a } ^ { x } f \left( x \right) dx + \int _ { x } ^ { b } f \left( x \right) dx \]
Let \( \epsilon \in \mathbb{ R } ^ + \) using bullet point 2 from the RMI characterization we know that there are partitions \( P _ 1, P _ 2 \) of \( \left[ a, x \right], \left[ x, b \right] \) such that \[ \left( U \left( g _ 1, P _ 1 \right) - L \left( g _ 1, P _ 1 \right) \right) + \left( U \left( g _ 2, P _ 2 \right) - L \left( g _ 2, P _ 2 \right) \right) \lt \epsilon \] Then note that \[ U \left( g _ 1, P _ 1 \right) + U \left( g _ 2, P _ 2 \right) = U \left( f, P _ 1 \cup P _ 2 \right) \] which also holds for the lower sums, and therefore from the previous inequality we have \[ U \left( f, P _ 1 \cup P _ 2 \right) - L \left( f, P _ 1 \cup P _ 2 \right) \lt \epsilon \] therefore by (2) of the RMI characterization \( f \) is Riemann integrable.
Every Piecewise Continuous Function is Riemann Integrable
Every piecewise continuous function is Riemann integrable

Since \( f \) is piecewise continuous, then suppose its jump discontinuity points are given by \( D = \left\{ d _ 1, d _ 2, \ldots d _ n \right\} \). We will show this by using strong induction on the number of discontinuities within \( \left[ a, b \right] \).

Note that one can manage to prove that if \( f \) is RMI over \( \left[ a, b \right] \) with \( 1 \le i \le k \) jump discontinuities, then it is RMI over \( \left[ a, b \right] \) if it has \( k + 1 \) discontinuities. This can be done because we can break \( \left[ a, b \right] \) into two intervals \( I _ 1, I _ 2 \) such that \( I _ 1 \sqcup I _ 2 = \left[ a, b \right] \), where \( I _ 1, I _ 2 \) each have a number of discontinuities within the range \( \left[ 1, \ldots , k \right] \) so that \( f \restriction I _ 1 \) and \( f \restriction I _ 2 \) are both integrable on their respective intervals, so that \( f \) is RMI on \( \left[ a, b \right] \).

Therefore all that remains is the base case, which is that \( f \) is RMI if there is exactly \( 1 \) jump discontinuity of \( f \) on the set \( \left[ a, b \right] \), let \( d \) be this discontinuity, since \( f \) is continuous on \( L := \left[ a, d - \delta \right] \) and \( R := \left[ d + \delta , b \right] \) for any small \( \delta \), then we know that \( f \) is RMI when restricted to these intervals.

Let \( \epsilon \in \mathbb{ R } ^ + \) by (2) of the RMI characterization we obtain \( P _ L, P _ R \) which are partitions of \( L, R \) respectively (note that \( P _ L \cap P _ R = \emptyset \) such that \[ U \left( f \restriction L, P _ L \right) - L \left( f \restriction L, P _ L \right) \lt \frac{\epsilon }{3} \] and \[ U \left( f \restriction R, P _ R \right) - L \left( f \restriction R, P _ R \right) \lt \frac{\epsilon}{3} \]

Finally Consider \( P _ L \cup P _ R \) which is a partition of \( \left[ a, b \right] \) which contains the section \( \left[ d - \delta, d + \delta \right] \) along with the left and right partitions. Focusing on this section we can see it will contribute \[ \left( \sup _ { x \in \left[ d - \delta, d + \delta \right] } f \left( x \right) - \sup _ { y \in \left[ d - \delta, d + \delta \right] } f \left( y \right) \right) 2 \delta \] to \( U \left( f, P _ L \cup P _ R \right) - L \left( f, P _ L \cup P _ R \right) \), now:

\[ \begin{align} U \left( f, P _ L \cup P _ R \right) - L \left( f, P _ L \cup P _ R \right) &= \left( U \left( f \restriction L, P _ L \right) + \left( \sup _ { x \in \left[ d - \delta, d + \delta \right] }f \left( x \right) \right) 2 \delta + U \left( f \restriction R, P _ R \right) \right) - \left( L \left( f \restriction L, P _ L \right) + \left( \inf _ { x \in \left[ d - \delta, d + \delta \right] }f \left( x \right) \right) 2 \delta + L \left( f \restriction R, P _ R \right) \right) \\ &= \left( U \left( f \restriction L , P _ L \right) - L \left( f \restriction L, P _ L \right) \right) + \left( \sup _ { x \in \left[ d - \delta, d + \delta \right] }f \left( x \right) - \inf _ { x \in \left[ d - \delta, d + \delta \right] } f \left( x \right) \right) 2 \delta + \left( U \left( f \restriction R, P _ R \right) - L \left( f \restriction L , P _ L \right) \right)\\ &\lt \frac{2 \epsilon }{3} + \sup _ { x, y \in \left[ d - \delta, d + \delta \right] } \left( \left\lvert f \left( x \right) - f \left( y \right) \right\rvert \right) 2 \delta \\ &\le \frac{2 \epsilon }{3} + 2 \sup _ { x \in \left[ d - \delta, d + \delta \right] } \left\lvert f \left( x \right) \right\rvert \\ &\le \frac{2 \epsilon }{3} + 2 \lVert f \rVert _ \infty 2 \delta \\ &= \frac{2 \epsilon }{3} + 4 \lVert f \rVert _ \infty \delta \end{align} \] Thus a selection of \( \delta = \frac{\epsilon }{4 \lVert f \rVert _ \infty 3 } \) shows that \( U \left( f, P _ L \cup P _ R \right) - L \left( f, P _ L \cup P _ R \right) \lt \epsilon \) as needed.
Volume of a Rectangle
Suppose that \[ A = \left[ a _ 1, b _ 1 \right] \times \cdots \times \left[ a _ n, b _ n \right] \subseteq \mathbb{ R } ^ n. \] The volume of the closed rectangle \( A \) is \[ v \left( A \right) = \prod _ { i = 1 } ^ n \left( b _ i - a _ i \right). \] We use the same formula for the corresponding open rectangle \( \left( a _ 1, b _ 1 \right) \times \cdots \times \left( a _ n, b _ n \right) \).
Partition of a Closed Rectangle
Suppose that \( A = \left[ a _ 1, b _ 1 \right] \times \cdots \times \left[ a _ n, b _ n \right] \) is a closed rectangle. A partition of \( A \) is a tuple \[ P = \left( P _ 1, \ldots, P _ n \right), \] where each \( P _ i \) is a partition of the interval \( \left[ a _ i, b _ i \right] \).
Subrectangle of a Partition
Let \( P = \left( P _ 1,\ldots,P _ n \right) \) be a partition of a closed rectangle \( A \), and write \[ P _ i = \left( t _ { i,0 }, t _ { i,1 }, \ldots, t _ { i,N_i } \right). \] A subrectangle of \( P \) is a rectangle of the form \[ \left[ t _ { 1,j_1 - 1 }, t _ { 1,j_1 } \right] \times \cdots \times \left[ t _ { n,j_n - 1 }, t _ { n,j_n } \right], \] where \( 1 \le j _ i \le N _ i \) for every \( i \).
Multivariable Upper and Lower Sums
Suppose that \( A \subseteq \mathbb{ R } ^ n \) is a closed rectangle, \( f: A \to \mathbb{ R } \) is bounded, and \( P \) is a partition of \( A \). For each subrectangle \( S \) of \( P \), define \[ m _ S \left( f \right) = \inf \left\{ f \left( x \right): x \in S \right\} \] and \[ M _ S \left( f \right) = \sup \left\{ f \left( x \right): x \in S \right\}. \] The lower sum and upper sum of \( f \) over \( P \) are \[ L \left( f, P \right) = \sum _ S m _ S \left( f \right) v \left( S \right) \] and \[ U \left( f, P \right) = \sum _ S M _ S \left( f \right) v \left( S \right), \] where \( v\left( S \right) \) is the volume of the rectangle \( S \), and the sums range over all subrectangles \( S \) of \( P \).
Riemann Integrable on a Closed Rectangle
Suppose that \( A \subseteq \mathbb{ R } ^ n \) is a closed rectangle and \( f: A \to \mathbb{ R } \) is bounded. We say that \( f \) is Riemann integrable on \( A \) if \[ \sup _ P L \left( f, P \right) = \inf _ P U \left( f, P \right), \] where \( P \) ranges over all partitions of \( A \). The common value is denoted \[ \int _ A f \] or \[ \int _ A f \left( x ^ 1,\ldots,x ^ n \right) dx ^ 1 \cdots dx ^ n. \]
Multivariable Refinement Improves Darboux Sums
Suppose that \( A \subseteq \mathbb{ R } ^ n \) is a closed rectangle, \( f: A \to \mathbb{ R } \) is bounded, and \( P' \) refines \( P \), meaning each subrectangle of \( P' \) is contained in a subrectangle of \( P \). Then \[ L \left( f, P \right) \le L \left( f, P' \right) \] and \[ U \left( f, P' \right) \le U \left( f, P \right). \]
Multivariable Lower Sums are Below Upper Sums
If \( P \) and \( Q \) are partitions of a closed rectangle \( A \), then \[ L \left( f, P \right) \le U \left( f, Q \right). \]
Multivariable Riemann Integrability Criterion
Suppose that \( A \subseteq \mathbb{ R } ^ n \) is a closed rectangle and \( f: A \to \mathbb{ R } \) is bounded. Then \( f \) is Riemann integrable on \( A \) if and only if for every \( \epsilon \in \mathbb{ R } ^ + \), there is a partition \( P \) of \( A \) such that \[ U \left( f, P \right) - L \left( f, P \right) \lt \epsilon. \]
Measure Zero in Euclidean Space
A set \( E \subseteq \mathbb{ R } ^ n \) has \( n \)-dimensional measure zero if for every \( \epsilon \in \mathbb{ R } ^ + \), there is a sequence \( U _ 1,U _ 2,\ldots \) of closed rectangles such that \[ E \subseteq \bigcup _ { i = 1 } ^ \infty U _ i \] and \[ \sum _ { i = 1 } ^ \infty v \left( U _ i \right) \lt \epsilon. \] Here \( v\left( U_i \right) \) is the volume of the rectangle \( U_i \).
Content Zero in Euclidean Space
A set \( E \subseteq \mathbb{ R } ^ n \) has \( n \)-dimensional content zero if for every \( \epsilon \in \mathbb{ R } ^ + \), there are finitely many closed rectangles \( U _ 1,\ldots,U _ k \) such that \[ E \subseteq U _ 1 \cup \cdots \cup U _ k \] and \[ \sum _ { i = 1 } ^ k v \left( U _ i \right) \lt \epsilon. \] Here \( v\left( U_i \right) \) is the volume of the rectangle \( U_i \).
Countable Union of Measure Zero Sets has Measure Zero
If \( E = \bigcup _ { i = 1 } ^ \infty E _ i \) and each \( E _ i \subseteq \mathbb{ R } ^ n \) has measure zero, then \( E \) has measure zero.
Compact Measure Zero Implies Content Zero
If \( E \subseteq \mathbb{ R } ^ n \) is compact and has measure zero, then \( E \) has content zero.
Lebesgue Criterion for Multivariable Riemann Integrability
Suppose that \( A \subseteq \mathbb{ R } ^ n \) is a closed rectangle and \( f: A \to \mathbb{ R } \) is bounded. Let \[ D = \left\{ x \in A: f \text{ is not continuous at } x \right\}. \] Then \( f \) is Riemann integrable on \( A \) if and only if \( D \) has measure zero.
Jordan Measurable Set
A bounded set \( C \subseteq \mathbb{ R } ^ n \) is Jordan measurable if its characteristic function \( \chi _ C \) is Riemann integrable on some closed rectangle \( A \) with \( C \subseteq A \). In that case its Jordan content is \[ \left\lvert C \right\rvert = \int _ A \chi _ C. \]
Jordan Measurable iff Boundary has Measure Zero
A bounded set \( C \subseteq \mathbb{ R } ^ n \) is Jordan measurable if and only if its boundary has measure zero.
Integral over a Jordan Measurable Set
Suppose that \( C \subseteq \mathbb{ R } ^ n \) is bounded, \( C \subseteq A \), \( A \) is a closed rectangle, and \( f: A \to \mathbb{ R } \) is bounded. If \( f \chi _ C \) is Riemann integrable on \( A \), define \[ \int _ C f = \int _ A f \chi _ C. \]
Removing a Measure-Zero Set Does Not Change a Riemann Integral
Suppose \( C \subseteq \mathbb{ R } ^ n \) is Jordan measurable, \( E \subseteq C \) has measure zero, and \( f:C\to\mathbb{ R } \) is bounded and integrable over \( C \). Then \[ \int_C f = \int_{C\setminus E} f. \]
Choose a closed rectangle \( A \) containing \( C \). The two integrals are \[ \int_A f\chi_C \qquad\text{and}\qquad \int_A f\chi_{C\setminus E}. \] These two functions differ only on \( E \). Since \( E \) has measure zero, changing a bounded function only on \( E \) does not change its upper or lower Riemann integrals: rectangles covering \( E \) can be chosen with arbitrarily small total volume, so the possible change in any Riemann sum is bounded by the bound for \( f \) times an arbitrarily small total volume. Therefore the two integrals are equal.
Fubini for Riemann Integrals on Rectangles
Suppose that \( A \subseteq \mathbb{ R } ^ n \) and \( B \subseteq \mathbb{ R } ^ m \) are closed rectangles and \( f: A \times B \to \mathbb{ R } \) is Riemann integrable. If for each \( x \in A \), the function \( y \mapsto f \left( x,y \right) \) is Riemann integrable on \( B \), and the function \[ x \mapsto \int _ B f \left( x,y \right) dy \] is Riemann integrable on \( A \), then \[ \int _ { A \times B } f = \int _ A \left( \int _ B f \left( x,y \right) dy \right) dx. \]
Support of a Function
The support of a function \( \varphi: A \to \mathbb{ R } \) is the closure of the set where \( \varphi \) is nonzero: \[ \operatorname{supp} \left( \varphi \right) = \overline{ \left\{ x \in A: \varphi \left( x \right) \neq 0 \right\} }. \]
Partition of Unity
Let \( A \subseteq \mathbb{ R } ^ n \) and let \( \mathcal{ U } \) be an open cover of \( A \). A partition of unity for \( A \), subordinate to \( \mathcal{ U } \), is a collection \( \Phi \) of nonnegative continuous functions \( \varphi: A \to \mathbb{ R } \) such that:
  • for each \( \varphi \in \Phi \), there is some \( U \in \mathcal{ U } \) with \( \operatorname{supp} \left( \varphi \right) \) \( \subseteq U \),
  • each point of \( A \) has a neighborhood on which only finitely many \( \varphi \in \Phi \) are nonzero,
  • for every \( x \in A \), \( \sum _ { \varphi \in \Phi } \varphi \left( x \right) = 1 \).
Existence of Partitions of Unity in Euclidean Space
If \( A \subseteq \mathbb{ R } ^ n \) and \( \mathcal{ U } \) is an open cover of \( A \), then there is a partition of unity for \( A \) subordinate to \( \mathcal{ U } \).
First suppose that \( A \) is compact. Choose a finite subcover \( U_1,\ldots,U_N \) of \( A \) from \( \mathcal{ U } \). We construct compact sets \( D_i\subseteq U_i \) whose interiors cover \( A \).

Suppose \( D_1,\ldots,D_k \) have been chosen so that \[ \operatorname{int}D_1,\ldots,\operatorname{int}D_k,U_{k+1},\ldots,U_N \] still cover \( A \). Let \[ C_{k+1} = A\setminus \left( \operatorname{int}D_1\cup\cdots\cup\operatorname{int}D_k \cup U_{k+2}\cup\cdots\cup U_N \right). \] Then \( C_{k+1} \) is compact and \( C_{k+1}\subseteq U_{k+1} \). Since \( U_{k+1} \) is open, choose a compact neighborhood \( D_{k+1} \) with \[ C_{k+1}\subseteq\operatorname{int}D_{k+1} \qquad\text{and}\qquad D_{k+1}\subseteq U_{k+1}. \] Continuing this process gives compact sets \( D_1,\ldots,D_N \) whose interiors cover \( A \).

For each \( i \), define a nonnegative continuous function \( \psi_i \) that is positive on \( D_i \) and zero outside \( U_i \). One explicit choice uses the Euclidean distance to a set: \[ \psi_i\left( x \right) = \frac{d\left( x,\mathbb{ R } ^ n\setminus U_i \right)} {d\left( x,\mathbb{ R } ^ n\setminus U_i \right)+d\left( x,D_i \right)}. \] This is continuous, equals \(0\) off \(U_i\), and is positive on \(D_i\). Since the interiors of the \(D_i\) cover \(A\), the sum \[ \sigma\left( x \right)=\psi_1\left( x \right)+\cdots+\psi_N\left( x \right) \] is positive on \( A \). Set \[ \varphi_i\left( x \right)=\frac{\psi_i\left( x \right)}{\sigma\left( x \right)}. \] Then the \( \varphi_i \) are nonnegative continuous functions, their sum is \(1\) on \(A\), and the support of \( \varphi_i \) is contained in \( U_i \). Thus they form the required partition in the compact case.

Next suppose that \( A=A_1\cup A_2\cup\cdots \), where each \( A_i \) is compact and \( A_i\subseteq\operatorname{int}A_{i+1} \). Put \( A_0=A_{-1}=\emptyset \) and \[ B_i=A_i\setminus\operatorname{int}A_{i-1}. \] Each \( B_i \) is compact. Cover \( B_i \) by the sets \[ U\cap\left( \operatorname{int}A_{i+1}\setminus A_{i-2} \right), \qquad U\in\mathcal{ U}. \] By the compact case, choose a partition of unity \( \Phi_i \) for \( B_i \) subordinate to this cover. At each \( x\in A \), only finitely many of the functions from all the \( \Phi_i \) are nonzero near \( x \), because the \(i\)-th family is supported in \( \operatorname{int}A_{i+1}\setminus A_{i-2} \). Therefore \[ \sigma\left( x \right)=\sum_i\sum_{\varphi\in\Phi_i}\varphi\left( x \right) \] is locally finite and positive on \( A \). Replacing every \( \varphi \) by \( \varphi/\sigma \) gives a partition of unity on \( A \) subordinate to \( \mathcal{ U} \).

If \( A \) is open, define \[ A_i = \left\{ x\in A:\left\lvert x\right\rvert\le i \text{ and } d\left( x,\mathbb{ R } ^ n\setminus A \right)\ge\frac{1}{i} \right\}. \] These sets are compact, their union is \( A \), and \( A_i\subseteq\operatorname{int}A_{i+1} \). Hence the previous case applies.

Finally let \( A \) be arbitrary and let \( B=\bigcup_{U\in\mathcal{ U}}U \). The set \( B \) is open and \( A\subseteq B \). Apply the open case to \( B \) and then restrict the resulting functions to \( A \). The restricted functions still sum to \(1\) on \(A\), are locally finite on \(A\), and have supports contained in elements of \( \mathcal{ U} \). Thus they form the desired partition of unity.

Change of Variables for Riemann Integrals
Let \( A \subseteq \mathbb{ R } ^ n \) be open and let \( g: A \to \mathbb{ R } ^ n \) be one-to-one and continuously differentiable with \[ \det \left[ Dg\left( x \right) \right] _ E \neq 0 \] for every \( x \in A \). If \( f: g \left( A \right) \to \mathbb{ R } \) is integrable, then \[ \int _ { g \left( A \right) } f = \int _ A \left( f \circ g \right) \left\lvert \det \left[ Dg \right] _ E \right\rvert. \]

First reduce the result to local pieces. Suppose \( \mathcal{ V} \) is an open cover of \( A \) such that the formula holds on each \( V\in\mathcal{ V} \). The sets \( g\left( V \right) \) form an open cover of \( g\left( A \right) \). Choose a partition of unity \( \Phi \) on \( g\left( A \right) \) subordinate to this cover. For each \( \varphi\in\Phi \), choose \( V_\varphi\in\mathcal{ V} \) with \[ \operatorname{supp}\left( \varphi \right)\subseteq g\left( V_\varphi \right). \] Since \( g \) is one-to-one, the function \( \left( \varphi f \right)\circ g \) is zero outside \( V_\varphi \). Applying the local formula to \( \varphi f \) gives \[ \int_{g\left( A \right)}\varphi f = \int_A \left( \left( \varphi f \right)\circ g \right) \left\lvert \det \left[ Dg \right] _ E \right\rvert. \] Summing over the locally finite family \( \Phi \), and using \( \sum_{\varphi\in\Phi}\varphi=1 \), gives \[ \int_{g\left( A \right)}f = \int_A \left( f\circ g \right) \left\lvert \det \left[ Dg \right] _ E \right\rvert. \]

It is enough to prove the formula for constant functions on rectangles. Indeed, let \( R\subseteq g\left( A \right) \) be a closed rectangle and let \( P \) be a partition of \( R \). For each subrectangle \( S \), let \( m_S\left( f \right) \) and \( M_S\left( f \right) \) be the lower and upper values of \( f \) on \( S \). Applying the constant-function case to the characteristic pieces of \(S\) gives the lower-sum inequality \[ L\left( f,P \right) \le \int_{g^{-1}\left( R \right)} \left( f\circ g \right)\left\lvert \det \left[ Dg \right] _ E \right\rvert \] and similarly the upper-sum inequality in the other direction. Taking the supremum of lower sums and the infimum of upper sums, using the definition of Riemann integrability, proves the formula for \( f \).

The formula is compatible with composition. If it holds for \( g:A\to\mathbb{ R } ^ n \) and for \( h:B\to\mathbb{ R } ^ n \), with \( g\left( A \right)\subseteq B \), then \[ \int_{h\left( g\left( A \right) \right)} f = \int_{g\left( A \right)} \left( f\circ h \right)\left\lvert \det \left[ Dh \right] _ E \right\rvert \] and applying the formula again to \( g \) gives \[ \int_A \left( f\circ h\circ g \right) \left\lvert \det \left[ Dh\left( g\left( x \right) \right) \right] _ E\right\rvert \left\lvert \det \left[ Dg\left( x \right) \right] _ E\right\rvert. \] By the chain rule and multiplicativity of determinants, this is \[ \int_A \left( f\circ h\circ g \right) \left\lvert \det \left[ D\left( h\circ g \right) \right] _ E\right\rvert. \]

The formula holds for invertible linear maps because the determinant is the signed volume-scaling factor of a linear map: for every rectangle \( R \), \[ \left\lvert T\left( R \right)\right\rvert = \left\lvert \det T \right\rvert\left\lvert R \right\rvert, \] and the constant-function reduction then gives the integral formula.

We now prove the theorem by induction on \( n \). The case \( n=1 \) is the ordinary one-dimensional substitution formula. Assume the theorem is known in dimension \( n-1 \), and let \( a\in A \). By the local reduction it is enough to prove the formula on some open neighborhood of \( a \). Composing with the inverse of the linear map \( Dg\left( a \right) \), and using the linear case and the composition step, we may assume \[ \left[ Dg\left( a \right) \right] _ E=I_n. \]

Write \( g=\left( g^1,\ldots,g^n \right) \). Define \[ h\left( x^1,\ldots,x^n \right) = \left( g^1\left( x \right),\ldots,g^{n-1}\left( x \right),x^n \right). \] Since \( \left[ Dh\left( a \right) \right] _ E=I_n \), the inverse function theorem gives an open neighborhood \( U' \) of \( a \) on which \( h \) is one-to-one and \( \det \left[ Dh \right] _ E \neq 0 \). On \( h\left( U' \right) \), define \[ k\left( y^1,\ldots,y^n \right) = \left( y^1,\ldots,y^{n-1},g^n\left( h^{-1}\left( y \right) \right) \right). \] Then \( g=k\circ h \) on \( U' \). Shrinking \( U' \) if needed, the inverse function theorem also gives \( \det \left[ Dk \right] _ E\neq0 \) on \( h\left( U' \right) \). By the composition step, it remains to prove the formula for maps of the form \( h \) and \( k \), each of which changes only one coordinate after the other coordinates are fixed.

We prove it for \( h \); the proof for \( k \) is the same. Let \( W=D\times\left[ a_n,b_n \right]\subseteq U' \), where \( D\subseteq\mathbb{ R } ^ { n-1 } \) is a rectangle. For each fixed \( x^n \), define \[ h_{x^n}\left( x^1,\ldots,x^{n-1} \right) = \left( g^1\left( x \right),\ldots,g^{n-1}\left( x \right) \right). \] Each \( h_{x^n} \) is a one-to-one continuously differentiable map in \( \mathbb{ R } ^ { n-1 } \), and \[ \det \left[ Dh_{x^n}\left( x^1,\ldots,x^{n-1} \right) \right] _ E = \det \left[ Dh\left( x^1,\ldots,x^n \right) \right] _ E. \] Applying the induction hypothesis to each slice and then applying Fubini's theorem gives \[ \int_{h\left( W \right)}1 = \int_{a_n}^{b_n} \int_D \left\lvert\det \left[ Dh\left( x^1,\ldots,x^n \right) \right] _ E\right\rvert dx^1\cdots dx^{n-1}dx^n = \int_W\left\lvert\det \left[ Dh \right] _ E\right\rvert. \] By the constant-function reduction, the change-of-variables formula holds for \( h \) on \( W \). These rectangles form a local cover of \( U' \), so the local reduction proves the formula near \( a \). Since \( a \) was arbitrary, the theorem follows on all of \( A \).

Polar Coordinate Change of Variables
Let \[ W=\left\{ \left( u,v \right)\in\mathbb{ R } ^ 2:u ^ 2+v ^ 2\lt R ^ 2 \right\} \] with \( R\gt 0 \). If \( q:W\to\mathbb{ R } \) is integrable, then \[ \int_W q\left( u,v \right)\;du\,dv = \int_0^{2\pi}\int_0^R q\left( r\cos\theta,r\sin\theta \right) r\;dr\,d\theta. \]
Let \[ \phi\left( r,\theta \right) = \left( r\cos\theta,r\sin\theta \right) \] on \( \left( 0,R \right)\times\left( 0,2\pi \right) \). This map is one-to-one onto \( W \) with the origin and one radius removed. Its Jacobian matrix is \[ \phi'\left( r,\theta \right) = \begin{bmatrix} \cos\theta & -r\sin\theta\\ \sin\theta & r\cos\theta \end{bmatrix}, \] so \[ \left\lvert \det \phi'\left( r,\theta \right) \right\rvert = r. \] The omitted origin and radius have measure zero, so by removing a measure-zero set, the integral over \( W \) equals the integral over the image of \( \phi \). Applying the change of variables theorem gives \[ \int_W q\left( u,v \right)\;du\,dv = \int_{\left( 0,R \right)\times\left( 0,2\pi \right)} q\left( r\cos\theta,r\sin\theta \right) r\;dr\,d\theta. \] Writing this integral as an iterated integral gives the stated formula.
Sard's Theorem for the Equal-Dimension \( C ^ 1 \) Case
Let \( A \subseteq \mathbb{ R } ^ n \) be open and let \( g: A \to \mathbb{ R } ^ n \) be continuously differentiable. If \[ B = \left\{ x \in A: \det \left[ Dg \left( x \right) \right] _ E = 0 \right\}, \] then \( g \left( B \right) \) has measure zero.

\( k \)-Tensor
Let \( V \) be a real vector space. A \( k \)-tensor on \( V \) is a multilinear function \[ T: V ^ k \to \mathbb{ R }. \] The vector space of \( k \)-tensors on \( V \) is denoted \( \mathcal{ T } ^ k\left( V \right) \).
Tensor Product
If \( S \in \)\( \mathcal{ T } ^ k\left( V \right) \) and \( T \in \mathcal{ T } ^ l\left( V \right) \), define the tensor product \( S \otimes T \in \mathcal{ T } ^ { k+l }\left( V \right) \) by \[ \left( S \otimes T \right)\left( v _ 1,\ldots,v _ { k+l } \right) = S\left( v _ 1,\ldots,v _ k \right) T\left( v _ { k+1 },\ldots,v _ { k+l } \right). \]
Tensor Product Basis
Let \( v _ 1,\ldots,v _ n \) be a basis for \( V \), and let \( \varphi ^ 1,\ldots,\varphi ^ n \) be the dual basis. Then the \( k \)-fold tensor products \[ \varphi ^ { i_1 } \otimes \cdots \otimes \varphi ^ { i_k } \] form a basis for \( \mathcal{ T } ^ k\left( V \right) \). Hence \( \dim \mathcal{ T } ^ k\left( V \right)=n ^ k \).
Alternating Tensor
A \( k \)-tensor \( \omega \) on \( V \) is alternating if interchanging any two arguments changes its sign. The vector space of alternating \( k \)-tensors on \( V \) is denoted \( \Lambda ^ k\left( V \right) \).
Alternating Operator
If \( T \in \)\( \mathcal{ T } ^ k\left( V \right) \), define \[ \operatorname{Alt}\left( T \right)\left( v _ 1,\ldots,v _ k \right) = \frac{1}{k!} \sum _ { \sigma \in S _ k } \operatorname{sgn}\left( \sigma \right) T\left( v _ { \sigma\left( 1 \right) },\ldots,v _ { \sigma\left( k \right) } \right). \]
Wedge Product
If \( \omega \in \)\( \Lambda ^ k\left( V \right) \) and \( \eta \in \Lambda ^ l\left( V \right) \), define the wedge product using the alternating operator and tensor product: \[ \omega \wedge \eta = \frac{\left( k+l \right)!}{k!l!} \operatorname{Alt}\left( \omega \otimes \eta \right). \]
Wedge Product Properties
The wedge product is bilinear, associative, and graded-commutative: \[ \omega \wedge \eta = \left( -1 \right) ^ { kl } \eta \wedge \omega \] for \( \omega \in \Lambda ^ k\left( V \right) \) and \( \eta \in \Lambda ^ l\left( V \right) \).
Alternating Tensor Basis
If \( v _ 1,\ldots,v _ n \) is a basis for \( V \) and \( \varphi ^ 1,\ldots,\varphi ^ n \) is its dual basis, then the elements \[ \varphi ^ { i_1 } \wedge \cdots \wedge \varphi ^ { i_k }, \qquad i _ 1 \lt \cdots \lt i _ k, \] form a basis for \( \Lambda ^ k\left( V \right) \). Hence \[ \dim \Lambda ^ k\left( V \right)=\binom{n}{k}. \]
Orientation of a Vector Space
An orientation of an \( n \)-dimensional real vector space \( V \) is one of the two classes of ordered bases, where two ordered bases are equivalent if the change-of-basis determinant between them is positive.
Volume Element of an Oriented Inner Product Space
If \( V \) is an oriented \( n \)-dimensional inner product space, its volume element is the unique alternating tensor \( \omega \in \Lambda ^ n\left( V \right) \) such that \[ \omega\left( v _ 1,\ldots,v _ n \right)=1 \] for every positively oriented orthonormal basis \( v _ 1,\ldots,v _ n \).
Vector Field on an Open Set
A vector field on an open set \( A \subseteq \mathbb{ R } ^ n \) is a function \( F \) assigning to each \( x \in A \) a vector \( F\left( x \right) \in \mathbb{ R } ^ n _ x \).
Differential Form on an Open Set
A \( k \)-form on an open set \( A \subseteq \mathbb{ R } ^ n \) assigns to each \( x \in A \) an alternating \( k \)-tensor on the tangent space \( \mathbb{ R } ^ n _ x \). In coordinates, a \( k \)-form can be written \[ \omega = \sum _ { i_1 \lt \cdots \lt i_k } \omega _ { i_1,\ldots,i_k } dx ^ { i_1 } \wedge \cdots \wedge dx ^ { i_k }. \]
Pullback of a Form
If \( f: A \to B \) is differentiable and \( \omega \) is a \( k \)-form on \( B \), then the pullback \( f ^ * \omega \) is the \( k \)-form on \( A \) defined by \[ \left( f ^ * \omega \right)\left( x \right)\left( v _ 1,\ldots,v _ k \right) = \omega\left( f\left( x \right) \right) \left( Df\left( x \right)v _ 1,\ldots,Df\left( x \right)v _ k \right). \]
Exterior Derivative
If \[ \omega = \sum _ { i_1 \lt \cdots \lt i_k } \omega _ { i_1,\ldots,i_k } dx ^ { i_1 } \wedge \cdots \wedge dx ^ { i_k }, \] then its exterior derivative is \[ d\omega = \sum _ { i_1 \lt \cdots \lt i_k } \sum _ { j = 1 } ^ n D _ j\omega _ { i_1,\ldots,i_k } dx ^ j \wedge dx ^ { i_1 } \wedge \cdots \wedge dx ^ { i_k }. \]
Exterior Derivative Properties
If \( \omega \) is a \( k \)-form and \( \eta \) is a form, then \[ d\left( \omega \wedge \eta \right) = d\omega \wedge \eta + \left( -1 \right) ^ k \omega \wedge d\eta. \] Also, \[ d\left( d\omega \right)=0. \]
Closed and Exact Forms
A form \( \omega \) is closed if its exterior derivative satisfies \( d\omega=0 \). It is exact if there is a form \( \eta \) such that \[ \omega=d\eta. \] Every exact form is closed.
Singular Cube
A singular \( k \)-cube in a set \( A \subseteq \mathbb{ R } ^ n \) is a differentiable map \[ c: \left[ 0,1 \right] ^ k \to A. \]
Singular Chain
A singular \( k \)-chain in \( A \) is a finite formal integer combination \[ c = \sum _ { i = 1 } ^ r a _ i c _ i \] of singular \( k \)-cubes \( c _ i \) in \( A \).
Boundary of a Singular Cube
For a singular \( k \)-cube \( c \), let \( c _ { \left( i,\alpha \right) } \) denote the \( \left( k-1 \right) \)-cube obtained by setting the \( i \)-th coordinate equal to \( \alpha \in \left\{ 0,1 \right\} \). The boundary of \( c \) is \[ \partial c = \sum _ { i = 1 } ^ k \sum _ { \alpha = 0 } ^ 1 \left( -1 \right) ^ { i+\alpha } c _ { \left( i,\alpha \right) }. \] This extends linearly to chains.
Boundary Squared is Zero
If \( c \) is a singular chain, then \[ \partial\left( \partial c \right)=0. \]
Integral of a Form over a Chain
If \( \omega \) is a \( k \)-form on \( A \) and \( c \) is a singular \( k \)-cube in \( A \), define the integral using the pullback \[ \int _ c \omega = \int _ { \left[ 0,1 \right] ^ k } c ^ *\omega. \] For a chain \( c=\sum a_i c_i \), define \[ \int _ c \omega = \sum a_i \int _ { c_i } \omega. \]
Stokes' Theorem for Chains
If \( \omega \) is a \( \left( k-1 \right) \)-form on an open set \( A \subseteq \mathbb{ R } ^ n \), and \( c \) is a singular \( k \)-chain in \( A \), then \[ \int _ c d\omega = \int _ { \partial c } \omega. \]

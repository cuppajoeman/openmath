Euclidean n-space
For \( n \in \mathbb{ N } \), Euclidean \( n \)-space is the set \[ \mathbb{ R } ^ n = \left\{ \left( x ^ 1, \ldots, x ^ n \right): x ^ i \in \mathbb{ R } \right\}. \] Its elements may be treated as points or as vectors in a real vector space, with vector addition and scalar multiplication defined coordinatewise.
Euclidean Norm
If \( x = \left( x ^ 1, \ldots, x ^ n \right) \in \mathbb{ R } ^ n \), then the Euclidean norm of \( x \) is \[ \left\lvert x \right\rvert = \sqrt{ \sum _ { i = 1 } ^ n \left( x ^ i \right) ^ 2 }. \]
Euclidean Distance
If \(x,y\in\mathbb R^n\), then the Euclidean distance from \(x\) to \(y\) is \[ d_{\mathbb R^n}(x,y) = \left\lvert x-y\right\rvert, \] where \(\left\lvert x-y\right\rvert\) is the Euclidean norm of the vector \(x-y\).
Euclidean Distance Is a Metric
The Euclidean distance \[ d_{\mathbb R^n}(x,y)=\left\lvert x-y\right\rvert \] is a metric on \(\mathbb R^n\).
Let \(x,y,z\in\mathbb R^n\). By the Euclidean norm properties, \[ d_{\mathbb R^n}(x,y)=\left\lvert x-y\right\rvert\ge0. \] Also, \[ d_{\mathbb R^n}(x,y)=0 \] if and only if \(\left\lvert x-y\right\rvert=0\), which holds if and only if \(x-y=0\), equivalently \(x=y\).

For symmetry, use \(x-y=-\left(y-x\right)\) and the scalar multiplication property of the Euclidean norm:

\[ d_{\mathbb R^n}(x,y) = \left\lvert x-y\right\rvert = \left\lvert -\left(y-x\right)\right\rvert = \left\lvert y-x\right\rvert = d_{\mathbb R^n}(y,x). \] Finally, since \(x-z=\left(x-y\right)+\left(y-z\right)\), the triangle inequality for the Euclidean norm gives \[ d_{\mathbb R^n}(x,z) = \left\lvert x-z\right\rvert = \left\lvert \left(x-y\right)+\left(y-z\right)\right\rvert \le \left\lvert x-y\right\rvert+\left\lvert y-z\right\rvert = d_{\mathbb R^n}(x,y)+d_{\mathbb R^n}(y,z). \] Therefore \(d_{\mathbb R^n}\) satisfies the definition of a metric.
Euclidean Inner Product
If \( x, y \in \mathbb{ R } ^ n \), then their Euclidean inner product is \[ \left( x, y \right) = \sum _ { i = 1 } ^ n x ^ i y ^ i. \]
Euclidean Norm Properties
If \( x, y \in \mathbb{ R } ^ n \) and \( a \in \mathbb{ R } \), then:
  • \( \left\lvert x \right\rvert \ge 0 \), and \( \left\lvert x \right\rvert = 0 \) if and only if \( x = 0 \).
  • \( \left\lvert \sum _ { i = 1 } ^ n x ^ i y ^ i \right\rvert \le \left\lvert x \right\rvert \left\lvert y \right\rvert \), with equality if and only if \( x \) and \( y \) are linearly dependent.
  • \( \left\lvert x + y \right\rvert \le \left\lvert x \right\rvert + \left\lvert y \right\rvert \).
  • \( \left\lvert a x \right\rvert = \left\lvert a \right\rvert \left\lvert x \right\rvert \).
Euclidean Inner Product Properties
If \( x, x _ 1, x _ 2, y, y _ 1, y _ 2 \in \mathbb{ R } ^ n \) and \( a \in \mathbb{ R } \), then:
  • \( \left( x, y \right) = \left( y, x \right) \).
  • \( \left( a x, y \right) = \left( x, a y \right) = a \left( x, y \right) \).
  • \( \left( x _ 1 + x _ 2, y \right) = \left( x _ 1, y \right) + \left( x _ 2, y \right) \).
  • \( \left( x, y _ 1 + y _ 2 \right) = \left( x, y _ 1 \right) + \left( x, y _ 2 \right) \).
  • \( \left( x, x \right) \ge 0 \), and \( \left( x, x \right) = 0 \) if and only if \( x = 0 \).
  • \( \left\lvert x \right\rvert = \sqrt{ \left( x, x \right) } \).
  • \( \left( x, y \right) = \frac{ \left\lvert x + y \right\rvert ^ 2 - \left\lvert x - y \right\rvert ^ 2 }{4} \).
Standard Basis
The standard basis of \( \mathbb{ R } ^ n \) is \( e _ 1, \ldots, e _ n \), where \( e _ i \) has \( 1 \) in its \( i \)-th coordinate and \( 0 \) in every other coordinate.
Product of Subsets of Euclidean Space
If \( A \subseteq \mathbb{ R } ^ m \) and \( B \subseteq \mathbb{ R } ^ n \), then their Cartesian product is \[ A \times B = \left\{ \left( x, y \right) \in \mathbb{ R } ^ { m + n }: x \in A \text{ and } y \in B \right\}. \]
Closed Rectangle in Euclidean Space
A closed rectangle in \( \mathbb{ R } ^ n \) is a product of closed intervals, a set of the form \[ \left[ a _ 1, b _ 1 \right] \times \cdots \times \left[ a _ n, b _ n \right]. \]
Open Rectangle in Euclidean Space
An open rectangle in \( \mathbb{ R } ^ n \) is a product of open intervals, a set of the form \[ \left( a _ 1, b _ 1 \right) \times \cdots \times \left( a _ n, b _ n \right). \]
Open Subset of Euclidean Space
A set \( U \subseteq \mathbb{ R } ^ n \) is open if for every \( x \in U \), there is an open rectangle \( A \) such that \[ x \in A \subseteq U. \]
Near a Point in Euclidean Space
A statement depending on points of \( \mathbb{ R } ^ n \) holds near \( a\in\mathbb{ R } ^ n \) if there is an open set \( U\subseteq\mathbb{ R } ^ n \) with \( a\in U \) such that the statement holds for every point of \( U \).

For example, a function has a property near \( a \) if it has that property on some open set containing \( a \).

Closed Subset of Euclidean Space
A set \( C \subseteq \mathbb{ R } ^ n \) is closed if \( \mathbb{ R } ^ n \setminus C \) is open.
Interior, Exterior, and Boundary
Let \( A \subseteq \mathbb{ R } ^ n \). A point \( x \in \mathbb{ R } ^ n \) is:
  • in the interior of \( A \) if there is an open rectangle \( B \) such that \( x \in B \subseteq A \),
  • in the exterior of \( A \) if there is an open rectangle \( B \) such that \( x \in B \subseteq \mathbb{ R } ^ n \setminus A \),
  • on the boundary of \( A \) if every open rectangle \( B \) containing \( x \) meets both \( A \) and \( \mathbb{ R } ^ n \setminus A \).
Open Cover
A collection \( \mathcal{ U } \) of open subsets of \( \mathbb{ R } ^ n \) is an open cover of \( A \subseteq \mathbb{ R } ^ n \) if every point of \( A \) lies in some \( U \in \mathcal{ U } \).
Compact Subset of Euclidean Space
A set \( A \subseteq \mathbb{ R } ^ n \) is compact if every open cover of \( A \) has a finite subcover.
Heine-Borel for Closed Intervals
Every closed interval \( \left[ a, b \right] \subseteq \mathbb{ R } \) is compact.
Tube Lemma for a Compact Factor
Let \( B \subseteq \mathbb{ R } ^ m \) be compact and let \( x \in \mathbb{ R } ^ n \). If \( \mathcal{ U } \) is an open cover of \( \left\{ x \right\} \times B \), then there is an open set \( V \subseteq \mathbb{ R } ^ n \) with \( x \in V \) such that \( V \times B \) is covered by finitely many sets from \( \mathcal{ U } \).
Product of Compact Sets is Compact
If \( A \subseteq \mathbb{ R } ^ n \) and \( B \subseteq \mathbb{ R } ^ m \) are compact, then \( A \times B \subseteq \mathbb{ R } ^ { n + m } \) is compact.
Finite Product of Compact Sets is Compact
If \( A _ 1, \ldots, A _ k \) are compact, then \( A _ 1 \times \cdots \times A _ k \) is compact. In particular, every closed rectangle in \( \mathbb{ R } ^ k \) is compact.
Closed Bounded Subset of Euclidean Space is Compact
Every closed bounded subset of \( \mathbb{ R } ^ n \) is compact.
Function Between Euclidean Spaces
A function \( f: A \to \mathbb{ R } ^ m \), where \( A \subseteq \mathbb{ R } ^ n \), assigns to each \( x \in A \) a point \( f \left( x \right) \in \mathbb{ R } ^ m \). The set \( A \) is called the domain of \( f \).
Image and Preimage
If \( f: A \to \mathbb{ R } ^ m \), \( B \subseteq A \), and \( C \subseteq \mathbb{ R } ^ m \), then the image and preimage are \[ f \left( B \right) = \left\{ f \left( x \right): x \in B \right\} \] and \[ f ^ { -1 } \left( C \right) = \left\{ x \in A: f \left( x \right) \in C \right\}. \]
Component Functions
A function between Euclidean spaces \( f: A \to \mathbb{ R } ^ m \) determines component functions \( f ^ 1, \ldots, f ^ m: A \to \mathbb{ R } \) by \[ f \left( x \right) = \left( f ^ 1 \left( x \right), \ldots, f ^ m \left( x \right) \right). \] These are called the component functions of \( f \).
Projection Function
The \( i \)-th projection function \( \pi ^ i: \mathbb{ R } ^ n \to \mathbb{ R } \) is defined by \[ \pi ^ i \left( x ^ 1, \ldots, x ^ n \right) = x ^ i. \]
Limit of a Vector-Valued Function
Let \( f: A \to \mathbb{ R } ^ m \), where \( A \subseteq \mathbb{ R } ^ n \). We write the limit \[ \lim _ { x \to a } f \left( x \right) = b \] when for every \( \epsilon \in \mathbb{ R } ^ + \), there is a \( \delta \in \mathbb{ R } ^ + \) such that if \( x \in A \) and \( 0 \lt \left\lvert x - a \right\rvert \lt \delta \), then \( \left\lvert f \left( x \right) - b \right\rvert \lt \epsilon \).
Continuity Between Euclidean Spaces
A function \( f: A \to \mathbb{ R } ^ m \), where \( A \subseteq \mathbb{ R } ^ n \), is continuous at \( a \in A \) if \[ \lim _ { x \to a } f \left( x \right) = f \left( a \right). \] It is continuous if it is continuous at every point of \( A \).
Continuity Open Set Characterization
Let \( A \subseteq \mathbb{ R } ^ n \). A function \( f: A \to \mathbb{ R } ^ m \) is continuous if and only if for every open set \( U \subseteq \mathbb{ R } ^ m \), there is an open set \( V \subseteq \mathbb{ R } ^ n \) such that \[ f ^ { -1 } \left( U \right) = V \cap A. \]
Continuous Image of Compact Set is Compact
If \( A \subseteq \mathbb{ R } ^ n \) is compact and \( f: A \to \mathbb{ R } ^ m \) is continuous, then the image \( f \left( A \right) \subseteq \mathbb{ R } ^ m \) is compact.
Oscillation
Let \( f: A \to \mathbb{ R } \) be bounded, where \( A \subseteq \mathbb{ R } ^ n \). For \( a \in A \) and \( \delta \in \mathbb{ R } ^ + \), define the upper and lower local bounds \[ M \left( a, f, \delta \right) = \sup \left\{ f \left( x \right): x \in A \text{ and } \left\lvert x - a \right\rvert \lt \delta \right\} \] and \[ m \left( a, f, \delta \right) = \inf \left\{ f \left( x \right): x \in A \text{ and } \left\lvert x - a \right\rvert \lt \delta \right\}. \] The oscillation of \( f \) at \( a \) is \[ o \left( f, a \right) = \lim _ { \delta \to 0 } \left( M \left( a, f, \delta \right) - m \left( a, f, \delta \right) \right). \]
Continuity Oscillation Characterization
A bounded function \( f: A \to \mathbb{ R } \) is continuous at \( a \in A \) if and only if \[ o \left( f, a \right) = 0. \]
Large Oscillation Set is Closed
Let \( A \subseteq \mathbb{ R } ^ n \) be closed. If \( f: A \to \mathbb{ R } \) is bounded and \( \epsilon \in \mathbb{ R } ^ + \), then \[ \left\{ x \in A: o \left( f, x \right) \gt \epsilon \right\} \] is closed.

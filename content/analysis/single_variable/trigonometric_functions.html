Trigonometric Functions

Trigonometric Angle Addition Formulas
For any \( \alpha, \beta \in \mathbb{R} \): \[ \begin{align} \cos(\alpha + \beta) &= \cos(\alpha)\cos(\beta) - \sin(\alpha)\sin(\beta) \\ \sin(\alpha + \beta) &= \sin(\alpha)\cos(\beta) + \cos(\alpha)\sin(\beta) \end{align} \]
Sine is Odd and Cosine is Even
For any \( \alpha \in \mathbb{R} \): \[ \sin(-\alpha) = -\sin(\alpha) \qquad\text{and}\qquad \cos(-\alpha) = \cos(\alpha). \]
Sine is Zero on Zero to Pi Only at the Endpoints
If \( \theta \in \left[ 0, \pi \right] \), then \[ \sin \left( \theta \right) = 0 \iff \theta = 0 \text{ or } \theta = \pi \]

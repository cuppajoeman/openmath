Peano Natural Numbers
A Peano system consists of a set \( \mathbb{ N } \), an element \( 0 \in \mathbb{ N } \), and a function \( S: \mathbb{ N } \to \mathbb{ N } \), called the successor function, such that:
  • \( 0 \notin S \left( \mathbb{ N } \right) \).
  • \( S \) is injective.
  • If \( A \subseteq \mathbb{ N } \), \( 0 \in A \), and \( S \left( A \right) \subseteq A \), then \( A = \mathbb{ N } \).
The elements of \( \mathbb{ N } \) are called natural numbers.
Recursive Addition and Multiplication on Naturals
Addition on \( \mathbb{ N } \) is defined recursively by \[ a + 0 = a, \qquad a + S \left( b \right) = S \left( a + b \right). \] Multiplication on \( \mathbb{ N } \) is defined recursively by \[ a \cdot 0 = 0, \qquad a \cdot S \left( b \right) = a \cdot b + a. \]
Natural Numbers Satisfy Arithmetic Laws
The recursively defined addition and multiplication on \( \mathbb{ N } \) are associative and commutative, multiplication distributes over addition, \( 0 \) is an additive identity, and \( S \left( 0 \right) \) is a multiplicative identity.
Order on Natural Numbers
For \( a, b \in \mathbb{ N } \), define \[ a \le b \] if and only if there exists \( c \in \mathbb{ N } \) such that \( a + c = b \). Define \( a \lt b \) if \( a \le b \) and \( a \neq b \).
Natural Numbers are Well-Ordered
Every non-empty subset of \( \mathbb{ N } \) has a least element with respect to the order on natural numbers.
Integer Pair Relation
On \( \mathbb{ N } \times \mathbb{ N } \), define a relation by \[ \left( a, b \right) \sim \left( c, d \right) \iff a + d = c + b. \]
Integer Pair Relation is an Equivalence Relation
The integer pair relation is an equivalence relation.
Integers as Equivalence Classes
The integers, denoted \( \mathbb{ Z } \), are the equivalence classes of \( \mathbb{ N } \times \mathbb{ N } \) under the integer pair relation. The class of \( \left( a, b \right) \) represents the formal difference \( a - b \).
Addition and Multiplication of Integers
Let \( \left[ \left( a, b \right) \right], \left[ \left( c, d \right) \right] \in \mathbb{ Z } \). Define \[ \left[ \left( a, b \right) \right] + \left[ \left( c, d \right) \right] = \left[ \left( a + c, b + d \right) \right] \] and \[ \left[ \left( a, b \right) \right] \cdot \left[ \left( c, d \right) \right] = \left[ \left( ac + bd, ad + bc \right) \right]. \]
Integer Addition and Multiplication are Well-Defined
The operations of addition and multiplication of integers do not depend on the chosen representatives of the equivalence classes.
Negation of Integers
Let \( \left[ \left( a, b \right) \right] \in \mathbb{ Z } \). Define \[ - \left[ \left( a, b \right) \right] = \left[ \left( b, a \right) \right]. \]
Order on Integers
Let \( \left[ \left( a, b \right) \right], \left[ \left( c, d \right) \right] \in \mathbb{ Z } \). Define \[ \left[ \left( a, b \right) \right] \le \left[ \left( c, d \right) \right] \iff a + d \le c + b. \]
Order on Integers is Well-Defined
The order on integers does not depend on the chosen representatives of the equivalence classes.
Integers Form an Ordered Integral Domain
The integers \( \mathbb{ Z } \), with the operations and order defined above, form an ordered integral domain.
Rational Number Pair Relation
On \( \mathbb{ Z } \times \left( \mathbb{ Z } \setminus \left\{ 0 \right\} \right) \), define a relation by \[ \left( a, b \right) \sim \left( c, d \right) \iff ad = bc. \]
Rational Number Pair Relation is an Equivalence Relation
The rational number pair relation is an equivalence relation.
Rational Numbers as Equivalence Classes
The rational numbers, denoted \( \mathbb{ Q } \), are the equivalence classes of \( \mathbb{ Z } \times \left( \mathbb{ Z } \setminus \left\{ 0 \right\} \right) \) under the rational number pair relation. The class of \( \left( a, b \right) \) is denoted by \( \frac{a}{b} \).
Addition and Multiplication of Rationals
Let \( \frac{a}{b}, \frac{c}{d} \in \mathbb{ Q } \). Define \[ \frac{a}{b} + \frac{c}{d} = \frac{ad + bc}{bd} \] and \[ \frac{a}{b} \cdot \frac{c}{d} = \frac{ac}{bd}. \]
Rational Addition and Multiplication are Well-Defined
The operations of addition and multiplication of rationals do not depend on the chosen representatives of the equivalence classes.
Additive and Multiplicative Inverses of Rationals
Let \( \frac{a}{b} \in \mathbb{ Q } \). Define \[ - \frac{a}{b} = \frac{-a}{b}. \] If \( a \neq 0 \), define \[ \left( \frac{a}{b} \right) ^ { -1 } = \frac{b}{a}. \]
Order on Rationals
Let \( \frac{a}{b}, \frac{c}{d} \in \mathbb{ Q } \). Choose representatives with \( b \gt 0 \) and \( d \gt 0 \). Define \[ \frac{a}{b} \lt \frac{c}{d} \iff ad \lt bc. \]
Order on Rationals is Well-Defined
The order on rationals does not depend on the chosen positive-denominator representatives.
Rationals Form an Ordered Field
The rational numbers \( \mathbb{ Q } \), with the operations and order defined above, form an ordered field.
Dedekind Cut
Let \( A \subseteq \mathbb{ Q } \). The set \( A \) is a Dedekind cut if:
  • \( A \neq \emptyset \) and \( A \neq \mathbb{ Q } \).
  • If \( x \in A \), \( y \in \mathbb{ Q } \), and \( y \ge x \), then \( y \in A \).
  • If \( x \in A \), then there exists \( y \in A \) such that \( y \lt x \).
Rational Upper Rays are Dedekind Cuts
Let \( r \in \mathbb{ Q } \). Then \[ \left\{ x \in \mathbb{ Q } : x \gt r \right\} \] is a Dedekind cut.
Rational Cut
Let \( r \in \mathbb{ Q } \). The rational cut at \( r \), denoted \( D _ r \), is the Dedekind cut \[ D _ r = \left\{ x \in \mathbb{ Q } : x \gt r \right\}. \]
Irrational Cut
An irrational cut is a Dedekind cut that is not equal to \( D _ r \) for any \( r \in \mathbb{ Q } \).
Square Root Two Cut
The set \[ T = \left\{ x \in \mathbb{ Q } : x \gt 0 \land x ^ 2 \gt 2 \right\} \] is the Dedekind cut corresponding to the missing rational boundary \( \sqrt{2} \).
Complement of a Dedekind Cut
Let \( A \subseteq \mathbb{ Q } \) be a Dedekind cut. Then:
  • \( \mathbb{ Q } \setminus A = \left\{ x \in \mathbb{ Q } : x \lt a \text{ for all } a \in A \right\} \).
  • If \( x \in \mathbb{ Q } \setminus A \), \( y \in \mathbb{ Q } \), and \( y \le x \), then \( y \in \mathbb{ Q } \setminus A \).
Dedekind Cuts are Linearly Ordered by Containment
Let \( A, B \subseteq \mathbb{ Q } \) be Dedekind cuts. Then exactly one of the following holds: \[ A \subsetneq B, \qquad A = B, \qquad B \subsetneq A. \]
Union of Dedekind Cuts is a Dedekind Cut
Let \( \mathcal{ A } \) be a non-empty family of subsets of \( \mathbb{ Q } \). Suppose that every \( X \in \mathcal{ A } \) is a Dedekind cut. If \[ \bigcup _ { X \in \mathcal{ A } } X \neq \mathbb{ Q }, \] then \( \bigcup _ { X \in \mathcal{ A } } X \) is a Dedekind cut.
Dedekind Cut Operation Closure
Let \( A, B \subseteq \mathbb{ Q } \) be Dedekind cuts. Then:
  • \( \left\{ r \in \mathbb{ Q } : r = a + b \text{ for some } a \in A \text{ and } b \in B \right\} \) is a Dedekind cut.
  • \( \left\{ r \in \mathbb{ Q } : -r \lt c \text{ for some } c \in \mathbb{ Q } \setminus A \right\} \) is a Dedekind cut.
  • If \( 0 \in \mathbb{ Q } \setminus A \) and \( 0 \in \mathbb{ Q } \setminus B \), then \( \left\{ r \in \mathbb{ Q } : r = ab \text{ for some } a \in A \text{ and } b \in B \right\} \) is a Dedekind cut.
  • If there exists \( q \in \mathbb{ Q } \setminus A \) such that \( q \gt 0 \), then \( \left\{ r \in \mathbb{ Q } : r \gt 0 \text{ and } \frac{1}{r} \lt c \text{ for some } c \in \mathbb{ Q } \setminus A \right\} \) is a Dedekind cut.
Dedekind Cut Approximation
Let \( A \subseteq \mathbb{ Q } \) be a Dedekind cut.
  • If \( y \in \mathbb{ Q } \) and \( y \gt 0 \), then there exist \( u \in A \) and \( v \in \mathbb{ Q } \setminus A \) such that \( y = u - v \), and \( v \lt e \) for some \( e \in \mathbb{ Q } \setminus A \).
  • If \( y \in \mathbb{ Q } \), \( y \gt 1 \), and there exists \( q \in \mathbb{ Q } \setminus A \) such that \( q \gt 0 \), then there exist \( r \in A \) and \( s \in \mathbb{ Q } \setminus A \) such that \( s \gt 0 \), \( y \gt \frac{r}{s} \), and \( s \lt g \) for some \( g \in \mathbb{ Q } \setminus A \).
Real Numbers as Dedekind Cuts
The set of real numbers, denoted \( \mathbb{ R } \), is defined by \[ \mathbb{ R } = \left\{ A \subseteq \mathbb{ Q } : A \text{ is a } \text{Dedekind cut} \right\}. \]
Order on Dedekind Real Numbers
Let \( A, B \in \mathbb{ R } \), where \( \mathbb{ R } \) is defined as Dedekind cuts. Define \[ A \lt B \iff A \supsetneq B, \] and \[ A \le B \iff A \supseteq B. \]
Addition and Negation of Dedekind Reals
Let \( A, B \in \mathbb{ R } \), where \( \mathbb{ R } \) is defined as Dedekind cuts. Define \[ A + B = \left\{ r \in \mathbb{ Q } : r = a + b \text{ for some } a \in A \text{ and } b \in B \right\} \] and \[ -A = \left\{ r \in \mathbb{ Q } : -r \lt c \text{ for some } c \in \mathbb{ Q } \setminus A \right\}. \]
Order Comparison with Rational Cuts
Let \( A \in \mathbb{ R } \) and \( r \in \mathbb{ Q } \).
  • \( A \gt D _ r \) if and only if there exists \( q \in \mathbb{ Q } \setminus A \) such that \( q \gt r \).
  • \( A \ge D _ r \) if and only if \( r \in \mathbb{ Q } \setminus A \), if and only if \( a \gt r \) for all \( a \in A \).
  • If \( A \lt D _ 0 \), then \( -A \ge D _ 0 \).
Multiplication and Inverse of Dedekind Reals
Let \( A, B \in \mathbb{ R } \), where \( \mathbb{ R } \) is defined as Dedekind cuts. Multiplication is defined by cases:
  • If \( A \ge D _ 0 \) and \( B \ge D _ 0 \), then \( A \cdot B = \left\{ r \in \mathbb{ Q } : r = ab \text{ for some } a \in A \text{ and } b \in B \right\} \).
  • If \( A \lt D _ 0 \) and \( B \ge D _ 0 \), then \( A \cdot B = - \left( \left( -A \right) \cdot B \right) \).
  • If \( A \ge D _ 0 \) and \( B \lt D _ 0 \), then \( A \cdot B = - \left( A \cdot \left( -B \right) \right) \).
  • If \( A \lt D _ 0 \) and \( B \lt D _ 0 \), then \( A \cdot B = \left( -A \right) \cdot \left( -B \right) \).
For \( A \neq D _ 0 \), multiplicative inverse is defined by cases:
  • If \( A \gt D _ 0 \), then \( A ^ { -1 } = \left\{ r \in \mathbb{ Q } : r \gt 0 \text{ and } \frac{1}{r} \lt c \text{ for some } c \in \mathbb{ Q } \setminus A \right\} \).
  • If \( A \lt D _ 0 \), then \( A ^ { -1 } = - \left( \left( -A \right) ^ { -1 } \right) \).
Dedekind Reals Form an Ordered Field
Let \( A, B, C \in \mathbb{ R } \), where \( \mathbb{ R } \) is defined as Dedekind cuts. Then \( \mathbb{ R } \), with the operations and order defined above, satisfies the ordered-field laws: associativity and commutativity of addition and multiplication, additive and multiplicative identities \( D _ 0 \) and \( D _ 1 \), additive inverses, multiplicative inverses for non-zero elements, distributivity, trichotomy, compatibility of addition with order, and compatibility of multiplication by positive elements with order.
Greatest Lower Bound Property for Dedekind Reals
Let \( \mathcal{ A } \subseteq \mathbb{ R } \). If \( \mathcal{ A } \) is non-empty and bounded below, then \( \mathcal{ A } \) has a greatest lower bound.
Least Upper Bound Property for Dedekind Reals
Let \( \mathcal{ A } \subseteq \mathbb{ R } \). If \( \mathcal{ A } \) is non-empty and bounded above, then \( \mathcal{ A } \) has a least upper bound.
Rationals Embed into Dedekind Reals
Define \( i: \mathbb{ Q } \to \mathbb{ R } \) by \[ i \left( r \right) = D _ r \] for all \( r \in \mathbb{ Q } \). Then:
  • The function \( i: \mathbb{ Q } \to \mathbb{ R } \) is injective.
  • \( i \left( 0 \right) = D _ 0 \) and \( i \left( 1 \right) = D _ 1 \).
  • For \( r, s \in \mathbb{ Q } \), \( i \left( r + s \right) = i \left( r \right) + i \left( s \right) \).
  • For \( r \in \mathbb{ Q } \), \( i \left( -r \right) = - i \left( r \right) \).
  • For \( r, s \in \mathbb{ Q } \), \( i \left( rs \right) = i \left( r \right) i \left( s \right) \).
  • If \( r \neq 0 \), then \( i \left( r ^ { -1 } \right) = i \left( r \right) ^ { -1 } \).
  • \( r \lt s \) if and only if \( i \left( r \right) \lt i \left( s \right) \).

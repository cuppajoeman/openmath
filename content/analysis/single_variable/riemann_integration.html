Partition of a Closed Interval
Suppose that \( a, b \in \mathbb{ R } \) and \( a \lt b \). A partition of \( [ a, b ] \) is a finite tuple \[ P = \left( x _ 0, x _ 1, \ldots, x _ n \right) \] such that \[ a = x _ 0 \lt x _ 1 \lt \cdots \lt x _ n = b. \]
Infimum of a Function on a Set
Suppose that \( f: X \to \mathbb{ R } \) and \( A \subseteq X \). Define the infimum of \( f \) on \( A \) by \[ \inf _ A f := \inf \left\{ f \left( x \right): x \in A \right\} \]
Supremum of a Function on a Set
Suppose that \( f: X \to \mathbb{ R } \) and \( A \subseteq X \). Define the supremum of \( f \) on \( A \) by \[ \sup _ A f := \sup \left\{ f \left( x \right): x \in A \right\}. \]
Lower Riemann Sum
Suppose that \( f: [ a, b ] \to \mathbb{ R } \) is a bounded function, and let \[ P = \left( x _ 0, x _ 1, \ldots, x _ n \right) \] be a partition of \( [ a, b ] \). The lower Riemann sum of \( f \) over \( P \) is \[ L \left( f, P, [ a, b ] \right) := \sum _ { j = 1 } ^ n \left( x _ j - x _ { j - 1 } \right) \inf _ { [ x _ { j - 1 }, x _ j ] } f, \] where \( \inf _ { [ x _ { j - 1 }, x _ j ] } f \) is the infimum of \( f \) on \( [ x _ { j - 1 }, x _ j ] \).
Upper Riemann Sum
Suppose that \( f: [ a, b ] \to \mathbb{ R } \) is a bounded function, and let \[ P = \left( x _ 0, x _ 1, \ldots, x _ n \right) \] be a partition of \( [ a, b ] \). The upper Riemann sum of \( f \) over \( P \) is \[ U \left( f, P, [ a, b ] \right) := \sum _ { j = 1 } ^ n \left( x _ j - x _ { j - 1 } \right) \sup _ { [ x _ { j - 1 }, x _ j ] } f, \] where \( \sup _ { [ x _ { j - 1 }, x _ j ] } f \) is the supremum of \( f \) on \( [ x _ { j - 1 }, x _ j ] \).
Refinement of a Riemann Partition
Suppose that \( P \) and \( P' \) are partitions of \( [ a, b ] \). We say that \( P' \) is a refinement of \( P \) if every point of \( P \) is also a point of \( P' \).
Riemann Sum Inequalities under Refinement
Suppose that \( f: [ a, b ] \to \mathbb{ R } \) is bounded, and suppose that \( P' \) is a refinement of \( P \). Then \[ L \left( f, P, [ a, b ] \right) \le L \left( f, P', [ a, b ] \right) \le U \left( f, P', [ a, b ] \right) \le U \left( f, P, [ a, b ] \right). \]
Any Lower Riemann Sum is Below Any Upper Riemann Sum
Suppose that \( f: [ a, b ] \to \mathbb{ R } \) is bounded, and let \( P \) and \( Q \) be partitions of \( [ a, b ] \). Then \[ L \left( f, P, [ a, b ] \right) \le U \left( f, Q, [ a, b ] \right). \]
Lower Riemann Integral
Suppose that \( f: [ a, b ] \to \mathbb{ R } \) is bounded. The lower Riemann integral of \( f \) over \( [ a, b ] \) is \[ L \left( f, [ a, b ] \right) := \sup _ P L \left( f, P, [ a, b ] \right) \] where \( P \) ranges over all partitions of \( [ a, b ] \).
Upper Riemann Integral
Suppose that \( f: [ a, b ] \to \mathbb{ R } \) is bounded. The upper Riemann integral of \( f \) over \( [ a, b ] \) is \[ U \left( f, [ a, b ] \right) := \inf _ P U \left( f, P, [ a, b ] \right), \] where \( P \) ranges over all partitions of \( [ a, b ] \).
Lower Riemann Integral is Below Upper Riemann Integral
Suppose that \( f: [ a, b ] \to \mathbb{ R } \) is bounded. Then \[ L \left( f, [ a, b ] \right) \le U \left( f, [ a, b ] \right). \]
Riemann Integrable Function
Suppose that \( f: [ a, b ] \to \mathbb{ R } \) is bounded. We say that \( f \) is Riemann integrable on \( [ a, b ] \) if \[ L \left( f, [ a, b ] \right) = U \left( f, [ a, b ] \right). \]
Riemann Integral
Suppose that \( f: [ a, b ] \to \mathbb{ R } \) is Riemann integrable. Its Riemann integral over \( [ a, b ] \) is the common value \[ \int _ a ^ b f := L \left( f, [ a, b ] \right) = U \left( f, [ a, b ] \right). \]
Continuous Functions on Closed Bounded Intervals are Riemann Integrable
Suppose that \( a, b \in \mathbb{ R } \) and \( a \lt b \). If \( f: [ a, b ] \to \mathbb{ R } \) is continuous, then \( f \) is Riemann integrable on \( [ a, b ] \).
Bounds on a Riemann Integral
Suppose that \( f: [ a, b ] \to \mathbb{ R } \) is Riemann integrable. Then \[ \left( b - a \right) \inf _ { [ a, b ] } f \le \int _ a ^ b f \le \left( b - a \right) \sup _ { [ a, b ] } f. \]
Interchanging Riemann Integral and Pointwise Limit
Suppose that \( a, b, M \in \mathbb{ R } \), \( a \lt b \), and \( f _ 1, f _ 2, \ldots \) is a sequence of Riemann integrable functions \( f _ k: [ a, b ] \to \mathbb{ R } \). Suppose that \[ \left\lvert f _ k \left( x \right) \right\rvert \le M \] for every \( k \in \mathbb{ Z } ^ + \) and every \( x \in [ a, b ] \), and suppose that \[ f \left( x \right) := \lim _ { k \to \infty } f _ k \left( x \right) \] exists for every \( x \in [ a, b ] \). If \( f \) is Riemann integrable on \( [ a, b ] \), then \[ \int _ a ^ b f = \lim _ { k \to \infty } \int _ a ^ b f _ k. \]

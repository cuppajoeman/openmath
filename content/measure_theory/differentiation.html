Average Value over an Interval
Suppose that \( f \in L ^ 1 \left( \mathbb{ R } \right) \) and \( I \subseteq \mathbb{ R } \) is an interval with positive finite Lebesgue measure. The average value of \( f \) over \( I \) is \[ \frac{1}{\lambda \left( I \right)} \int _ I f \, d\lambda . \]
Lebesgue Point
Suppose that \( f \in L ^ 1 _ { \operatorname{ loc } } \left( \mathbb{ R } \right) \). A point \( a \in \mathbb{ R } \) is a Lebesgue point of \( f \) if \[ \lim _ { r \downarrow 0 } \frac{1}{2r} \int _ { a-r } ^ { a+r } \left\lvert f \left( x \right) - f \left( a \right) \right\rvert \, d\lambda \left( x \right) = 0. \]
Lebesgue Differentiation Theorem
Suppose that \( f \in L ^ 1 _ { \operatorname{ loc } } \left( \mathbb{ R } \right) \). Then almost every \( a \in \mathbb{ R } \) is a Lebesgue point of \( f \).
Differentiation of Indefinite Integrals
Suppose that \( f \in L ^ 1 \left( \mathbb{ R } \right) \) and \[ F \left( x \right) := \int _ { -\infty } ^ x f \, d\lambda . \] Then \( F ^ \prime \left( x \right) = f \left( x \right) \) for almost every \( x \in \mathbb{ R } \).
Hardy-Littlewood Maximal Function
Suppose that \( f \in L ^ 1 _ { \operatorname{ loc } } \left( \mathbb{ R } \right) \). The Hardy-Littlewood maximal function of \( f \) is \[ M f \left( x \right) := \sup _ { r \in \mathbb{ R } ^ + } \frac{1}{2r} \int _ { x-r } ^ { x+r } \left\lvert f \right\rvert \, d\lambda . \]
Hardy-Littlewood Maximal Inequality
Suppose that \( f \in L ^ 1 \left( \mathbb{ R } \right) \) and \( \alpha \in \mathbb{ R } ^ + \). Then \[ \lambda \left( \left\{ x \in \mathbb{ R } : M f \left( x \right) \gt \alpha \right\} \right) \le \frac{3}{\alpha} \lVert f \rVert _ 1 . \]
Absolutely Continuous Function on an Interval
Suppose that \( a \lt b \). A function \( F: [a,b] \to \mathbb{ R } \) is absolutely continuous if for every \( \epsilon \in \mathbb{ R } ^ + \), there exists \( \delta \in \mathbb{ R } ^ + \) such that for every finite pairwise disjoint collection of intervals \( (a _ k, b _ k) \subseteq [a,b] \), \[ \sum _ k \left( b _ k - a _ k \right) \lt \delta \implies \sum _ k \left\lvert F \left( b _ k \right) - F \left( a _ k \right) \right\rvert \lt \epsilon . \]
Absolutely Continuous Functions are Integrals of Their Derivatives
Suppose that \( F: [a,b] \to \mathbb{ R } \) is absolutely continuous. Then \( F ^ \prime \) exists almost everywhere, \( F ^ \prime \in L ^ 1 \left( [a,b] \right) \), and \[ F \left( y \right) - F \left( x \right) = \int _ x ^ y F ^ \prime \, d\lambda \] whenever \( a \le x \le y \le b \).

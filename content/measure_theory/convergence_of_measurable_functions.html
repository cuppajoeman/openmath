Pointwise Convergence of Functions
Suppose that \( X \) is a set, \( f _ 1, f _ 2, \ldots: X \to \mathbb{ R } \) is a sequence of functions, and \( f: X \to \mathbb{ R } \). We say that \( f _ 1, f _ 2, \ldots \) converges pointwise on \( X \) to \( f \) if \[ \lim _ { k \to \infty } f _ k \left( x \right) = f \left( x \right) \] for every \( x \in X \).
Uniform Convergence of Functions
Suppose that \( X \) is a set, \( f _ 1, f _ 2, \ldots: X \to \mathbb{ R } \) is a sequence of functions, and \( f: X \to \mathbb{ R } \). We say that \( f _ 1, f _ 2, \ldots \) converges uniformly on \( X \) to \( f \) if for every \( \epsilon \in \mathbb{ R } ^ + \), there exists \( n \in \mathbb{ Z } ^ + \) such that \[ \left\lvert f _ k \left( x \right) - f \left( x \right) \right\rvert \lt \epsilon \] for every \( k \ge n \) and every \( x \in X \).
Uniform Limit of Continuous Functions is Continuous
Suppose that \( B \subseteq \mathbb{ R } \), \( f _ 1, f _ 2, \ldots: B \to \mathbb{ R } \) converges uniformly on \( B \) to \( f: B \to \mathbb{ R } \), and \( b \in B \). If each \( f _ k \) is continuous at \( b \), then \( f \) is continuous at \( b \).

Let \( \epsilon \in \mathbb{ R } ^ + \). By uniform convergence, choose \( N \) such that \( \left\lvert f _ k \left( x \right)-f\left( x \right)\right\rvert \lt \epsilon/3 \) for every \( k \ge N \) and every \( x \in B \). Since \( f _ N \) is continuous at \( b \), choose \( \delta \gt 0 \) such that \( \left\lvert f _ N \left( x \right)-f _ N \left( b \right)\right\rvert \lt \epsilon/3 \) whenever \( x \in B \) and \( \left\lvert x-b\right\rvert \lt \delta \). The triangle inequality gives \( \left\lvert f\left( x\right)-f\left( b\right)\right\rvert \lt \epsilon \), so \( f \) is continuous at \( b \).

Egorov's Theorem
Suppose that \( \left( X, \mathcal{ S }, \mu \right) \) is a measure space with \( \mu \left( X \right) \lt \infty \). If \( f _ 1, f _ 2, \ldots: X \to \mathbb{ R } \) are \( \mathcal{ S } \)-measurable and converge pointwise on \( X \) to \( f: X \to \mathbb{ R } \), then for every \( \epsilon \in \mathbb{ R } ^ + \), there exists \( E \in \mathcal{ S } \) such that \[ \mu \left( X \setminus E \right) \lt \epsilon \] and \( f _ 1, f _ 2, \ldots \) converges uniformly to \( f \) on \( E \).

For \( m,n \in \mathbb{ N } _ 1 \), set \[ E _ { m,n } := \left\{ x \in X : \left\lvert f _ j \left( x \right)-f\left( x \right)\right\rvert \lt \frac{1}{m} \text{ for every } j \ge n \right\}. \] These sets are measurable, increase in \( n \), and their union is \( X \) by pointwise convergence. Since \( \mu \left( X \right)\lt \infty \), the measure of an increasing union lets us choose \( n_m \) such that \( \mu \left( X \setminus E _ { m,n_m } \right)\lt \epsilon/2 ^ m \). Let \( E=\bigcap _ { m=1 } ^ \infty E _ { m,n_m } \). Then \( \mu \left( X\setminus E\right)\lt \epsilon \) by countable subadditivity, and the definition of the sets gives uniform convergence on \( E \).

Simple Function
A function is simple if it takes only finitely many values.
Approximation by Simple Functions
Suppose that \( \left( X, \mathcal{ S } \right) \) is a measurable space and \( f: X \to [ -\infty, \infty ] \) is \( \mathcal{ S } \)-measurable. Then there exists a sequence \( f _ 1, f _ 2, \ldots: X \to \mathbb{ R } \) such that
  • each \( f _ k \) is simple and \( \mathcal{ S } \)-measurable,
  • \( \left\lvert f _ k \left( x \right) \right\rvert \le \left\lvert f _ { k + 1 } \left( x \right) \right\rvert \le \left\lvert f \left( x \right) \right\rvert \) for every \( k \in \mathbb{ Z } ^ + \) and every \( x \in X \),
  • \( \lim _ { k \to \infty } f _ k \left( x \right) = f \left( x \right) \) for every \( x \in X \),
  • if \( f \) is bounded, then \( f _ 1, f _ 2, \ldots \) converges uniformly to \( f \) on \( X \).

For \( f \ge 0 \), divide the range into dyadic intervals of length \( 2^{-k} \) up to height \( k \), and let \( f _ k \left( x \right) \) be the left endpoint of the interval containing \( f \left( x \right) \), truncated at height \( k \). For general \( f \), apply this construction to the positive and negative parts and subtract. Measurability of the level sets follows from the measurability of \( f \), each \( f _ k \) has finite range, and the dyadic mesh tends to \( 0 \), giving pointwise convergence. If \( f \) is bounded, the truncation height eventually exceeds the bound, so the dyadic mesh gives uniform convergence.

Luzin's Theorem
Suppose that \( g: \mathbb{ R } \to \mathbb{ R } \) is Borel measurable. For every \( \epsilon \in \mathbb{ R } ^ + \), there exists a closed set \( F \subseteq \mathbb{ R } \) such that \[ \operatorname{omes} \left( \mathbb{ R } \setminus F \right) \lt \epsilon \] and \( g \vert _ F \) is continuous on \( F \).

First prove the claim for simple Borel functions by choosing a closed subset of each level set whose complement has arbitrarily small outer measure, using the regularity characterization of Lebesgue measurable sets. On the union of those closed pieces, the simple function is continuous. For a general Borel measurable \( g \), use approximation by simple functions and Egorov's theorem to find a closed set outside a set of outer measure less than \( \epsilon \) on which the simple approximants converge uniformly. The uniform limit of continuous functions is continuous, so \( g \vert _ F \) is continuous.

Continuous Extensions from Closed Subsets of the Real Line
Suppose that \( F \subseteq \mathbb{ R } \) is closed and \( g: F \to \mathbb{ R } \) is continuous. Then there exists a continuous function \( h: \mathbb{ R } \to \mathbb{ R } \) such that \( h \vert _ F = g \).

The complement of a closed subset of \( \mathbb{ R } \) is a countable union of disjoint open intervals. On each bounded complementary interval \( \left( a,b\right) \), define \( h \) by linear interpolation between the endpoint values \( g\left( a\right) \) and \( g\left( b\right) \); on unbounded complementary intervals, extend constantly from the finite endpoint. Set \( h=g \) on \( F \). The endpoint definitions and the continuity of \( g \) on \( F \) make \( h \) continuous on all of \( \mathbb{ R } \).

Luzin's Theorem, Extension Version
Suppose that \( E \subseteq \mathbb{ R } \) and \( g: E \to \mathbb{ R } \) is Borel measurable. For every \( \epsilon \in \mathbb{ R } ^ + \), there exists a closed set \( F \subseteq E \) and a continuous function \( h: \mathbb{ R } \to \mathbb{ R } \) such that \[ \operatorname{omes} \left( E \setminus F \right) \lt \epsilon \qquad \text{and} \qquad h \vert _ F = g \vert _ F. \]

Extend \( g \) from \( E \) to a Borel measurable function on \( \mathbb{ R } \), for example by assigning value \( 0 \) off \( E \). By Luzin's theorem, choose a closed set \( K \subseteq \mathbb{ R } \) such that the exceptional set has outer measure less than \( \epsilon \) and the extension is continuous on \( K \). Put \( F=E\cap K \), using the regularity characterization to replace it by a closed subset of \( E \) without increasing the exceptional outer measure. Then apply continuous extension from closed subsets to obtain \( h \) with \( h\vert _ F=g\vert _ F \).

Lebesgue Measurable Function
Suppose that \( A \subseteq \mathbb{ R } \). A function \( f: A \to \mathbb{ R } \) is Lebesgue measurable if \( f ^ { -1 } \left( B \right) \) is a Lebesgue measurable set for every Borel set \( B \subseteq \mathbb{ R } \).
Every Lebesgue Measurable Function is Almost Borel Measurable
Suppose that \( f: \mathbb{ R } \to \mathbb{ R } \) is Lebesgue measurable. Then there exists a Borel measurable function \( g: \mathbb{ R } \to \mathbb{ R } \) such that \[ \operatorname{omes} \left( \left\{ x \in \mathbb{ R } : g \left( x \right) \neq f \left( x \right) \right\} \right) = 0. \]

By the definition of Lebesgue measurable set, each inverse image \( f ^ {-1} \left( B \right) \) differs by a null set from a Borel set. Applying this to the inverse images of intervals with rational endpoints and using countable unions, construct a Borel measurable \( g \) that agrees with \( f \) outside the union of countably many null exceptional sets. By countable subadditivity, that union has outer measure \( 0 \), which is exactly the displayed conclusion.

Sigma Algebra
Suppose that \( X \) is a set. A collection \( \mathcal{ S } \subseteq \mathcal{ P } \left( X \right) \) is a \( \sigma \)-algebra on \( X \) if
  • \( \emptyset \in \mathcal{ S } \),
  • if \( E \in \mathcal{ S } \), then \( X \setminus E \in \mathcal{ S } \),
  • if \( E _ 1, E _ 2, \ldots \in \mathcal{ S } \) is a sequence, then \( \bigcup _ { k = 1 } ^ \infty E _ k \in \mathcal{ S } \).
Sigma Algebras are Closed under Standard Set Operations
Suppose that \( \mathcal{ S } \) is a \( \sigma \)-algebra on \( X \). Then
  • \( X \in \mathcal{ S } \),
  • if \( D, E \in \mathcal{ S } \), then \( D \cup E \in \mathcal{ S } \), \( D \cap E \in \mathcal{ S } \), and \( D \setminus E \in \mathcal{ S } \),
  • if \( E _ 1, E _ 2, \ldots \in \mathcal{ S } \), then \( \bigcap _ { k = 1 } ^ \infty E _ k \in \mathcal{ S } \).

Since \( \emptyset \in \mathcal{ S } \) and \( \mathcal{ S } \) is closed under complements, \( X = X \setminus \emptyset \in \mathcal{ S } \) by the definition of a \( \sigma \)-algebra.

If \( D,E \in \mathcal{ S } \), then \( D \cup E \in \mathcal{ S } \) by closure under countable unions, using \( \emptyset \) for all remaining terms. Also, \[ D \cap E = X \setminus \left( \left( X \setminus D \right) \cup \left( X \setminus E \right) \right), \] so \( D \cap E \in \mathcal{ S } \). Finally, \[ D \setminus E = D \cap \left( X \setminus E \right), \] so \( D \setminus E \in \mathcal{ S } \).

If \( E _ 1,E _ 2,\ldots \in \mathcal{ S } \), then by De Morgan's law, \[ \bigcap _ { k = 1 } ^ \infty E _ k = X \setminus \bigcup _ { k = 1 } ^ \infty \left( X \setminus E _ k \right). \] The right side belongs to \( \mathcal{ S } \), again by the defining closure properties of a \( \sigma \)-algebra.

Measurable Space
A measurable space is an ordered pair \( \left( X, \mathcal{ S } \right) \), where \( X \) is a set and \( \mathcal{ S } \) is a \( \sigma \)-algebra on \( X \).
Measurable Set
Suppose that \( \left( X, \mathcal{ S } \right) \) is a measurable space. A set \( E \subseteq X \) is \( \mathcal{ S } \)-measurable if \( E \in \mathcal{ S } \).
Smallest Sigma Algebra Containing a Collection
Suppose that \( X \) is a set and \( \mathcal{ A } \subseteq \mathcal{ P } \left( X \right) \). The intersection of all \( \sigma \)-algebras on \( X \) that contain \( \mathcal{ A } \) is a \( \sigma \)-algebra on \( X \).

Let \( \mathcal{ T } \) be the intersection of all \( \sigma \)-algebras on \( X \) that contain \( \mathcal{ A } \). This collection is nonempty because \( \mathcal{ P } \left( X \right) \) itself is a \( \sigma \)-algebra containing \( \mathcal{ A } \).

Every \( \sigma \)-algebra in the intersection contains \( \emptyset \), so \( \emptyset \in \mathcal{ T } \). If \( E \in \mathcal{ T } \), then \( E \) belongs to every \( \sigma \)-algebra in the intersection, so \( X \setminus E \) belongs to every one of them, and hence \( X \setminus E \in \mathcal{ T } \). If \( E _ 1,E _ 2,\ldots \in \mathcal{ T } \), then each \( E _ k \) belongs to every \( \sigma \)-algebra in the intersection, so \( \bigcup _ { k = 1 } ^ \infty E _ k \) belongs to every one of them, and hence belongs to \( \mathcal{ T } \). Thus \( \mathcal{ T } \) is a \( \sigma \)-algebra on \( X \).

Borel \( \sigma \)-Algebra
The Borel \( \sigma \)-algebra on \( \mathbb{ R } \) is the smallest \( \sigma \)-algebra on \( \mathbb{ R } \) that contains every open subset of \( \mathbb{ R } \).
Borel Set
A Borel set is an element of the Borel \( \sigma \)-algebra.
Singletons are Borel Sets
For every \( x \in \mathbb{ R } \), the singleton set \( \left\{ x \right\} \) is a Borel set.
The set \[ \mathbb{ R } \setminus \left\{ x \right\} = \left( -\infty,x \right) \cup \left( x,\infty \right) \] is open. Since the Borel \( \sigma \)-algebra contains every open subset of \( \mathbb{ R } \), we have \( \mathbb{ R } \setminus \left\{ x \right\} \in \mathcal{ B } \). Since a \( \sigma \)-algebra is closed under set difference from the ambient set, \[ \left\{ x \right\} = \mathbb{ R } \setminus \left( \mathbb{ R } \setminus \left\{ x \right\} \right) \in \mathcal{ B }. \] Therefore \( \left\{ x \right\} \) is a Borel set.
Inverse Image
Suppose that \( f: X \to Y \) is a function and \( A \subseteq Y \). The inverse image of \( A \) under \( f \) is \[ f ^ { -1 } \left( A \right) := \left\{ x \in X : f \left( x \right) \in A \right\}. \]
Algebra of Inverse Images
Suppose that \( f: X \to Y \). Then
  • \( f ^ { -1 } \left( Y \setminus A \right) = X \setminus f ^ { -1 } \left( A \right) \) for every \( A \subseteq Y \),
  • \( f ^ { -1 } \left( \bigcup _ { A \in \mathcal{ A } } A \right) = \bigcup _ { A \in \mathcal{ A } } f ^ { -1 } \left( A \right) \) for every collection \( \mathcal{ A } \subseteq \mathcal{ P } \left( Y \right) \),
  • \( f ^ { -1 } \left( \bigcap _ { A \in \mathcal{ A } } A \right) = \bigcap _ { A \in \mathcal{ A } } f ^ { -1 } \left( A \right) \) for every collection \( \mathcal{ A } \subseteq \mathcal{ P } \left( Y \right) \).

We prove each identity by element chasing from the definition of inverse image.

For the complement identity, \( x \in f ^ { -1 } \left( Y \setminus A \right) \) if and only if \( f \left( x \right) \in Y \setminus A \), if and only if \( f \left( x \right) \notin A \), if and only if \( x \in X \setminus f ^ { -1 } \left( A \right) \).

For unions, \[ x \in f ^ { -1 } \left( \bigcup _ { A \in \mathcal{ A } } A \right) \] if and only if \( f \left( x \right) \in A \) for some \( A \in \mathcal{ A } \), if and only if \( x \in f ^ { -1 } \left( A \right) \) for some \( A \in \mathcal{ A } \), if and only if \[ x \in \bigcup _ { A \in \mathcal{ A } } f ^ { -1 } \left( A \right). \] The proof for intersections is the same with "for some" replaced by "for every".

Inverse Image of a Composition
Suppose that \( f: X \to Y \), \( g: Y \to W \), and \( A \subseteq W \). Then \[ \left( g \circ f \right) ^ { -1 } \left( A \right) = f ^ { -1 } \left( g ^ { -1 } \left( A \right) \right). \]

For \( x \in X \), we have \[ x \in \left( g \circ f \right) ^ { -1 } \left( A \right) \] if and only if \( g \left( f \left( x \right) \right) \in A \), if and only if \( f \left( x \right) \in g ^ { -1 } \left( A \right) \), if and only if \[ x \in f ^ { -1 } \left( g ^ { -1 } \left( A \right) \right). \] Therefore the two sets are equal by set equality by elements.

Measurable Function
Suppose that \( \left( X, \mathcal{ S } \right) \) is a measurable space. A function \( f: X \to \mathbb{ R } \) is \( \mathcal{ S } \)-measurable if \[ f ^ { -1 } \left( B \right) \in \mathcal{ S } \] for every Borel set \( B \subseteq \mathbb{ R } \).
Characteristic Function
Suppose that \( E \subseteq X \). The characteristic function of \( E \) is the function \( \chi _ E: X \to \mathbb{ R } \) defined by \[ \chi _ E \left( x \right) = \begin{cases} 1 & x \in E, \\ 0 & x \notin E. \end{cases} \]
Ray Test for Measurability
Suppose that \( \left( X, \mathcal{ S } \right) \) is a measurable space and \( f: X \to \mathbb{ R } \). If \[ f ^ { -1 } \left( \left( a, \infty \right) \right) \in \mathcal{ S } \] for every \( a \in \mathbb{ R } \), then \( f \) is \( \mathcal{ S } \)-measurable.

Let \[ \mathcal{ T} := \left\{ B \subseteq \mathbb{ R } : f ^ { -1 } \left( B \right) \in \mathcal{ S } \right\}. \] By the algebra of inverse images and the definition of a \( \sigma \)-algebra, the collection \( \mathcal{ T} \) is a \( \sigma \)-algebra on \( \mathbb{ R } \). By hypothesis, \( \left( a,\infty \right) \in \mathcal{ T} \) for every \( a \in \mathbb{ R } \). The \( \sigma \)-algebra generated by these rays is the Borel \( \sigma \)-algebra, so every Borel set belongs to \( \mathcal{ T} \). Hence \( f \) is \( \mathcal{ S } \)-measurable.

Borel Measurable Function
Suppose that \( X \subseteq \mathbb{ R } \). A function \( f: X \to \mathbb{ R } \) is Borel measurable if \( f ^ { -1 } \left( B \right) \) is a Borel set for every Borel set \( B \subseteq \mathbb{ R } \).
Continuous Functions are Borel Measurable
Every continuous real-valued function whose domain is a Borel subset of \( \mathbb{ R } \) is Borel measurable.

Let \( X \subseteq \mathbb{ R } \) be a Borel set, and let \( f: X \to \mathbb{ R } \) be continuous. By the ray test for measurability, it is enough to prove that \( f ^ { -1 } \left( \left( a,\infty \right) \right) \) is Borel for each \( a \in \mathbb{ R } \).

If \( x \in f ^ { -1 } \left( \left( a,\infty \right) \right) \), then by continuity, there is \( \delta _ x \in \mathbb{ R } ^ + \) such that \( f \left( y \right) \gt a \) whenever \( y \in X \cap \left( x - \delta _ x, x + \delta _ x \right) \). Therefore \[ f ^ { -1 } \left( \left( a,\infty \right) \right) = X \cap \bigcup _ { x \in f ^ { -1 } \left( \left( a,\infty \right) \right) } \left( x - \delta _ x, x + \delta _ x \right). \] The union is open in \( \mathbb{ R } \), hence Borel, and its intersection with the Borel set \( X \) is Borel by closure of \( \sigma \)-algebras under intersections. Thus \( f \) is Borel measurable.

Increasing Function
Suppose that \( X \subseteq \mathbb{ R } \) and \( f: X \to \mathbb{ R } \). The function \( f \) is increasing if \( f \left( x \right) \le f \left( y \right) \) whenever \( x, y \in X \) and \( x \lt y \).
Strictly Increasing Function
Suppose that \( X \subseteq \mathbb{ R } \) and \( f: X \to \mathbb{ R } \). The function \( f \) is strictly increasing if \( f \left( x \right) \lt f \left( y \right) \) whenever \( x, y \in X \) and \( x \lt y \).
Increasing Functions are Borel Measurable
Every increasing real-valued function whose domain is a Borel subset of \( \mathbb{ R } \) is Borel measurable.

Let \( X \subseteq \mathbb{ R } \) be Borel and let \( f: X \to \mathbb{ R } \) be increasing. By the ray test, it is enough to show that \( f ^ { -1 } \left( \left( a,\infty \right) \right) \) is Borel for each \( a \in \mathbb{ R } \).

If this inverse image is empty, it is Borel. Otherwise let \( b = \inf f ^ { -1 } \left( \left( a,\infty \right) \right) \). Since \( f \) is increasing, the set \( f ^ { -1 } \left( \left( a,\infty \right) \right) \) is either \( X \cap \left( b,\infty \right) \) or \( X \cap [ b,\infty ) \). Both are Borel because \( X \) is Borel and the Borel \( \sigma \)-algebra is closed under intersections. Therefore \( f \) is Borel measurable.

Composition of Measurable Functions
Suppose that \( \left( X, \mathcal{ S } \right) \) is a measurable space and \( f: X \to \mathbb{ R } \) is \( \mathcal{ S } \)-measurable. If \( g \) is a Borel measurable real-valued function on a subset of \( \mathbb{ R } \) that contains the range of \( f \), then \( g \circ f \) is \( \mathcal{ S } \)-measurable.

Let \( B \subseteq \mathbb{ R } \) be a Borel set. By the inverse image formula for a composition, \[ \left( g \circ f \right) ^ { -1 } \left( B \right) = f ^ { -1 } \left( g ^ { -1 } \left( B \right) \right). \] Since \( g \) is Borel measurable, \( g ^ { -1 } \left( B \right) \) is Borel. Since \( f \) is \( \mathcal{ S } \)-measurable, the right side belongs to \( \mathcal{ S } \). Thus \( g \circ f \) is \( \mathcal{ S } \)-measurable.

Algebraic Operations with Measurable Functions
Suppose that \( \left( X, \mathcal{ S } \right) \) is a measurable space and \( f, g: X \to \mathbb{ R } \) are \( \mathcal{ S } \)-measurable. Then \( f + g \), \( f - g \), and \( fg \) are \( \mathcal{ S } \)-measurable. If \( g \left( x \right) \neq 0 \) for every \( x \in X \), then \( f / g \) is \( \mathcal{ S } \)-measurable.

For addition, fix \( a \in \mathbb{ R } \). We have \[ \left( f + g \right) ^ { -1 } \left( \left( a,\infty \right) \right) = \bigcup _ { r \in \mathbb{ Q } } \left( f ^ { -1 } \left( \left( r,\infty \right) \right) \cap g ^ { -1 } \left( \left( a-r,\infty \right) \right) \right). \] The equality follows because if \( f(x)+g(x)\gt a \), then some rational \( r \) lies between \( a-g(x) \) and \( f(x) \). The right side belongs to \( \mathcal{ S } \) because \( f \) and \( g \) are measurable, \( \mathbb{ Q } \) is countable, and \( \mathcal{ S } \) is a \( \sigma \)-algebra. Thus \( f+g \) is measurable by the ray test.

The function \( -g \) is measurable because it is the composition of \( g \) with the continuous function \( x \mapsto -x \), using continuous functions are Borel measurable and composition of measurable functions. Hence \( f-g=f+(-g) \) is measurable.

The square of a measurable function is measurable for the same composition reason, using \( x \mapsto x^2 \). Since \[ fg = \frac{ \left( f + g \right) ^ 2 - f ^ 2 - g ^ 2 }{2}, \] the product \( fg \) is measurable. If \( g(x)\neq 0 \) for all \( x \), then \( 1/g \) is the composition of \( g \) with the continuous function \( x \mapsto 1/x \) on \( \mathbb{ R } \setminus \left\{ 0 \right\} \), so \( 1/g \) is measurable. Therefore \( f/g=f\cdot (1/g) \) is measurable.

Pointwise Limit of Measurable Functions is Measurable
Suppose that \( \left( X, \mathcal{ S } \right) \) is a measurable space and \( f _ 1, f _ 2, \ldots: X \to \mathbb{ R } \) are \( \mathcal{ S } \)-measurable. If \( f \left( x \right) := \lim _ { k \to \infty } f _ k \left( x \right) \) exists for every \( x \in X \), then \( f \) is \( \mathcal{ S } \)-measurable.

Let \( a \in \mathbb{ R } \). We claim that \[ f ^ { -1 } \left( \left( a,\infty \right) \right) = \bigcup _ { j = 1 } ^ \infty \bigcup _ { m = 1 } ^ \infty \bigcap _ { k = m } ^ \infty f _ k ^ { -1 } \left( \left( a + \frac{1}{j},\infty \right) \right). \] If \( f(x)\gt a \), choose \( j \) with \( f(x)\gt a+1/j \). Since \( f_k(x)\to f(x) \), eventually \( f_k(x)\gt a+1/j \), putting \( x \) in the right side. Conversely, if \( x \) is in the right side, then for some \( j,m \), all \( k\ge m \) satisfy \( f_k(x)\gt a+1/j \), so taking limits gives \( f(x)\ge a+1/j\gt a \).

The right side belongs to \( \mathcal{ S } \) because each \( f_k \) is measurable and \( \mathcal{ S } \) is closed under countable unions and intersections. By the ray test, \( f \) is measurable.

Borel Subsets of the Extended Real Line
A subset \( C \subseteq [ -\infty, \infty ] \) is Borel if \( C \cap \mathbb{ R } \) is a Borel subset of \( \mathbb{ R } \).
Extended Real-Valued Measurable Function
Suppose that \( \left( X, \mathcal{ S } \right) \) is a measurable space. A function \( f: X \to [ -\infty, \infty ] \) is \( \mathcal{ S } \)-measurable if \( f ^ { -1 } \left( B \right) \in \mathcal{ S } \) for every Borel set \( B \subseteq [ -\infty, \infty ] \).
Extended Ray Test for Measurability
Suppose that \( \left( X, \mathcal{ S } \right) \) is a measurable space and \( f: X \to [ -\infty, \infty ] \). If \[ f ^ { -1 } \left( \left( a, \infty \right] \right) \in \mathcal{ S } \] for every \( a \in \mathbb{ R } \), then \( f \) is \( \mathcal{ S } \)-measurable.

Let \[ \mathcal{ T} := \left\{ B \subseteq [ -\infty,\infty ] : f ^ { -1 } \left( B \right) \in \mathcal{ S } \right\}. \] The algebra of inverse images shows that \( \mathcal{ T} \) is a \( \sigma \)-algebra on \( [ -\infty,\infty ] \). By hypothesis, it contains every ray \( \left( a,\infty \right] \). These rays generate the Borel subsets of the extended real line, so every Borel \( B \subseteq [ -\infty,\infty ] \) belongs to \( \mathcal{ T} \). Therefore \( f \) is \( \mathcal{ S } \)-measurable.

Infimum and Supremum of Measurable Functions
Suppose that \( \left( X, \mathcal{ S } \right) \) is a measurable space and \( f _ 1, f _ 2, \ldots: X \to [ -\infty, \infty ] \) are \( \mathcal{ S } \)-measurable. Define \[ g \left( x \right) := \inf \left\{ f _ k \left( x \right): k \in \mathbb{ Z } ^ + \right\} \qquad \text{and} \qquad h \left( x \right) := \sup \left\{ f _ k \left( x \right): k \in \mathbb{ Z } ^ + \right\}. \] Then \( g \) and \( h \) are \( \mathcal{ S } \)-measurable.

For \( h \), note that for every \( a \in \mathbb{ R } \), \[ h ^ { -1 } \left( \left( a,\infty \right] \right) = \bigcup _ { k = 1 } ^ \infty f _ k ^ { -1 } \left( \left( a,\infty \right] \right). \] The right side belongs to \( \mathcal{ S } \), so \( h \) is measurable by the extended ray test.

For \( g \), observe that \[ g(x) = - \sup \left\{ -f _ k(x) : k \in \mathbb{ N } _ 1 \right\}. \] Each \( -f _ k \) is measurable as a composition with \( x \mapsto -x \), and the supremum result just proved applies. Therefore \( g \) is measurable.

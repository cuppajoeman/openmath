Outer Measure is Additive when One Set is Open
Suppose that \( A, G \subseteq \mathbb{ R } \), \( A \cap G = \emptyset \), and \( G \) is open. Then \[ \operatorname{omes} \left( A \cup G \right) = \operatorname{omes} \left( A \right) + \operatorname{omes} \left( G \right). \]

By countable subadditivity of outer measure, \( \operatorname{omes} \left( A \cup G \right) \le \operatorname{omes} \left( A \right) + \operatorname{omes} \left( G \right) \). For the reverse inequality, cover \( A \cup G \) by open intervals with total length arbitrarily close to \( \operatorname{omes} \left( A \cup G \right) \). Since \( G \) is open, each closed bounded part of \( G \) has positive distance from \( A \), so the intervals in the cover can be split into a part covering \( A \) and a disjoint part covering that closed part of \( G \). Taking suprema over closed bounded subsets of \( G \), and then the infimum over covers of \( A \cup G \), gives the reverse inequality.

Outer Measure is Additive when One Set is Closed
Suppose that \( A, F \subseteq \mathbb{ R } \), \( A \cap F = \emptyset \), and \( F \) is closed. Then \[ \operatorname{omes} \left( A \cup F \right) = \operatorname{omes} \left( A \right) + \operatorname{omes} \left( F \right). \]

Let \( G = \mathbb{ R } \setminus F \). Then \( G \) is open, and \( A \cap F = \emptyset \) implies \( A \subseteq G \). Apply additivity when one set is open to \( A \cup F \) inside large bounded intervals and pass to the limit over those intervals. Equivalently, approximate \( F \) from inside by closed bounded sets \( F \cap [ -n,n ] \), use the open-set case on their complements, and use order preservation to let \( n \to \infty \). This yields \( \operatorname{omes} \left( A \cup F \right) \ge \operatorname{omes} \left( A \right) + \operatorname{omes} \left( F \right) \), while the opposite inequality is subadditivity.

Borel Sets Can be Approximated from Below by Closed Sets
Suppose that \( B \subseteq \mathbb{ R } \) is a Borel set. For every \( \epsilon \in \mathbb{ R } ^ + \), there exists a closed set \( F \subseteq B \) such that \[ \operatorname{omes} \left( B \setminus F \right) \lt \epsilon. \]

Let \( \mathcal{ C } \) be the class of Borel sets with this approximation property. Open sets belong to \( \mathcal{ C } \) because they are increasing unions of closed sets \( F _ n = \left\{ x \in [ -n,n ] : \operatorname{dist} \left( x, \mathbb{ R } \setminus B \right) \ge 1/n \right\} \). The class \( \mathcal{ C } \) is closed under complements and countable unions by using countable subadditivity and distributing an error \( \epsilon / 2 ^ k \) over the pieces. Since the Borel \( \sigma \)-algebra is generated by the open sets, every Borel set belongs to \( \mathcal{ C } \).

Outer Measure is Additive when One Set is Borel
Suppose that \( A, B \subseteq \mathbb{ R } \), \( A \cap B = \emptyset \), and \( B \) is a Borel set. Then \[ \operatorname{omes} \left( A \cup B \right) = \operatorname{omes} \left( A \right) + \operatorname{omes} \left( B \right). \]

Choose, by Borel approximation from below, a closed set \( F \subseteq B \) such that \( \operatorname{omes} \left( B \setminus F \right) \lt \epsilon \). Since \( A \cap F = \emptyset \), additivity with a closed set gives \[ \operatorname{omes} \left( A \cup F \right) = \operatorname{omes} \left( A \right) + \operatorname{omes} \left( F \right). \] Using subadditivity on \( B = F \cup \left( B \setminus F \right) \) and on \( A \cup B = \left( A \cup F \right) \cup \left( B \setminus F \right) \), we get the reverse inequality up to \( \epsilon \). Letting \( \epsilon \downarrow 0 \) proves equality; the other inequality is again subadditivity.

There is a Non-Borel Set of Finite Outer Measure
There exists a set \( B \subseteq \mathbb{ R } \) such that \( \operatorname{omes} \left( B \right) \lt \infty \) and \( B \) is not a Borel set.

By outer measure is not additive on all subsets, there is a subset \( V \subseteq [0,1] \) for which outer measure fails to be additive on disjoint unions. Since \( V \subseteq [0,1] \), order preservation and the closed interval computation give \( \operatorname{omes} \left( V \right) \le 1 \). If \( V \) were Borel, then outer measure would be additive when one set is Borel, contradicting the construction. Thus \( V \) is a non-Borel set of finite outer measure.

Outer Measure is a Measure on Borel Sets
Let \( \mathcal{ B } \) be the Borel \( \sigma \)-algebra on \( \mathbb{ R } \). Outer measure restricted to \( \mathcal{ B } \) is a measure on \( \left( \mathbb{ R }, \mathcal{ B } \right) \).

The empty set has outer measure \( 0 \). If \( A,B \in \mathcal{ B} \) are disjoint, then additivity when one set is Borel gives finite additivity. For a disjoint sequence \( B _ 1,B _ 2,\ldots \in \mathcal{ B} \), apply finite additivity to the first \( n \) sets and then use order preservation for the lower bound. The upper bound is countable subadditivity. Letting \( n \to \infty \) gives countable additivity, so the restriction is a measure.

Lebesgue Measure on Borel Sets
Let \( \mathcal{ B } \) be the Borel \( \sigma \)-algebra on \( \mathbb{ R } \). Lebesgue measure on Borel sets is the measure on \( \left( \mathbb{ R }, \mathcal{ B } \right) \) that sends each Borel set to its outer measure.
Lebesgue Measurable Set
A set \( A \subseteq \mathbb{ R } \) is Lebesgue measurable if there exists a Borel set \( B \subseteq A \) such that \[ \operatorname{omes} \left( A \setminus B \right) = 0. \]
Equivalent Characterizations of Lebesgue Measurable Sets
Suppose that \( A \subseteq \mathbb{ R } \). The following are equivalent:
  • \( A \) is Lebesgue measurable.
  • For every \( \epsilon \in \mathbb{ R } ^ + \), there exists a closed set \( F \subseteq A \) such that \( \operatorname{omes} \left( A \setminus F \right) \lt \epsilon \).
  • There are closed sets \( F _ 1, F _ 2, \ldots \subseteq A \) such that \( \operatorname{omes} \left( A \setminus \bigcup _ { k = 1 } ^ \infty F _ k \right) = 0 \).
  • There exists a Borel set \( B \subseteq A \) such that \( \operatorname{omes} \left( A \setminus B \right) = 0 \).
  • For every \( \epsilon \in \mathbb{ R } ^ + \), there exists an open set \( G \supseteq A \) such that \( \operatorname{omes} \left( G \setminus A \right) \lt \epsilon \).
  • There are open sets \( G _ 1, G _ 2, \ldots \supseteq A \) such that \( \operatorname{omes} \left( \left( \bigcap _ { k = 1 } ^ \infty G _ k \right) \setminus A \right) = 0 \).
  • There exists a Borel set \( B \supseteq A \) such that \( \operatorname{omes} \left( B \setminus A \right) = 0 \).

The definition immediately gives the fourth condition. The approximation theorem for Borel sets gives the second condition from the fourth, and the second implies the third by choosing \( F _ k \subseteq A \) with \( \operatorname{omes} \left( A \setminus F _ k \right) \lt 1/k \). The third implies the fourth because a Borel set is closed under countable unions. The open-set conditions follow by applying the closed-set conditions to complements and using subadditivity; conversely, if \( B \supseteq A \) is Borel and \( \operatorname{omes} \left( B \setminus A \right)=0 \), then a Borel subset of \( B \) differing from \( A \) by outer measure zero gives the defining condition. Hence all seven conditions are equivalent.

Outer Measure is a Measure on Lebesgue Measurable Sets
The Lebesgue measurable subsets of \( \mathbb{ R } \) form a \( \sigma \)-algebra \( \mathcal{ L } \), and outer measure restricted to \( \mathcal{ L } \) is a measure on \( \left( \mathbb{ R }, \mathcal{ L } \right) \).

The equivalent characterizations show that complements and countable unions of Lebesgue measurable sets are again Lebesgue measurable, so they form a \( \sigma \)-algebra. If \( A _ 1,A _ 2,\ldots \) are disjoint Lebesgue measurable sets, choose Borel sets \( B _ k \supseteq A _ k \) with \( \operatorname{omes} \left( B _ k \setminus A _ k \right)=0 \). Countable additivity on Borel sets and subadditivity over the null differences give countable additivity on \( \mathcal{ L} \). Therefore outer measure restricted to \( \mathcal{ L} \) is a measure.

Lebesgue Measure on Lebesgue Measurable Sets
Let \( \mathcal{ L } \) be the \( \sigma \)-algebra of Lebesgue measurable subsets of \( \mathbb{ R } \). Lebesgue measure on Lebesgue measurable sets is the measure on \( \left( \mathbb{ R }, \mathcal{ L } \right) \) that sends each Lebesgue measurable set to its outer measure.
Cantor Set
Define \( G _ 1 := \left( 1 / 3, 2 / 3 \right) \). For \( n \gt 1 \), let \( G _ n \) be the union of the open middle thirds of the intervals remaining in \[ [ 0, 1 ] \setminus \bigcup _ { j = 1 } ^ { n - 1 } G _ j. \] The Cantor set is \[ C := [ 0, 1 ] \setminus \bigcup _ { n = 1 } ^ \infty G _ n. \]
Base Three Description of the Cantor Set
The Cantor set is the set of all \( x \in [ 0, 1 ] \) that have a base \( 3 \) representation using only the digits \( 0 \) and \( 2 \).

A point \( x \in [0,1] \) remains in the Cantor set exactly when no stage of the construction removes it. In base \( 3 \), the first stage removes the numbers whose first ternary digit is \( 1 \), the second removes those whose first allowed digit is followed by a \( 1 \), and so on. Thus the points never removed are precisely those admitting a ternary expansion using only \( 0 \) and \( 2 \). At ternary-expansion endpoints, choose the expansion that avoids a trailing string of \( 2 \)'s, which keeps the same description valid.

Basic Properties of the Cantor Set
The Cantor set \( C \) is closed, has Lebesgue measure \( 0 \), and contains no interval with more than one element.

At stage \( n \), the removed set is a finite union of open intervals, so its complement is closed; hence \( C \), an intersection of closed sets, is closed. The total length removed is \[ \sum _ { n=1 } ^ \infty \frac{2 ^ { n-1 }}{3 ^ n}=1, \] so \( C \subseteq [0,1] \) has Lebesgue measure \( 0 \) by subadditivity and the interval computation. Every nontrivial interval contains a subinterval removed at some finite stage, so no interval with more than one point is contained in \( C \).

Cantor Function
The Cantor function \( \Lambda: [ 0, 1 ] \to [ 0, 1 ] \) is defined by converting base \( 3 \) representations to base \( 2 \) representations: for \( x \in C \), replace each digit \( 2 \) in the \( 0 \)-and-\( 2 \) base \( 3 \) representation by \( 1 \); for \( x \notin C \), truncate its base \( 3 \) representation immediately after the first digit \( 1 \), replace any earlier \( 2 \)'s by \( 1 \)'s, and read the result as a base \( 2 \) number.
Cantor Function
The Cantor function \( \Lambda \) is a continuous increasing function from \( [ 0, 1 ] \) onto \( [ 0, 1 ] \). Moreover, \[ \Lambda \left( C \right) = [ 0, 1 ]. \]

The base-three description sends \( x = 0.x _ 1x _ 2\cdots _ 3 \in C \), with each \( x _ k \in \left\{0,2\right\} \), to \( 0.y _ 1y _ 2\cdots _ 2 \), where \( y _ k = x _ k/2 \). This produces every binary expansion in \( [0,1] \), so \( \Lambda \left( C \right)=[0,1] \). If \( x \le y \), then the first ternary digit at which they differ is smaller for \( x \), so the corresponding binary value is no larger; hence \( \Lambda \) is increasing. On each removed middle-third interval the truncation rule makes \( \Lambda \) constant, and the endpoint values match the limiting Cantor values, so \( \Lambda \) is continuous.

Cantor Set is Uncountable
The Cantor set is uncountable.

Since \( \Lambda \left( C \right)=[0,1] \) by the Cantor function theorem, there is a surjection from \( C \) onto \( [0,1] \). If \( C \) were countable, then its image under any function would be countable, contradicting that \( [0,1] \) is uncountable by nontrivial intervals are uncountable.

Continuous Image of a Lebesgue Measurable Set Can be Nonmeasurable
There exists a Lebesgue measurable set \( A \subseteq [ 0, 1 ] \) such that \( \operatorname{omes} \left( A \right) = 0 \) and \( \Lambda \left( A \right) \) is not Lebesgue measurable.

Let \( N \subseteq [0,1] \) be a non-Lebesgue-measurable set, and choose \( A \subseteq C \) with \( \Lambda \left( A \right)=N \), possible because \( \Lambda \left( C \right)=[0,1] \). Since \( A \subseteq C \) and \( C \) has Lebesgue measure \( 0 \), order preservation gives \( \operatorname{omes} \left( A \right)=0 \), so \( A \) is Lebesgue measurable. But \( \Lambda \left( A \right)=N \) is not Lebesgue measurable by construction.

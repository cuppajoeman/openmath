Outer Measure is Additive when One Set is Open
Suppose that \( A, G \subseteq \mathbb{ R } \), \( A \cap G = \emptyset \), and \( G \) is open. Then \[ \left\lvert A \cup G \right\rvert = \left\lvert A \right\rvert + \left\lvert G \right\rvert. \]
Outer Measure is Additive when One Set is Closed
Suppose that \( A, F \subseteq \mathbb{ R } \), \( A \cap F = \emptyset \), and \( F \) is closed. Then \[ \left\lvert A \cup F \right\rvert = \left\lvert A \right\rvert + \left\lvert F \right\rvert. \]
Borel Sets Can be Approximated from Below by Closed Sets
Suppose that \( B \subseteq \mathbb{ R } \) is a Borel set. For every \( \epsilon \in \mathbb{ R } ^ + \), there exists a closed set \( F \subseteq B \) such that \[ \left\lvert B \setminus F \right\rvert \lt \epsilon. \]
Outer Measure is Additive when One Set is Borel
Suppose that \( A, B \subseteq \mathbb{ R } \), \( A \cap B = \emptyset \), and \( B \) is a Borel set. Then \[ \left\lvert A \cup B \right\rvert = \left\lvert A \right\rvert + \left\lvert B \right\rvert. \]
There is a Non-Borel Set of Finite Outer Measure
There exists a set \( B \subseteq \mathbb{ R } \) such that \( \left\lvert B \right\rvert \lt \infty \) and \( B \) is not a Borel set.
Outer Measure is a Measure on Borel Sets
Let \( \mathcal{ B } \) be the Borel \( \sigma \)-algebra on \( \mathbb{ R } \). Outer measure restricted to \( \mathcal{ B } \) is a measure on \( \left( \mathbb{ R }, \mathcal{ B } \right) \).
Lebesgue Measure on Borel Sets
Let \( \mathcal{ B } \) be the Borel \( \sigma \)-algebra on \( \mathbb{ R } \). Lebesgue measure on Borel sets is the measure on \( \left( \mathbb{ R }, \mathcal{ B } \right) \) that sends each Borel set to its outer measure.
Lebesgue Measurable Set
A set \( A \subseteq \mathbb{ R } \) is Lebesgue measurable if there exists a Borel set \( B \subseteq A \) such that \[ \left\lvert A \setminus B \right\rvert = 0. \]
Equivalent Characterizations of Lebesgue Measurable Sets
Suppose that \( A \subseteq \mathbb{ R } \). The following are equivalent:
  • \( A \) is Lebesgue measurable.
  • For every \( \epsilon \in \mathbb{ R } ^ + \), there exists a closed set \( F \subseteq A \) such that \( \left\lvert A \setminus F \right\rvert \lt \epsilon \).
  • There are closed sets \( F _ 1, F _ 2, \ldots \subseteq A \) such that \( \left\lvert A \setminus \bigcup _ { k = 1 } ^ \infty F _ k \right\rvert = 0 \).
  • There exists a Borel set \( B \subseteq A \) such that \( \left\lvert A \setminus B \right\rvert = 0 \).
  • For every \( \epsilon \in \mathbb{ R } ^ + \), there exists an open set \( G \supseteq A \) such that \( \left\lvert G \setminus A \right\rvert \lt \epsilon \).
  • There are open sets \( G _ 1, G _ 2, \ldots \supseteq A \) such that \( \left\lvert \left( \bigcap _ { k = 1 } ^ \infty G _ k \right) \setminus A \right\rvert = 0 \).
  • There exists a Borel set \( B \supseteq A \) such that \( \left\lvert B \setminus A \right\rvert = 0 \).
Outer Measure is a Measure on Lebesgue Measurable Sets
The Lebesgue measurable subsets of \( \mathbb{ R } \) form a \( \sigma \)-algebra \( \mathcal{ L } \), and outer measure restricted to \( \mathcal{ L } \) is a measure on \( \left( \mathbb{ R }, \mathcal{ L } \right) \).
Lebesgue Measure on Lebesgue Measurable Sets
Let \( \mathcal{ L } \) be the \( \sigma \)-algebra of Lebesgue measurable subsets of \( \mathbb{ R } \). Lebesgue measure on Lebesgue measurable sets is the measure on \( \left( \mathbb{ R }, \mathcal{ L } \right) \) that sends each Lebesgue measurable set to its outer measure.
Cantor Set
Define \( G _ 1 := \left( 1 / 3, 2 / 3 \right) \). For \( n \gt 1 \), let \( G _ n \) be the union of the open middle thirds of the intervals remaining in \[ [ 0, 1 ] \setminus \bigcup _ { j = 1 } ^ { n - 1 } G _ j. \] The Cantor set is \[ C := [ 0, 1 ] \setminus \bigcup _ { n = 1 } ^ \infty G _ n. \]
Base Three Description of the Cantor Set
The Cantor set is the set of all \( x \in [ 0, 1 ] \) that have a base \( 3 \) representation using only the digits \( 0 \) and \( 2 \).
Basic Properties of the Cantor Set
The Cantor set \( C \) is closed, has Lebesgue measure \( 0 \), and contains no interval with more than one element.
Cantor Function
The Cantor function \( \Lambda: [ 0, 1 ] \to [ 0, 1 ] \) is defined by converting base \( 3 \) representations to base \( 2 \) representations: for \( x \in C \), replace each digit \( 2 \) in the \( 0 \)-and-\( 2 \) base \( 3 \) representation by \( 1 \); for \( x \notin C \), truncate its base \( 3 \) representation immediately after the first digit \( 1 \), replace any earlier \( 2 \)'s by \( 1 \)'s, and read the result as a base \( 2 \) number.
Cantor Function
The Cantor function \( \Lambda \) is a continuous increasing function from \( [ 0, 1 ] \) onto \( [ 0, 1 ] \). Moreover, \[ \Lambda \left( C \right) = [ 0, 1 ]. \]
Cantor Set is Uncountable
The Cantor set is uncountable.
Continuous Image of a Lebesgue Measurable Set Can be Nonmeasurable
There exists a Lebesgue measurable set \( A \subseteq [ 0, 1 ] \) such that \( \left\lvert A \right\rvert = 0 \) and \( \Lambda \left( A \right) \) is not Lebesgue measurable.

Length of an Open Interval
The length \( \ell \left( I \right) \) of an open interval \( I \subseteq \mathbb{ R } \) is defined by \[ \ell \left( I \right) = \begin{cases} b - a & I = \left( a, b \right) \text{ with } a \lt b, \\ 0 & I = \emptyset, \\ \infty & I = \left( -\infty, a \right) \text{ or } I = \left( a, \infty \right), \\ \infty & I = \left( -\infty, \infty \right). \end{cases} \]
Outer Measure
Suppose that \( A \subseteq \mathbb{ R } \). The outer measure of \( A \) is \[ \left\lvert A \right\rvert := \inf \left\{ \sum _ { k = 1 } ^ \infty \ell \left( I _ k \right) : A \subseteq \bigcup _ { k = 1 } ^ \infty I _ k \text{ and each } I _ k \text{ is an open interval} \right\}. \]
Countable Sets Have Outer Measure Zero
Every countable subset of \( \mathbb{ R } \) has outer measure \( 0 \).
Outer Measure Preserves Order
Suppose that \( A, B \subseteq \mathbb{ R } \) and \( A \subseteq B \). Then \[ \left\lvert A \right\rvert \le \left\lvert B \right\rvert. \]
Translation of a Set
Suppose that \( t \in \mathbb{ R } \) and \( A \subseteq \mathbb{ R } \). Define the translation of \( A \) by \( t \) as \[ t + A := \left\{ t + a : a \in A \right\}. \]
Outer Measure is Translation Invariant
Suppose that \( t \in \mathbb{ R } \) and \( A \subseteq \mathbb{ R } \). Then \[ \left\lvert t + A \right\rvert = \left\lvert A \right\rvert. \]
Countable Subadditivity of Outer Measure
Suppose that \( A _ 1, A _ 2, \ldots \subseteq \mathbb{ R } \). Then \[ \left\lvert \bigcup _ { k = 1 } ^ \infty A _ k \right\rvert \le \sum _ { k = 1 } ^ \infty \left\lvert A _ k \right\rvert. \]
Open Cover
Suppose that \( A \subseteq \mathbb{ R } \). A collection \( \mathcal{ C } \) of open subsets of \( \mathbb{ R } \) is an open cover of \( A \) if \[ A \subseteq \bigcup _ { G \in \mathcal{ C } } G. \]
Finite Subcover
Suppose that \( \mathcal{ C } \) is an open cover of \( A \subseteq \mathbb{ R } \). We say that \( \mathcal{ C } \) has a finite subcover if there are \( G _ 1, \ldots, G _ n \in \mathcal{ C } \) such that \[ A \subseteq G _ 1 \cup \cdots \cup G _ n. \]
Heine-Borel Theorem
Every open cover of a closed bounded subset of \( \mathbb{ R } \) has a finite subcover.
Outer Measure of a Closed Interval
Suppose that \( a, b \in \mathbb{ R } \) and \( a \lt b \). Then \[ \left\lvert [ a, b ] \right\rvert = b - a. \]
Nontrivial Intervals are Uncountable
Every interval in \( \mathbb{ R } \) that contains at least two distinct elements is uncountable.
Outer Measure is Not Additive
There exist disjoint subsets \( A, B \subseteq \mathbb{ R } \) such that \[ \left\lvert A \cup B \right\rvert \neq \left\lvert A \right\rvert + \left\lvert B \right\rvert. \]
No Countably Additive Translation Invariant Extension of Length to All Subsets
There is no function \( \mu: \mathcal{ P } \left( \mathbb{ R } \right) \to [ 0, \infty ] \) such that
  • \( \mu \left( I \right) = \ell \left( I \right) \) for every open interval \( I \subseteq \mathbb{ R } \),
  • \( \mu \left( \bigcup _ { k = 1 } ^ \infty A _ k \right) = \sum _ { k = 1 } ^ \infty \mu \left( A _ k \right) \) for every pairwise disjoint sequence \( A _ 1, A _ 2, \ldots \subseteq \mathbb{ R } \),
  • \( \mu \left( t + A \right) = \mu \left( A \right) \) for every \( t \in \mathbb{ R } \) and every \( A \subseteq \mathbb{ R } \).

Continuous Random Variable
We say that a random variable \( X \) is continuous if \[ P \left( X = x \right) = 0 \] for every \( x \in \mathbb{ R } \).

This is not the same as saying that \(X\) is not a discrete random variable. A discrete random variable has a countable set of possible values. A continuous random variable has no single value with positive probability. These are different conditions.

Continuous Random Variable Has Uncountable Range
If \(X\) is a continuous random variable, then \(X\left(\Omega\right)\) is not countable.
Suppose, toward a contradiction, that \(X\left(\Omega\right)\) is countable. Since every outcome \(\omega\in\Omega\) has \(X\left(\omega\right)\in X\left(\Omega\right)\), \[ \Omega = \bigcup_{x\in X\left(\Omega\right)} \left\{\omega\in\Omega:X\left(\omega\right)=x\right\}. \] Because \(X\) is continuous, each event \(\{X=x\}\) has probability \(0\). By countable additivity in the definition of a probability measure, \[ P\left(\Omega\right) = \sum_{x\in X\left(\Omega\right)} P\left(X=x\right) = 0. \] This contradicts \(P\left(\Omega\right)=1\), so \(X\left(\Omega\right)\) is not countable.
Uncountable Range Does Not Imply Continuous
A random variable can have an uncountable set of possible values without being continuous.
Let \(X\) be a random variable with \(X\left(\Omega\right)=[0,1]\) whose distribution function is \[ F_X(x) = \begin{cases} 0, & x \lt 0, \\ \frac{1}{2}+\frac{x}{2}, & 0\le x \le 1, \\ 1, & x \gt 1. \end{cases} \] The set of possible values of \(X\) contains \([0,1]\), so it is not countable because nontrivial intervals are uncountable. Thus \(X\) is not a discrete random variable. However, \[ P\left(X=0\right) = F_X(0)-\lim_{x\to 0^-}F_X(x) = \frac{1}{2}, \] so \(X\) is not continuous.
Random Variable Neither Discrete Nor Continuous
Give an example of a random variable that is neither discrete nor continuous.
Let \(X\) be a random variable with \(X\left(\Omega\right)=[0,1]\) whose distribution function is \[ F_X(x) = \begin{cases} 0, & x \lt 0, \\ \frac{1}{2}+\frac{x}{2}, & 0\le x \le 1, \\ 1, & x \gt 1. \end{cases} \] The set of possible values of \(X\) contains \([0,1]\), which is uncountable. Therefore \(X\left(\Omega\right)\) is not countable, so \(X\) is not discrete.

On the other hand,

\[ P\left(X=0\right) = F_X(0)-\lim_{x\to 0^-}F_X(x) = \frac{1}{2}, \] so it is not true that \(P\left(X=x\right)=0\) for every \(x\in\mathbb{R}\). Therefore \(X\) is not continuous.

This is different from a die roll. If \(X\) is the result of a fair six-sided die, then \(P\left(X=3\right)=1/6\), so a single exact outcome can have positive probability. Here \(P\left(X=3\right)\) is shorthand for the probability of the event determined by the binary relation \(X=3\). For a continuous random variable, every exact value has probability \(0\); probabilities live on intervals or regions rather than individual values.

One way to picture the transition is to imagine a fair die with \(N\) equally likely faces. Each single face has probability \(1/N\). As \(N\) gets larger, \(1/N\) gets smaller. If the faces become tiny patches approximating a smooth sphere and the limiting roll is modeled as landing on a point of the sphere, then any single exact point has limiting probability \(0\), even though a region of the sphere can still have positive probability.

Continuous Uniform Distribution
The random variable \(U\) is said to have a (continuous) uniform distribution on the unit interval \([0,1]\) denoted by \( U \sim \text { unif }[0,1]\) iff \[ P(U \leq u)=u \quad \forall 0 \leq u \leq 1 . \]
Probability of an Interval in the Uniform Distribution
For a continuous uniform random variable, \[ P \left( a \le X \le b \right) = b - a \]
Density Function
Let \( X \) be a random variable. A function \( f: \mathbb{ R } \to [0,\infty) \) is a density function for \( X \) if, for every interval \( I \subseteq \mathbb{ R } \), \[ P\left(X\in I\right) = \int_I f\left(t\right)\,dt. \]

For example, the function \(f:\mathbb{R}\to[0,\infty)\) defined by \(f(t)=1\) for \(t\in[0,1]\) and \(f(t)=0\) otherwise is a density function for a random variable uniformly distributed on \([0,1]\). For any interval \(I\), the integral \(\int_I f(t)\,dt\) is the length of the part of \(I\) that lies inside \([0,1]\).

The reason an integral appears is not that we add the probabilities of the individual points. For a discrete random variable, that pointwise method works because the possible values form a countable set. The analogous object is usually called a probability mass function, not a density function. If \(X\) is a fair six-sided die roll, then its mass function satisfies \(p_X(k)=1/6\) for \(k\in\{1,2,3,4,5,6\}\). For the interval \([2,4]\), the probability is the sum of the masses at the possible values in that interval: \[ P\left(X\in[2,4]\right) = p_X(2)+p_X(3)+p_X(4) = \frac{3}{6}. \] For a continuous random variable, trying to keep this rule fails: if every point has probability \(0\), then summing point probabilities cannot recover a positive probability for an interval. The repair is to assign probability to the event that the random value lands in a small interval, not to the individual points. A density says that if \(I\) is a small interval of length \(\Delta t\) near \(t\), for example \(I=\left(t-\epsilon,t+\epsilon\right)\), then \[ P\left(X\in I\right) \approx f(t)\Delta t. \] These small interval probabilities are compatible with the basic additivity rule for disjoint events, because adjacent small intervals contribute by addition. When the interval is divided into finer and finer pieces, the limiting sum of these interval contributions is the integral. Thus the integral is the continuous replacement for summing masses at possible values, while the value \(f(t)\) itself is not the probability of the point \(t\).

Density has Total Mass One
If \( f \) is a density function, then \[ \int_{-\infty}^{\infty} f\left(t\right)\,dt=1. \]
Apply the definition of density to the interval \(I=\mathbb{R}\). Since a random variable is real-valued with probability one, \(P\left(X\in\mathbb{R}\right)=1\). Therefore the integral over \(\mathbb{R}\) is \(1\).
Density Determines Distribution Function
If \( X \) has density \( f \), then its distribution function satisfies \[ F _ X \left( x \right) = \int _ {-\infty} ^ x f \left( t \right) dt \] for every \( x \in \mathbb{ R } \).
Apply the definition of density to the interval \(I=\left(-\infty,x\right]\).
Density Determines Interval Probabilities
If \( X \) has density \( f \), then for \( a \lt b \), \[ P \left( a \lt X \le b \right) = \int _ a ^ b f \left( x \right) \, dx . \]
Apply the definition of density to the interval \(I=\left(a,b\right]\).
CDF Derivative is the Density
Suppose that \( X \) has density function \( f \), and let \( F _ X \) be the distribution function of \( X \). If \( f \) is continuous at \( x _ 0 \in \mathbb{ R } \), then \( F _ X \) is differentiable at \( x _ 0 \) and \[ F _ X ^ \prime \left( x _ 0 \right) = f \left( x _ 0 \right). \]
Choose \( a,b \in \mathbb{ R } \) with \( a \lt x _ 0 \lt b \). For \( x \in (a,b) \), density determines interval probabilities gives \[ F _ X \left( x \right) - F _ X \left( a \right) = P \left( a \lt X \le x \right) = \int _ a ^ x f \left( t \right) \, dt. \] Hence, on \( (a,b) \), \[ F _ X \left( x \right) = F _ X \left( a \right) + \int _ a ^ x f \left( t \right) \, dt. \] The constant term \( F _ X \left( a \right) \) has derivative \( 0 \), and by the Fundamental Theorem of Calculus I, the derivative of \( \int _ a ^ x f \left( t \right) \, dt \) at \( x _ 0 \) is \( f \left( x _ 0 \right) \), because \( f \) is continuous at \( x _ 0 \). Therefore \( F _ X ^ \prime \left( x _ 0 \right)=f \left( x _ 0 \right) \).
Expectation of a Continuous Random Variable
If \( X \) has density \( f \), then \[ E \left( X \right) = \int _ { -\infty } ^ \infty x f \left( x \right) \, dx, \] whenever the integral is defined.
Expectation of an Interval Indicator
Suppose that \( X \) has density \( f \) and \( a,b \in \mathbb{ R } \) with \( a \lt b \). Then \[ E \left( \chi _ { (a,b] } \left( X \right) \right) = \int _ { -\infty } ^ \infty \chi _ { (a,b] } \left( x \right) f \left( x \right) \, dx. \]
Let \[ A := \left\{ \omega \in \Omega : a \lt X \left( \omega \right) \le b \right\}. \] For every \( \omega \in \Omega \), \[ \chi _ { (a,b] } \left( X \left( \omega \right) \right) = \begin{cases} 1, & \omega \in A, \\ 0, & \omega \notin A, \end{cases} \] so \( \chi _ { (a,b] } \left( X \right)=\mathbf{ 1 } _ A \) by the definition of an indicator random variable. Therefore, by expectation of an indicator, \[ E \left( \chi _ { (a,b] } \left( X \right) \right) = E \left( \mathbf{ 1 } _ A \right) = P \left( A \right) = P \left( a \lt X \le b \right). \] By density determines interval probabilities, \[ P \left( a \lt X \le b \right) = \int _ a ^ b f \left( x \right) \, dx. \] Since \( \chi _ { (a,b] } \left( x \right) \) is \( 1 \) on \( (a,b] \) and \( 0 \) off \( (a,b] \), \[ \int _ a ^ b f \left( x \right) \, dx = \int _ { -\infty } ^ \infty \chi _ { (a,b] } \left( x \right) f \left( x \right) \, dx. \]
Expectation of a Function of a Continuous Random Variable
If \( X \) has density \( f \) and \( g: \mathbb{ R } \to \mathbb{ R } \) is measurable, then \[ E \left( g \left( X \right) \right) = \int _ { -\infty } ^ \infty g \left( x \right)f \left( x \right) \, dx, \] whenever the integral is defined.
The formula holds for interval indicators by expectation of an interval indicator. By finite additivity and linearity of expectation for finite sums, the same formula holds for every non-negative simple function \( s \) built from finitely many intervals: \[ E \left( s \left( X \right) \right) = \int _ { -\infty } ^ \infty s \left( x \right) f \left( x \right) \, dx. \] For a non-negative measurable function \( g \), choose an increasing sequence of non-negative simple functions \( s _ n \) with \( s _ n \left( x \right) \to g \left( x \right) \). Applying the simple-function case to each \( s _ n \), then using the monotone convergence theorem on both sides, gives \[ E \left( g \left( X \right) \right) = \int _ { -\infty } ^ \infty g \left( x \right) f \left( x \right) \, dx. \] For a general measurable \( g \), apply the non-negative case to the positive part \( g ^ + \) and the negative part \( g ^ - \), then subtract, whenever the resulting integral is defined.
Continuous Expectation Respects Scalar Multiplication
Suppose that \( X \) has density \( f _ X \), \( g: \mathbb{ R } \to \mathbb{ R } \) is measurable, and \( a \in \mathbb{ R } \). If the expectations are defined, then \[ E \left( a g \left( X \right) \right) = a E \left( g \left( X \right) \right). \]
By expectation of a function of a continuous random variable, \[ E \left( a g \left( X \right) \right) = \int _ { -\infty } ^ \infty a g \left( x \right) f _ X \left( x \right) \, dx. \] Pulling the constant \( a \) out of the integral gives \[ \int _ { -\infty } ^ \infty a g \left( x \right) f _ X \left( x \right) \, dx = a \int _ { -\infty } ^ \infty g \left( x \right) f _ X \left( x \right) \, dx. \] Applying expectation of a function of a continuous random variable again gives \[ a \int _ { -\infty } ^ \infty g \left( x \right) f _ X \left( x \right) \, dx = a E \left( g \left( X \right) \right). \]
Continuous Expectation is Linear for Finite Sums
Suppose that \( X _ 1,\ldots,X _ n \) are continuous random variables and \( g _ 1,\ldots,g _ n: \mathbb{ R } \to \mathbb{ R } \) are measurable functions. If the expectations are defined, then \[ E \left( \sum _ { i=1 } ^ n g _ i \left( X _ i \right) \right) = \sum _ { i=1 } ^ n E \left( g _ i \left( X _ i \right) \right). \]
This is the finite-sum case of linearity of expectation, applied to the random variables \( g _ 1 \left( X _ 1 \right),\ldots,g _ n \left( X _ n \right) \).
Joint Density Function
Random variables \( X \) and \( Y \) have joint density \( f _ { X,Y } \) if it is a two-variable analogue of a density function: \( f _ { X,Y } \ge 0 \), \( \int _ { -\infty } ^ \infty \int _ { -\infty } ^ \infty f _ { X,Y } \left( x,y \right) \, dx \, dy = 1 \), and \[ P \left( \left( X,Y \right) \in A \right) = \iint _ A f _ { X,Y } \left( x,y \right) \, dx \, dy \] for suitable sets \( A \subseteq \mathbb{ R } ^ 2 \).
Continuous Marginal Density Functions
If \( X \) and \( Y \) have joint density \( f _ { X,Y } \), then their marginal densities are \[ f _ X \left( x \right) = \int _ { -\infty } ^ \infty f _ { X,Y } \left( x,y \right) \, dy, \qquad f _ Y \left( y \right) = \int _ { -\infty } ^ \infty f _ { X,Y } \left( x,y \right) \, dx. \]
Independent Continuous Random Variables
Continuous random variables \( X \) and \( Y \) are independent if their joint density factors as \[ f _ { X,Y } \left( x,y \right) = f _ X \left( x \right)f _ Y \left( y \right) \] for all \( x,y \) except possibly on a set of area zero.
Conditional Density Function
If \( X \) and \( Y \) have joint density and \( f _ X \left( x \right) \gt 0 \), the conditional density of \( Y \) given \( X=x \) is \[ f _ { Y \mid X } \left( y \mid x \right) = \frac{f _ { X,Y } \left( x,y \right)}{f _ X \left( x \right)}. \]
Continuous Conditional Expectation Given a Value
If the conditional density \( f _ { Y \mid X } \left( y \mid x \right) \) is defined, then \[ E \left( Y \mid X=x \right) = \int _ { -\infty } ^ \infty y f _ { Y \mid X } \left( y \mid x \right) \, dy, \] whenever the integral is defined.
Continuous Convolution Formula
If \( X \) and \( Y \) are independent continuous random variables with densities \( f _ X \) and \( f _ Y \), then \( X+Y \) has density \[ f _ { X+Y } \left( z \right) = \int _ { -\infty } ^ \infty f _ X \left( x \right)f _ Y \left( z-x \right) \, dx. \]
Integrate the joint density \( f _ X \left( x \right)f _ Y \left( y \right) \) over the region \( x+y \le z \), then differentiate with respect to \( z \).
Change of Variables for Joint Densities
Let \( \left( Y _ 1,Y _ 2 \right)=T\left( X _ 1,X _ 2 \right) \), where \( T \) is one-to-one and differentiable with differentiable inverse. If \( \left( X _ 1,X _ 2 \right) \) has joint density \( f _ X \), then \[ f _ Y \left( y _ 1,y _ 2 \right) = f _ X \left( T ^ { -1 } \left( y _ 1,y _ 2 \right) \right) \left\lvert \det \left[ D\left( T ^ { -1 } \right) \left( y _ 1,y _ 2 \right) \right] _ E \right\rvert . \]
Density Gives Zero Probability to Points
If \( X \) has a density function, then \[ P \left( X = a \right) = 0 \] for every \( a \in \mathbb{ R } \).
Apply the definition of density to the interval \(I=\left[a,a\right]\). The integral of a density over a single point is \(0\), so \(P\left(X=a\right)=0\).
Exponential 1 Distribution
Suppose that \( Z \) is a random variable. We write \( Z \sim \operatorname{ exp } \left( 1 \right) \) iff \[ Z = - \ln \left( U \right) \] where \( U \) is a continuous uniform random variable on \( \left[ 0,1 \right] \).
Probability of an Interval in the Exponential Distribution
Suppose that \( Z \sim \operatorname{ exp } \left( 1 \right) \). Using uniform interval probabilities, \[ P \left( s \le Z \le t \right) = e ^ { -s } - e ^ { -t } \]
Observe that the following sets are equal: \[ \begin{align} s \le Z \le t &\boldsymbol{=} s \le -\ln \left( U \right) \le t \\ &\boldsymbol{=} -t \le \ln \left( U \right) \le -s \\ &\boldsymbol{=} e ^ { -t } \le U \le e ^ { -s } \\ \end{align} \] therefore we know that \[ P \left( s \le Z \le t \right) = P \left( e ^ { -t } \le U \le e ^ { -s } \right) = e ^ { -s } - e ^ { -t } \]

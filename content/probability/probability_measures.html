Probability Space
A probability space is a measure space \( \left( \Omega,\mathcal{ F },P \right) \) such that \( P \left( \Omega \right) = 1 \).
Event
Suppose that \( \left( \Omega,\mathcal{ F },P \right) \) is a probability space. An event is an element of \( \mathcal{ F } \).
Random Variable
Suppose that \( \left( \Omega,\mathcal{ F },P \right) \) is a probability space. A random variable is a measurable function \( X: \Omega \to \mathbb{ R } \).
Distribution of a Random Variable
Suppose that \( X: \Omega \to \mathbb{ R } \) is a random variable. The distribution of \( X \) is the probability measure \( \mu _ X \) on \( \mathbb{ R } \) defined by \[ \mu _ X \left( B \right) := P \left( X ^ { -1 } \left( B \right) \right) \] for every Borel set \( B \subseteq \mathbb{ R } \).
Expectation as an Integral
Suppose that \( X \) is a random variable on \( \left( \Omega,\mathcal{ F },P \right) \). If the integral is defined, the expected value of \( X \) is \[ E \left( X \right) := \int _ \Omega X \, dP . \]
Independence of Events
Events \( A \) and \( B \) are independent if \[ P \left( A \cap B \right) = P \left( A \right) P \left( B \right). \]
Independence of Random Variables
Random variables \( X _ 1,\ldots,X _ n \) are independent if for all Borel sets \( B _ 1,\ldots,B _ n \subseteq \mathbb{ R } \), \[ P \left( X _ 1 \in B _ 1,\ldots,X _ n \in B _ n \right) = \prod _ { k=1 } ^ n P \left( X _ k \in B _ k \right). \]
Monotone Convergence Theorem for Expectations
Suppose that \( 0 \le X _ 1 \le X _ 2 \le \cdots \) are random variables and \( X \left( \omega \right) = \lim _ { n \to \infty } X _ n \left( \omega \right) \). Then \[ E \left( X \right) = \lim _ { n \to \infty } E \left( X _ n \right). \]
Dominated Convergence Theorem for Expectations
Suppose that \( X _ 1,X _ 2,\ldots \) are random variables that converge pointwise to \( X \), and suppose that \( \left\lvert X _ n \right\rvert \le Y \) for every \( n \), where \( E \left( Y \right) \lt \infty \). Then \[ E \left( X \right) = \lim _ { n \to \infty } E \left( X _ n \right). \]
Weak Law of Large Numbers
Suppose that \( X _ 1,X _ 2,\ldots \) are independent identically distributed random variables with finite expected value \( \mu \). Then \[ \frac{1}{n} \sum _ { k=1 } ^ n X _ k \to \mu \] in probability.
Strong Law of Large Numbers
Suppose that \( X _ 1,X _ 2,\ldots \) are independent identically distributed random variables with finite expected value \( \mu \). Then \[ \frac{1}{n} \sum _ { k=1 } ^ n X _ k \to \mu \] almost surely.

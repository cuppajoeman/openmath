Cumulative Distribution Function
Suppose that \( X \) is a random variable. The cumulative distribution function of \( X \) is the function \( F _ X: \mathbb{ R } \to [0,1] \) defined by \[ F _ X \left( x \right) := P \left( X \le x \right). \]
Joint Density Function
Suppose that \( X \) and \( Y \) are real-valued random variables. A function \( p: \mathbb{ R } ^ 2 \to [0,\infty) \) is a joint density function for \( \left( X, Y \right) \) if, for every suitable set \( A \subseteq \mathbb{ R } ^ 2 \), \[ P \left( \left( X, Y \right) \in A \right) = \int _ A p \left( x, y \right) \, d x \, d y. \]
Marginal Density Function
Suppose that \( p \left( x, y \right) \) is a joint density function. The marginal density function of \( X \) is \[ p _ X \left( x \right) := \int _ { -\infty } ^ \infty p \left( x, y \right) \, d y. \]
Conditional Density Function
Suppose that \( p \left( x, y \right) \) is a joint density function and \( p _ X \left( x \right) \gt 0 \). The conditional density function of \( Y \) given \( X = x \) is \[ p _ { Y \mid X } \left( y \mid x \right) := \frac{ p \left( x, y \right) }{ p _ X \left( x \right) }. \]
Joint Density Factors as Marginal Times Conditional Density
Suppose that \( p \left( x, y \right) \) is a joint density. Then \[ p \left( x, y \right) = p _ X \left( x \right) p _ { Y \mid X } \left( y \mid x \right) \] whenever \( p _ X \left( x \right) \gt 0 \).
Estimator
Suppose that \( Q \in \mathbb{ R } \) is a fixed quantity to be estimated. An estimator for \( Q \) is a real-valued random variable \( \widehat{ Q } \) whose values are used as approximations to \( Q \). In practice, \( \widehat{ Q } \) is often a function of random samples, so its value changes with the sampled outcomes while \( Q \) stays fixed.

For example, \( Q \) might be the true average height of every adult in a country. Measuring every adult is unrealistic, so we randomly sample some people and compute a sample average \( \widehat{Q} \). The true average \( Q \) is fixed, but the sample average \( \widehat{Q} \) is random because a different sample can give a different value.

Unbiased Estimator
Suppose that \( \widehat{ Q } \) is an estimator for \( Q \). The estimator \( \widehat{ Q } \) is unbiased if \[ E \left( \widehat{ Q } \right) = Q. \]
Biased Estimator
Suppose that \( \widehat{ Q } \) is an estimator for a fixed \( Q \in \mathbb{ R } \). The bias of \( \widehat{ Q } \) is the difference between the average value of the estimator and the target value: \[ E \left( \widehat{ Q } \right) - Q. \] The estimator is biased if its bias is not \( 0 \).
Monte Carlo Estimator
Suppose that \( D \subseteq \mathbb{ R } ^ n \), \( f: D \to \mathbb{ R } \), and \( p \) is a density function on \( D \) such that \( p \left( x \right) \gt 0 \) whenever \( f \left( x \right) \neq 0 \). If \( X _ 1, \ldots, X _ N \) are independent samples with density \( p \), the Monte Carlo estimator for \[ Q := \int _ D f \left( x \right) \, d x \] is \[ F _ N := \frac{1}{N} \sum _ { i = 1 } ^ N \frac{ f \left( X _ i \right) }{ p \left( X _ i \right) }. \]
Monte Carlo Estimator is Unbiased
Under the hypotheses of the Monte Carlo estimator, \[ E \left( F _ N \right) = \int _ D f \left( x \right) \, d x. \]
By the linearity of expectation for finite sums, \[ E \left( F _ N \right) = E \left( \frac{1}{N} \sum _ { i=1 } ^ N \frac{f \left( X _ i \right)}{p \left( X _ i \right)} \right) = \frac{1}{N} \sum _ { i=1 } ^ N E \left( \frac{f \left( X _ i \right)}{p \left( X _ i \right)} \right). \] Since each \( X _ i \) has density \( p \), the expectation of the weighted sample is \[ E \left( \frac{f \left( X _ i \right)}{p \left( X _ i \right)} \right) = \int _ D \frac{f \left( x \right)}{p \left( x \right)} p \left( x \right) \, dx = \int _ D f \left( x \right) \, dx. \] Therefore \[ E \left( F _ N \right) = \frac{1}{N} \sum _ { i=1 } ^ N \int _ D f \left( x \right) \, dx = \int _ D f \left( x \right) \, dx. \] By the definition of unbiased estimator, \( F _ N \) is unbiased for \( Q \).
Uniform Monte Carlo Estimator is Unbiased
Suppose that \( a,b \in \mathbb{ R } \), \( a \lt b \), and \( f: [a,b] \to \mathbb{ R } \). If \( X _ 1,\ldots,X _ N \) are independent samples from the uniform density on \( [a,b] \), then \[ F _ N := \frac{b-a}{N} \sum _ { i=1 } ^ N f \left( X _ i \right) \] satisfies \[ E \left( F _ N \right) = \int _ a ^ b f \left( x \right) \, dx. \]
The uniform density on \( [a,b] \) is \[ p \left( x \right) = \frac{1}{b-a} \] for \( x \in [a,b] \). By linearity of expectation for finite sums, \[ E \left( F _ N \right) = E \left( \frac{b-a}{N} \sum _ { i=1 } ^ N f \left( X _ i \right) \right) = \frac{b-a}{N} \sum _ { i=1 } ^ N E \left( f \left( X _ i \right) \right). \] Since each \( X _ i \) has density \( p \), expectation of a function of a continuous random variable gives \[ E \left( f \left( X _ i \right) \right) = \int _ a ^ b f \left( x \right)p \left( x \right) \, dx = \int _ a ^ b f \left( x \right)\frac{1}{b-a} \, dx. \] Thus \[ E \left( F _ N \right) = \frac{b-a}{N} \sum _ { i=1 } ^ N \int _ a ^ b f \left( x \right)\frac{1}{b-a} \, dx = \frac{1}{N} \sum _ { i=1 } ^ N \int _ a ^ b f \left( x \right) \, dx = \int _ a ^ b f \left( x \right) \, dx. \]
Variance of an Average of Independent Identically Distributed Random Variables
Suppose that \( X _ 1, \ldots, X _ N \) are independent identically distributed random variables with finite variance. Then \[ \operatorname{ Var } \left( \frac{1}{N} \sum _ { i = 1 } ^ N X _ i \right) = \frac{ \operatorname{ Var } \left( X _ 1 \right) }{ N }. \]
Let \[ S:=\sum _ { i=1 } ^ N X _ i. \] By linearity of expectation, \[ E\left( S\right) = \sum _ { i=1 } ^ N E\left( X_i\right). \] Therefore \[ S-E\left( S\right) = \sum _ { i=1 } ^ N \left( X_i-E\left( X_i\right)\right). \] Now apply the definition of variance: \[ \operatorname{ Var }\left( S\right) = E\left( \left( \sum _ { i=1 } ^ N \left( X_i-E\left( X_i\right)\right) \right)^2 \right). \] To see the square expansion, first use distributivity on an arbitrary finite sum: \[ \left( \sum _ { i=1 } ^ N Y_i \right)^2 = \left( \sum _ { i=1 } ^ N Y_i \right) \left( \sum _ { j=1 } ^ N Y_j \right) = \sum _ { i=1 } ^ N \sum _ { j=1 } ^ N Y_iY_j. \] The terms with \( i=j \) are the diagonal terms \( Y_i^2 \). The terms with \( i\ne j \) occur in pairs: for each \( i\lt j \), the double sum contains both \( Y_iY_j \) and \( Y_jY_i \), and these are equal. Hence \[ \left( \sum _ { i=1 } ^ N Y_i \right)^2 = \sum _ { i=1 } ^ N Y_i^2 + 2 \sum _ { 1\le i\lt j\le N } Y_iY_j. \] Applying this with \( Y_i=X_i-E\left( X_i\right) \) gives \[ \left( \sum _ { i=1 } ^ N \left( X_i-E\left( X_i\right)\right) \right)^2 = \sum _ { i=1 } ^ N \left( X_i-E\left( X_i\right)\right)^2 + 2 \sum _ { 1\le i\lt j\le N } \left( X_i-E\left( X_i\right)\right) \left( X_j-E\left( X_j\right)\right). \] Taking expectations and using linearity of expectation, \[ \operatorname{ Var } \left( \sum _ { i=1 } ^ N X _ i \right) = \sum _ { i=1 } ^ N E\left( \left( X_i-E\left( X_i\right)\right)^2\right) + 2 \sum _ { 1\le i\lt j\le N } E\left( \left( X_i-E\left( X_i\right)\right) \left( X_j-E\left( X_j\right)\right) \right). \] The first expectation is \( \operatorname{ Var}\left( X_i\right) \) by the definition of variance. The second expectation is \( \operatorname{ Cov}\left( X_i,X_j\right) \) by the definition of covariance. Hence \[ \operatorname{ Var } \left( \sum _ { i=1 } ^ N X _ i \right) = \sum _ { i=1 } ^ N \operatorname{ Var } \left( X _ i \right) + 2 \sum _ { 1 \le i \lt j \le N } \operatorname{ Cov } \left( X _ i,X _ j \right). \] Since the random variables are independent, they are uncorrelated, so each covariance term is \( 0 \). Since they are identically distributed, each variance is \( \operatorname{ Var } \left( X _ 1 \right) \). Therefore \[ \operatorname{ Var } \left( \sum _ { i=1 } ^ N X _ i \right) = N \operatorname{ Var } \left( X _ 1 \right). \] Finally, scaling by \( 1/N \) scales variance by \( 1/N^2 \), so \[ \operatorname{ Var } \left( \frac{1}{N}\sum _ { i=1 } ^ N X _ i \right) = \frac{1}{N^2} N\operatorname{ Var } \left( X _ 1 \right) = \frac{\operatorname{ Var } \left( X _ 1 \right)}{N}. \]
Monte Carlo Estimator Variance
Under the hypotheses of the Monte Carlo estimator, define \[ Y := \frac{f \left( X _ 1 \right)}{p \left( X _ 1 \right)}. \] If \( Y \) has finite variance, then \[ \operatorname{ Var } \left( F _ N \right) = \frac{\operatorname{ Var} \left( Y \right)}{N}. \]
The random variables \( f \left( X _ i \right)/p \left( X _ i \right) \) are independent and identically distributed because the samples \( X _ 1,\ldots,X _ N \) are independent and identically distributed. Apply variance of an average of independent identically distributed random variables to \[ F _ N = \frac{1}{N} \sum _ { i=1 } ^ N \frac{f \left( X _ i \right)}{p \left( X _ i \right)}. \]
Mean Squared Error
Suppose that \( \widehat{Q} \) is an estimator for \( Q \). The mean squared error of \( \widehat{Q} \) is \[ \operatorname{ MSE } \left( \widehat{Q} \right) := E \left( \left( \widehat{Q}-Q \right)^2 \right). \]
Unbiased Estimator MSE is Variance
If \( \widehat{Q} \) is an unbiased estimator for \( Q \), then \[ \operatorname{ MSE } \left( \widehat{Q} \right) = \operatorname{ Var } \left( \widehat{Q} \right). \]
Since \( \widehat{Q} \) is unbiased, \( E \left( \widehat{Q} \right)=Q \). Substituting this into the mean squared error gives \[ \operatorname{ MSE } \left( \widehat{Q} \right) = E \left( \left( \widehat{Q}-E \left( \widehat{Q} \right) \right)^2 \right), \] which is exactly the variance of \( \widehat{Q} \).
Monte Carlo Efficiency
If \( F \) is an estimator with finite positive variance and expected runtime \( T \left( F \right) \), its Monte Carlo efficiency is \[ \epsilon \left[ F \right] := \frac{1} {\operatorname{ Var } \left( F \right) T \left( F \right)}. \]
Sample Variance
Suppose that \( x _ 1, \ldots, x _ N \in \mathbb{ R } \), with \( N \ge 2 \), and let \[ \overline{x} := \frac{1}{N} \sum _ { i = 1 } ^ N x _ i. \] The sample variance of \( x _ 1, \ldots, x _ N \) is \[ s ^ 2 := \frac{1}{N - 1} \sum _ { i = 1 } ^ N \left( x _ i - \overline{x} \right) ^ 2. \]
Inversion Method
Suppose that \( F \) is a cumulative distribution function that has an inverse \( F ^ { -1 } \), and suppose that \( U \sim \operatorname{ unif } [0,1] \). The inversion method samples from \( F \) by setting \[ X := F ^ { -1 } \left( U \right). \]

The inversion method is one way to turn a uniform random number into a random variable with a desired distribution. Start with \(U\), which is uniformly distributed on \([0,1]\). If the desired cumulative distribution function is \(F\), then the interval \([0,F(x)]\) has length \(F(x)\), so \(P\left(U\le F(x)\right)=F(x)\). Choosing \(X=F^{-1}(U)\) makes the event \(X\le x\) match the event \(U\le F(x)\). Thus the transformation \(F^{-1}\) reshapes uniform randomness so that the output \(X\) has distribution \(F\).

Discrete Inversion Method
Suppose that \( x _ 1,x _ 2,\ldots \) are possible values with probabilities \( p _ i \ge 0 \) and \[ \sum _ i p _ i = 1. \] Let \[ P _ i := \sum _ { j=1 } ^ i p _ j. \] If \( U \sim \operatorname{ unif }[0,1] \), the discrete inversion method returns \( x _ i \) when \[ P _ { i-1 } \le U \lt P _ i, \] where \( P _ 0 := 0 \).
Discrete Inversion Method Has the Target Probabilities
Under the hypotheses of the discrete inversion method, \[ P \left( X=x _ i \right) = p _ i. \]
The event \( \left\{ X=x _ i \right\} \) is the event \( \left\{ P _ { i-1 } \le U \lt P _ i \right\} \). Since \( U \) is uniform on \( [0,1] \), the probability of this interval is its length: \[ P \left( X=x _ i \right) = P _ i-P _ { i-1 } = \sum _ { j=1 } ^ i p _ j - \sum _ { j=1 } ^ { i-1 } p _ j = p _ i. \]
Inversion Method Samples from the Target Distribution
Suppose that \( F \) is an invertible cumulative distribution function, \( U \sim \operatorname{ unif } [0,1] \), and \( X = F ^ { -1 } \left( U \right) \). Then \( X \) has cumulative distribution function \( F \).
For any \( x \in \mathbb{ R } \), \[ P \left( X \le x \right) = P \left( F ^ { -1 } \left( U \right) \le x \right). \] Since a cumulative distribution function is non-decreasing, applying \( F \) to both sides gives the equivalent event \( \left\{ U \le F \left( x \right) \right\} \). Because \( U \sim \operatorname{ unif }[0,1] \), \[ P \left( F ^ { -1 } \left( U \right) \le x \right) = P \left( U \le F \left( x \right) \right) = F \left( x \right). \] Therefore the cumulative distribution function of \( X \) is \( F \).

For example, suppose we want density \(p(x)=2x\) on \([0,1]\). Its cumulative distribution function is \(F(x)=x^2\). If \(U\) is uniform on \([0,1]\), set \(X=\sqrt{U}\). Then \[ P\left(X\le x\right) = P\left(\sqrt{U}\le x\right) = P\left(U\le x^2\right) = x^2, \] so \(X\) has density \(2x\). The random number generator only produced a uniform value; the transformation \(u\mapsto\sqrt{u}\) changed how often different output values occur.

Linear Function Density
Suppose that \( a,b \in [0,\infty) \), not both \( 0 \), and define \( f: [0,1] \to \mathbb{ R } \) by \[ f \left( x \right) = \left( 1-x \right)a+xb. \] The normalized density proportional to \( f \) is \[ p \left( x \right) = \frac{2f \left( x \right)}{a+b}. \]
Inversion Sampling for a Linear Function Density
Under the hypotheses of linear function density, if \( U \sim \operatorname{ unif }[0,1] \), then a sample from density \( p \) is \[ X = \frac {a-\sqrt{\left( 1-U \right)a^2+Ub^2}} {a-b}, \] when \( a \ne b \). Equivalently, to avoid cancellation, \[ X = \frac{U\left( a+b \right)} {a+\sqrt{\left( 1-U \right)a^2+Ub^2}}. \]
The cumulative distribution function of \( p \) is \[ F \left( x \right) = \int _ 0 ^ x \frac{2\left( \left( 1-t \right)a+tb \right)}{a+b} \, dt = \frac{x\left( a\left( 2-x \right)+bx \right)}{a+b}. \] Setting \( U=F \left( X \right) \) gives a quadratic equation in \( X \). Solving the quadratic and choosing the root in \( [0,1] \) gives \[ X = \frac {a-\sqrt{\left( 1-U \right)a^2+Ub^2}} {a-b}. \] Multiplying numerator and denominator by \( a+\sqrt{\left( 1-U \right)a^2+Ub^2} \) gives the equivalent form.
Change of Variables for Density Functions
Suppose that \( X \) is an \( n \)-dimensional random variable with density \( p _ X \), and suppose that \( T: \mathbb{ R } ^ n \to \mathbb{ R } ^ n \) is a differentiable bijection whose Jacobian determinant is nonzero. If \( Y = T \left( X \right) \), then the density of \( Y \) is \[ p _ Y \left( y \right) = \frac{ p _ X \left( T ^ { -1 } \left( y \right) \right) } { \left\lvert \det \left[ DT \left( T ^ { -1 } \left( y \right) \right) \right] _ E \right\rvert }. \]
For every suitable set \( A \subseteq \mathbb{ R } ^ n \), \[ P \left( Y \in A \right) = P \left( X \in T ^ { -1 } \left( A \right) \right) = \int _ { T ^ { -1 } \left( A \right) } p _ X \left( x \right) \, dx. \] By the multivariable change of variables formula, with \( y=T\left( x \right) \), \[ \int _ { T ^ { -1 } \left( A \right) } p _ X \left( x \right) \, dx = \int _ A \frac{ p _ X \left( T ^ { -1 } \left( y \right) \right) } { \left\lvert \det \left[ DT \left( T ^ { -1 } \left( y \right) \right) \right] _ E \right\rvert } \, dy. \] Since this holds for every suitable \( A \), the displayed integrand is the density of \( Y \).
One Dimensional Density Transformation
Suppose that \( X \) has density \( p _ X \), and \( T: \mathbb{ R } \to \mathbb{ R } \) is differentiable and one-to-one with \( T ^ \prime \left( x \right) \ne 0 \). If \( Y=T\left( X \right) \), then \[ p _ Y \left( y \right) = \frac{p _ X \left( T ^ { -1 } \left( y \right) \right)} {\left\lvert T ^ \prime \left( T ^ { -1 } \left( y \right) \right) \right\rvert}. \]
This is the one-dimensional case of change of variables for density functions, where the Jacobian determinant is \( T ^ \prime \).
Polar Coordinate Density Transformation
Under the polar coordinate map \( T \left( r,\theta \right)=\left( r\cos\theta,r\sin\theta \right) \), \[ p _ { R,\Theta } \left( r,\theta \right) = r\,p _ { X,Y } \left( r\cos\theta,r\sin\theta \right). \]
The absolute value of the Jacobian determinant of the polar coordinate map is \( r \). Apply change of variables for density functions.
Spherical Coordinate Density Transformation
Under the spherical coordinate map \( T \left( r,\theta,\phi \right)=\left( r\sin\theta\cos\phi,r\sin\theta\sin\phi,r\cos\theta \right) \), \[ p _ { R,\Theta,\Phi } \left( r,\theta,\phi \right) = r^2\sin\theta\, p _ { X,Y,Z } \left( T \left( r,\theta,\phi \right) \right). \]
The absolute value of the Jacobian determinant of the spherical coordinate map is \( r^2\sin\theta \). Apply change of variables for density functions.
Stratified Sampling
Suppose that a domain \( D \) is partitioned into disjoint measurable sets \( D _ 1, \ldots, D _ m \). Stratified sampling is the practice of estimating an integral over \( D \) by taking samples separately within each stratum \( D _ i \) and combining the resulting estimates.
Stratified Monte Carlo Estimator
Suppose that \( D \) is partitioned into measurable strata \( D _ 1,\ldots,D _ m \), with \( v _ i \) denoting the measure of \( D _ i \). If \( F _ i \) is a Monte Carlo estimator for \[ \frac{1}{v _ i}\int _ { D _ i } f \left( x \right)\,dx, \] then the stratified Monte Carlo estimator is \[ F := \sum _ { i=1 } ^ m v _ i F _ i. \]
Stratified Sampling Does Not Increase Variance
Suppose that \( D \) is partitioned into measurable strata \( D _ 1,\ldots,D _ m \), and samples are allocated proportionally to stratum measure: \( n _ i = v _ iN \). If \( \mu _ i \) and \( \sigma _ i ^ 2 \) are the mean and variance of \( f \) under the uniform distribution on \( D _ i \), and \( Q \) is the mean of \( f \) under the uniform distribution on \( D \), then the stratified estimator has variance \[ \operatorname{ Var } \left( F \right) = \frac{1}{N} \sum _ { i=1 } ^ m v _ i\sigma _ i ^ 2, \] while the unstratified estimator has variance \[ \frac{1}{N} \left( \sum _ { i=1 } ^ m v _ i\sigma _ i ^ 2 + \sum _ { i=1 } ^ m v _ i\left( \mu _ i-Q \right)^2 \right). \] Hence stratification does not increase variance.
For stratum \( D _ i \), averaging \( n _ i \) independent samples gives variance \( \sigma _ i ^ 2/n _ i \) by variance of an average of independent identically distributed random variables. The stratified estimator is \( F=\sum_i v_iF_i \), so independence of samples across strata gives \[ \operatorname{ Var } \left( F \right) = \sum _ { i=1 } ^ m v _ i ^ 2 \operatorname{ Var } \left( F _ i \right) = \sum _ { i=1 } ^ m v _ i ^ 2 \frac{\sigma _ i ^ 2}{n _ i}. \] Substituting \( n _ i=v _ iN \) gives the displayed stratified variance.

For unstratified sampling, the total variance decomposes into the average within-stratum variance plus the variance of the stratum means:

\[ \operatorname{ Var } \left( f\left( X \right) \right) = \sum _ { i=1 } ^ m v _ i\sigma _ i ^ 2 + \sum _ { i=1 } ^ m v _ i\left( \mu _ i-Q \right)^2. \] Averaging \( N \) independent samples divides this by \( N \). Since each term \( v _ i\left( \mu _ i-Q \right)^2 \ge 0 \), the stratified variance is no larger.
Importance Sampling
Importance sampling estimates an integral \[ \int _ D f \left( x \right) \, d x \] by drawing samples from a density \( p \) and using the weighted terms \[ \frac{ f \left( X \right) }{ p \left( X \right) }. \] The density \( p \) should be positive wherever \( f \) is nonzero.

For example, to estimate \( \int_0^1 x^2\,dx \), ordinary uniform sampling uses \( X\in[0,1] \) with density \( p\left(x\right)=1 \), so each weighted term is \( X^2 \). Importance sampling could instead use the density \( p\left(x\right)=2x \), which samples more often near \(1\), where \(x^2\) is larger. Then each weighted term is \( X^2 / \left(2X\right)=X/2 \), for samples \(X\gt0\).

Ideal Importance Sampling Has Zero Variance for Nonnegative Integrands
Suppose that \( f:D\to[0,\infty) \) and \( 0 \lt \int _ D f\left( x\right)\,dx \lt \infty \). If \[ p \left( x \right) = \frac{f \left( x \right)} {\int _ D f \left( t \right)\,dt}, \] then each importance sampling term satisfies \[ \frac{f\left( X \right)}{p\left( X \right)} = \int _ D f \left( t \right)\,dt. \] Hence the Monte Carlo estimator has zero variance.
By the definition of \( p \), \[ \frac{f\left( X \right)}{p\left( X \right)} = \frac{f\left( X \right)} {f\left( X \right)/\int_D f\left( t\right)\,dt} = \int _ D f\left( t\right)\,dt, \] whenever \( f\left( X\right)\ne 0 \). When \( f\left( X\right)=0 \), this value occurs only where the density also vanishes and does not affect the estimator. Thus the estimator is constant almost surely, and a constant random variable has variance \( 0 \).
Multiple Importance Sampling
Multiple importance sampling estimates an integral by combining samples drawn from several densities \( p _ 1, \ldots, p _ m \). If \( w _ i \) is a weight function for density \( p _ i \), then a typical weighted term has the form \[ w _ i \left( X \right) \frac{ f \left( X \right) }{ p _ i \left( X \right) }. \]
Multiple Importance Sampling Estimator
Suppose that \( p _ 1,\ldots,p _ m \) are sampling densities and that \( X _ { i,j } \) is sampled from \( p _ i \), for \( 1\le j\le n _ i \). The multiple importance sampling estimator is \[ F := \sum _ { i=1 } ^ m \frac{1}{n _ i} \sum _ { j=1 } ^ { n _ i } w _ i \left( X _ { i,j } \right) \frac{f \left( X _ { i,j } \right)} {p _ i \left( X _ { i,j } \right)}. \]
Multiple Importance Sampling Unbiasedness Conditions
The multiple importance sampling estimator is unbiased for \( \int_D f\left( x\right)\,dx \) if, whenever \( f\left( x\right)\ne 0 \), \[ \sum _ { i=1 } ^ m w _ i \left( x \right)=1, \] and \( w _ i\left( x\right)=0 \) whenever \( p _ i\left( x\right)=0 \).
Using linearity of expectation and the definition of density, \[ E \left( F \right) = \sum _ { i=1 } ^ m \int _ D w _ i \left( x \right) \frac{f \left( x \right)}{p _ i \left( x \right)} p _ i \left( x \right) \, dx. \] The condition \( w_i\left( x\right)=0 \) where \( p_i\left( x\right)=0 \) makes the terms well-defined, and the remaining factors cancel: \[ E \left( F \right) = \int _ D f \left( x \right) \sum _ { i=1 } ^ m w _ i \left( x \right) \, dx = \int _ D f \left( x \right)\,dx. \]
Balance Heuristic
Suppose that \( p _ 1, \ldots, p _ m \) are sampling densities and \( n _ i \) samples are drawn from \( p _ i \). The balance heuristic assigns the weight \[ w _ i \left( x \right) := \frac{ n _ i p _ i \left( x \right) } { \sum _ j n _ j p _ j \left( x \right) }. \]
Balance Heuristic Weights Sum to One
If \( \sum_j n_jp_j\left( x\right)\ne 0 \), the balance heuristic weights satisfy \[ \sum _ { i=1 } ^ m w _ i \left( x \right)=1. \]
Substitute the definition: \[ \sum _ { i=1 } ^ m w _ i \left( x \right) = \sum _ { i=1 } ^ m \frac{n_i p_i\left( x\right)} {\sum_j n_jp_j\left( x\right)} = \frac{\sum_i n_i p_i\left( x\right)} {\sum_j n_jp_j\left( x\right)} = 1. \]
Power Heuristic
Suppose that \( p _ 1, \ldots, p _ m \) are sampling densities, \( n _ i \) samples are drawn from \( p _ i \), and \( \beta \in \mathbb{ R } ^ + \). The power heuristic assigns the weight \[ w _ i \left( x \right) := \frac{ \left( n _ i p _ i \left( x \right) \right) ^ \beta } { \sum _ j \left( n _ j p _ j \left( x \right) \right) ^ \beta }. \]
MIS Compensation
Given a density \( p \) and \( \delta \ge 0 \), the MIS compensated density is \[ p ^ \prime \left( x \right) := \frac{\max \left( 0,p\left( x\right)-\delta \right)} {\int_D \max \left( 0,p\left( t\right)-\delta \right)\,dt}, \] whenever the denominator is positive.
Russian Roulette Estimator
Suppose that \( X \) is an unbiased estimator for \( Q \), and let \( B \) be a Bernoulli random variable with \( P \left( B = 1 \right) = q \gt 0 \), independent of \( X \). The Russian roulette estimator is \[ Y := \begin{cases} X / q & B = 1, \\ 0 & B = 0. \end{cases} \]
Russian Roulette Estimator is Unbiased
The Russian roulette estimator is unbiased: \[ E \left( Y \right) = Q. \]
By conditioning on the two values of the Bernoulli random variable \( B \), \[ E \left( Y \right) = P \left( B=1 \right)E \left( X/q \right) + P \left( B=0 \right)E \left( 0 \right). \] Since \( P\left( B=1\right)=q \) and constants pull out of expectation, \[ E \left( Y \right) = q\frac{1}{q}E\left( X\right) = E\left( X\right). \] Since \( X \) is unbiased for \( Q \), \( E\left( X\right)=Q \).
Russian Roulette Estimator with a Fallback Value
Suppose that \( F \) is an estimator, \( q \in [0,1) \), \( c \in \mathbb{ R } \), and \( U\sim\operatorname{ unif }[0,1] \). The Russian roulette estimator with fallback value is \[ F ^ \prime := \begin{cases} \dfrac{F-qc}{1-q}, & U \gt q,\\ c, & U \le q. \end{cases} \]
Russian Roulette with a Fallback Value Preserves Expectation
Under the hypotheses of Russian roulette estimator with a fallback value, \[ E \left( F ^ \prime \right) = E \left( F \right). \]
Since \( P\left( U\le q\right)=q \) and \( P\left( U\gt q\right)=1-q \), \[ E \left( F ^ \prime \right) = \left( 1-q \right) E \left( \frac{F-qc}{1-q} \right) + q c. \] Pulling constants out of expectation gives \[ E \left( F ^ \prime \right) = E \left( F \right)-qc+qc = E \left( F \right). \]
Splitting Estimator
Suppose that \( X _ 1,\ldots,X _ n \) are samples from a density \( p_X \), and for each \( X_i \), the conditional samples \( Y _ { i,1},\ldots,Y _ { i,m } \) are drawn from a conditional density \( p _ { Y \mid X } \left( y \mid X_i \right) \). The splitting estimator for \[ \int_A \int_B f\left( x,y\right)\,dy\,dx \] is \[ F := \frac{1}{n} \sum _ { i=1 } ^ n \frac{1}{m} \sum _ { j=1 } ^ m \frac{f\left( X_i,Y_{i,j}\right)} {p_X\left( X_i\right)p_{Y\mid X}\left( Y_{i,j}\mid X_i\right)}. \]
Splitting Estimator is Unbiased
Under the hypotheses of the splitting estimator, \[ E \left( F \right) = \int_A \int_B f\left( x,y\right)\,dy\,dx. \]
By linearity of expectation, \[ E\left( F\right) = \frac{1}{n} \sum _ { i=1 } ^ n \frac{1}{m} \sum _ { j=1 } ^ m E\left( \frac{f\left( X_i,Y_{i,j}\right)} {p_X\left( X_i\right)p_{Y\mid X}\left( Y_{i,j}\mid X_i\right)} \right). \] For each \( i,j \), use the joint density factorization \( p\left( x,y\right)=p_X\left( x\right)p_{Y\mid X}\left( y\mid x\right) \): \[ E\left( \frac{f\left( X_i,Y_{i,j}\right)} {p_X\left( X_i\right)p_{Y\mid X}\left( Y_{i,j}\mid X_i\right)} \right) = \int_A\int_B \frac{f\left( x,y\right)} {p_X\left( x\right)p_{Y\mid X}\left( y\mid x\right)} p_X\left( x\right)p_{Y\mid X}\left( y\mid x\right) \,dy\,dx. \] The density factors cancel, so each summand has expectation \( \int_A\int_B f\left( x,y\right)\,dy\,dx \). Averaging \( nm \) identical expectations leaves that same integral.

Random Variable
A random variable is a function from the sample space \( \Omega \) to \( \mathbb{ R } \).
Random Variable is Real-Valued with Probability One
If \(X:\Omega\to\mathbb{R}\) is a random variable, then \[ P\left(X\in\mathbb{R}\right)=1. \]
Since \(X\) is a function from \(\Omega\) to \(\mathbb{R}\), every \(\omega\in\Omega\) satisfies \(X\left(\omega\right)\in\mathbb{R}\). Therefore \[ \left\{\omega\in\Omega:X\left(\omega\right)\in\mathbb{R}\right\} = \Omega. \] Hence \(P\left(X\in\mathbb{R}\right)=P\left(\Omega\right)=1\) by the definition of a probability measure.
Random Variable as a Measurable Function
On a probability space \( \left( \Omega,\mathcal{ F },P \right) \), a real-valued random variable is a function \( X: \Omega \to \mathbb{ R } \) such that \( \left\{ \omega \in \Omega : X \left( \omega \right) \le x \right\} \in \mathcal{ F } \) for every \( x \in \mathbb{ R } \).
Random Variable and Binary Relation Syntax Sugar
Suppose that \( X : \Omega \to \mathbb{ R } \) is a random variable and that \( \star \) is a binary relation on \( \mathbb{ R } \). Then for any \( a \in \mathbb{ R } \) we define \[ \left\{ X \star a \right\} := \left\{ \omega \in \Omega : X \left( \omega \right) \star a \right\}. \]

Since \( \left\{ X \star a \right\} \) is an event, we can take its probability. When the meaning is clear, we use the syntactic reduction

\[ P \left( X \star a \right) := P \left( \left\{ X \star a \right\} \right). \]
Distribution Function
The distribution function of a random variable \( X \) is the function \( F _ X : \mathbb{ R } \to \left[ 0,1 \right] \) defined by \[ F _ X \left( x \right) := P \left( X \le x \right). \]
Distribution Function Properties
If \( F \) is the distribution function of a random variable, then \( F \) is non-decreasing, right-continuous, \( \lim _ { x \to -\infty } F \left( x \right) = 0 \), and \( \lim _ { x \to \infty } F \left( x \right) = 1 \).
Let \( F = F _ X \). If \( x \le y \), then every \( \omega \in \left\{ X \le x \right\} \) satisfies \( X \left( \omega \right) \le x \le y \), so \( \omega \in \left\{ X \le y \right\} \). Thus \( \left\{ X \le x \right\} \subseteq \left\{ X \le y \right\} \), and monotonicity of probability gives \[ F \left( x \right) = P \left( X \le x \right) \le P \left( X \le y \right) = F \left( y \right). \] Therefore \( F \) is non-decreasing.

For the limit at infinity, define \( A _ n := \left\{ X \le n \right\} \). Then \( A _ n \nearrow \Omega \), because every real value \( X \left( \omega \right) \) is at most some integer \( n \). By continuity of probability,

\[ \lim _ { n \to \infty } F \left( n \right) = \lim _ { n \to \infty } P \left( A _ n \right) = P \left( \Omega \right) = 1. \] Since \( F \) is non-decreasing and \( F \left( x \right) \le 1 \), integer values determine the monotone function limit at infinity gives \( \lim _ { x \to \infty } F \left( x \right) = 1 \).

For the limit at negative infinity, define \( B _ n := \left\{ X \le -n \right\} \). Then \( B _ n \searrow \emptyset \): the events are decreasing, and no real number is less than or equal to \( -n \) for every \( n \). By continuity of probability and probability of the impossible event,

\[ \lim _ { n \to \infty } F \left( -n \right) = \lim _ { n \to \infty } P \left( B _ n \right) = P \left( \emptyset \right) = 0. \] Since \( F \) is non-decreasing and \( F \left( x \right) \ge 0 \), this implies \( \lim _ { x \to -\infty } F \left( x \right) = 0 \).

Finally, fix \( x \in \mathbb{ R } \) and define \( C _ n := \left\{ X \le x + \frac{1}{n} \right\} \). Then \( C _ n \searrow \left\{ X \le x \right\} \): the events are decreasing, and \( \omega \) lies in every \( C _ n \) exactly when \( X \left( \omega \right) \le x \). By continuity of probability,

\[ \lim _ { n \to \infty } F \left( x + \frac{1}{n} \right) = \lim _ { n \to \infty } P \left( C _ n \right) = P \left( X \le x \right) = F \left( x \right). \] Since \( F \) is non-decreasing, the values \( F \left( x + h \right) \) with \( h \to 0 ^ + \) are squeezed between \( F \left( x \right) \) and values of the form \( F \left( x + \frac{1}{n} \right) \) with \( n \to \infty \). Hence \( \lim _ { h \to 0 ^ + } F \left( x + h \right) = F \left( x \right) \), so \( F \) is right-continuous.
Probability from Distribution Function
If \( F _ X \) is the distribution function of \( X \), then for \( a \lt b \), \[ P \left( a \lt X \le b \right) = F _ X \left( b \right) - F _ X \left( a \right). \]
Let \[ E _ a := \left\{ X \le a \right\} \quad \text{and} \quad E _ b := \left\{ X \le b \right\}. \] Since \( a \lt b \), if \( \omega \in E _ a \), then \( X \left( \omega \right) \le a \lt b \), so \( \omega \in E _ b \). Thus \( E _ a \subseteq E _ b \).

By set difference decomposition, \( E _ b \) is the disjoint union of \( E _ b \setminus E _ a \) and \( E _ b \cap E _ a \). Also,

\[ E _ b \setminus E _ a = \left\{ a \lt X \le b \right\}, \] because \( \omega \in E _ b \setminus E _ a \) exactly when \( X \left( \omega \right) \le b \) and it is not true that \( X \left( \omega \right) \le a \), which is the same as \( a \lt X \left( \omega \right) \le b \). Since \( E _ a \subseteq E _ b \), intersection with a superset is itself gives \( E _ a \cap E _ b = E _ a \).

Therefore, by probability of set difference,

\[ P \left( a \lt X \le b \right) = P \left( E _ b \setminus E _ a \right) = P \left( E _ b \right) - P \left( E _ b \cap E _ a \right) = P \left( E _ b \right) - P \left( E _ a \right) = F _ X \left( b \right) - F _ X \left( a \right). \]
Random Variable Function Composition
Suppose that \( X: \Omega \to D \) is a random variable and that \( g: D \to Y \) is a function. Then \( g \left( X \right) \) denotes the composition \[ g \left( X \right) := g \circ X, \] so \( g \left( X \right): \Omega \to Y \) is the function defined by \[ \left( g \left( X \right) \right) \left( \omega \right) = g \left( X \left( \omega \right) \right). \]

Note that when we compose a measurable real-valued function with a random variable, we obtain a new random variable.

Discrete Random Variable
A random variable \( X \) is discrete if the set of values \( X \left( \Omega \right) \) is countable.
Identically Distributed Random Variables
Random variables \( X \) and \( Y \) are identically distributed if their distribution functions agree: \( P \left( X \le x \right) = P \left( Y \le x \right) \) for every \( x \in \mathbb{ R } \).
Independent Random Variables
Given two random variables \( X,Y \) on the same sample space, we say that \( X \) is independent of \( Y \) and write \( X \perp Y \) iff for any \( A,B \subseteq \mathbb{ R } \) with \( P \left( Y \in B \right) \gt 0 \), the conditional probability satisfies \[ P \left( X \in A \mid Y \in B \right) = P \left( X \in A \right). \]
Multivariate Random Variable
Suppose that we have a finite collection of random variables \( X _ i \) each with their own sample space \( \Omega _ i \). Then the tuple \[ \textbf{ X } = \left( X _ 1, \ldots X _ n \right) \] is said to be a multivariate random variable and is a function from \( \Omega _ 1 \times \cdots \times \Omega _ n \to \mathbb{ R } ^ n\).
Joint Distribution Function
The joint distribution function of a random vector \( \left( X _ 1,\ldots,X _ n \right) \) is \[ F \left( x _ 1,\ldots,x _ n \right) = P \left( X _ 1 \le x _ 1,\ldots,X _ n \le x _ n \right). \]
Independent Random Variables by Events
Random variables \( X _ 1,\ldots,X _ n \) are independent if the events \( \left\{ X _ 1 \in B _ 1 \right\},\ldots,\left\{ X _ n \in B _ n \right\} \) are independent for every choice of Borel sets \( B _ 1,\ldots,B _ n \subseteq \mathbb{ R } \).
Functions of Independent Random Variables are Independent
If \( X \) and \( Y \) are independent random variables and \( g,h: \mathbb{ R } \to \mathbb{ R } \) are Borel measurable functions, then \( g \left( X \right) \) and \( h \left( Y \right) \) are independent.
Let \( A,B \subseteq \mathbb{ R } \) be Borel sets. Since \( g \) and \( h \) are Borel measurable, the inverse images \( g ^ { -1 } \left( A \right) \) and \( h ^ { -1 } \left( B \right) \) are Borel sets.

By the definition of random variable function composition and inverse image of a composition,

\[ \left\{ g \left( X \right) \in A \right\} = \left\{ X \in g ^ { -1 } \left( A \right) \right\} \] and \[ \left\{ h \left( Y \right) \in B \right\} = \left\{ Y \in h ^ { -1 } \left( B \right) \right\}. \] Because \( X \) and \( Y \) are independent random variables and \( g ^ { -1 } \left( A \right) \), \( h ^ { -1 } \left( B \right) \) are Borel sets, we have \[ \begin{aligned} P \left( \left\{ g \left( X \right) \in A \right\} \cap \left\{ h \left( Y \right) \in B \right\} \right) &= P \left( \left\{ X \in g ^ { -1 } \left( A \right) \right\} \cap \left\{ Y \in h ^ { -1 } \left( B \right) \right\} \right) \\ &= P \left( X \in g ^ { -1 } \left( A \right) \right) P \left( Y \in h ^ { -1 } \left( B \right) \right) \\ &= P \left( g \left( X \right) \in A \right) P \left( h \left( Y \right) \in B \right). \end{aligned} \] Therefore \( g \left( X \right) \) and \( h \left( Y \right) \) are independent by the definition of independent random variables.

Note that above simply says that given any \( A, B \subseteq R \) the events \( E _ A := X \in A \) and \( E _ B := Y \in B \)

Independence as a Product
By independence of random variables, \[ A \perp B \iff P \left( X \in A \cap X \in B \right) = P \left( X \in A \right) P \left( X \in B \right) \]
Independence is Reflexive
\[ A \perp B \iff B \perp A \]
Follows from the previous corollary.
Independence Practice
Suppose that \( A, B \) are two events such that \( P \left( A \setminus B \right) = P \left( B \right) = \frac{2}{5} \) and \( A \perp B \):
  • Determine \( P \left( A \right) \)
  • Obtain the probabilities \( P \left( A \cap B \right), P \left( A \cap B ^ \complement \right) , P \left( A ^ \complement \cap B ^ \complement \right) \)
  • Determine the conditional probability \( P \left( A \setminus B \mid A \cap B \right) \)

We start by determining \( P \left( A \right) \), firstly we know that \[ \begin{align} \frac{2}{5} &= P \left( B \right) = P \left( A \setminus B \right) \\ &= P \left( A \right) - P \left( A \cap B \right) \\ &= P \left( A \right) - P \left( A \right) P \left( B \right) \\ &= P \left( A \right) \left( 1 - P \left( B \right) \right) \\ &= P \left( A \right) \left( 1 - \frac{2}{5} \right) \\ &= P \left( A \right) \left( \frac{3}{5} \right) \end{align} \] therefore we deduce that \( P \left( A \right) = \frac{2}{3} \)

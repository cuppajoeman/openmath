Expected Value
Suppose that \( X \) is a discrete random variable. Then we define \[ E \left( X \right) := \sum x P \left( X = x \right) \]
Expectation of a Function of a Discrete Random Variable
If \( X \) is a discrete random variable with probability mass function \( p _ X \) and \( g: \mathbb{ R } \to \mathbb{ R } \), then \[ E \left( g \left( X \right) \right) = \sum _ x g \left( x \right) p _ X \left( x \right), \] whenever the sum is defined.
Moment
The \( k \)-th moment of a random variable \( X \) is defined using expected value as \( E \left( X ^ k \right) \), whenever this expectation exists.
Squared Expectation
Suppose \( X \) is a random variable such that \( P \left( X = -2 \right) = 0.2, P \left( X = 1 \right) = 0.5, P \left( X = 2 \right) = 0.3 \) and \( P \left( X = a \right) = 0 \) for all other values. Compute \( E \left( X ^ 2 \right) \) and show it's not equal to \( E \left( X \right) ^ 2 \)

Recall that \( X ^ 2 \) is simply the composition of the function \( X \) and the function \( \cdot ^ 2 \), denote \( Y := X ^ 2 \) so that \( Y \) may only take on values \( 1 \) and \( 4 \) and also \( P \left( Y = 4 \right) = P \left( \left( X = 2 \right) \cup \left( X = -2 \right) \right) = P \left( X = 2 \right) + P \left( X = -2 \right) = 0.2 + 0.3 = 0.5 \) and for all other values \( P \left( Y = a \right) = 0 \) \[ E \left( X ^ 2 \right) = E \left( Y \right) = 4 \cdot 0.5 + 1 \cdot 0.5 = 2 + .5 = 2.5 \] on the other hand \( E \left( X \right) = -2 \cdot 0.2 + 1 \cdot 0.5 + 2 \cdot 0.3 = 0.7 \) so that \( E \left( X \right) ^ 2 = 0. 49 \) so \( E \left( X \right) ^ 2 \neq E \left( X ^ 2 \right) \)

Expectation is Linear
Suppose that \( X \) is a random variable and that \( a, b \in \mathbb{ R } \). Then \( a X + b \) is also a random variable and \[ E \left( a X + b \right) = a E \left( X \right) + b \]
Expectation is Linear for Finite Sums
If \( X _ 1,\ldots,X _ n \) have finite expectations and \( a _ 1,\ldots,a _ n \in \mathbb{ R } \), then \[ E \left( \sum _ { k=1 } ^ n a _ k X _ k \right) = \sum _ { k=1 } ^ n a _ k E \left( X _ k \right). \]
Let \[ Y := \sum _ { k=1 } ^ n a _ k X _ k. \] By expectation as an integral, \[ E\left( Y\right) = \int _ \Omega \left( \sum _ { k=1 } ^ n a _ k X _ k \right) dP. \] We first handle two summands. If \( U \) and \( V \) have finite expectations, then by additivity of integration, \[ E\left( U+V\right) = \int _ \Omega \left( U+V\right)dP = \int _ \Omega U\,dP+\int _ \Omega V\,dP = E\left( U\right)+E\left( V\right). \] Also, for \( a\in\mathbb{ R } \), \[ E\left( aU\right) = \int _ \Omega aU\,dP = a\int _ \Omega U\,dP = aE\left( U\right), \] because multiplying a function by a constant multiplies its integral by that constant. Applying these two facts repeatedly to the finite sum gives \[ E \left( \sum _ { k=1 } ^ n a _ k X _ k \right) = \sum _ { k=1 } ^ n E\left( a_kX_k\right) = \sum _ { k=1 } ^ n a_kE\left( X_k\right). \]
Variance
Let \( X \) be a random variable. The variance of \( X \) is \[ \operatorname{ Var } \left( X \right) := E \left( \left( X - E \left( X \right) \right) ^ 2 \right), \] whenever this expected value is defined.
Variance as Squares of Expectation
From the definition of variance, \[ \operatorname{ Var } \left( X \right) = E \left( X ^ 2 \right) - E \left( X \right) ^ 2 \]
Variance is Not Linear
By the definition of variance, \[ \operatorname{ Var } \left( a X + b \right) = a ^ 2 \operatorname{ Var } \left( X \right) \]
Covariance
If \( X \) and \( Y \) have finite second moments, their covariance is \[ \operatorname{ Cov } \left( X,Y \right) := E \left( \left( X - E \left( X \right) \right)\left( Y - E \left( Y \right) \right) \right). \]
Covariance as Expectation Product
If \( X \) and \( Y \) have finite second moments, then by covariance, \[ \operatorname{ Cov } \left( X,Y \right) = E \left( XY \right) - E \left( X \right)E \left( Y \right). \]
Expand the product \( \left( X - E \left( X \right) \right)\left( Y - E \left( Y \right) \right) \) and use linearity of expectation.
Uncorrelated Random Variables
Random variables \( X \) and \( Y \) with finite second moments are uncorrelated if their covariance is zero, equivalently if \[ E \left( XY \right) = E \left( X \right)E \left( Y \right), \] equivalently \( \operatorname{ Cov } \left( X,Y \right) = 0 \).
Independent Random Variables Factor Expectation
If \( X \) and \( Y \) are independent and the expectations exist, then \[ E \left( XY \right) = E \left( X \right)E \left( Y \right). \]
In the discrete case, \[ E \left( XY \right) = \sum _ { x,y } xy P \left( \left\{ X=x \right\} \cap \left\{ Y=y \right\} \right) = \sum _ { x,y } xy P \left( X=x \right)P \left( Y=y \right), \] which factors into the product of the two expectations. The continuous proof is the analogous factorization of the product density or product measure.
Correlation Coefficient
If \( \operatorname{ Var } \left( X \right) \gt 0 \) and \( \operatorname{ Var } \left( Y \right) \gt 0 \), the correlation coefficient of \( X \) and \( Y \) is the normalized covariance \[ \rho \left( X,Y \right) := \frac{\operatorname{ Cov } \left( X,Y \right)} {\sqrt{\operatorname{ Var } \left( X \right)\operatorname{ Var } \left( Y \right)}} . \]
Cauchy Schwarz Inequality for Random Variables
If \( X \) and \( Y \) have finite second moments, then \[ \left\lvert E \left( XY \right) \right\rvert ^ 2 \le E \left( X ^ 2 \right)E \left( Y ^ 2 \right). \]
If \( E \left( Y ^ 2 \right)=0 \), then \( Y=0 \) almost surely and the result is immediate. Otherwise the quadratic \( E \left( \left( X - tY \right) ^ 2 \right) \ge 0 \) has non-positive discriminant as a function of \( t \).
Correlation Coefficient is Bounded
If the correlation coefficient \( \rho \left( X,Y \right) \) is defined, then \[ \left\lvert \rho \left( X,Y \right) \right\rvert \le 1. \]
Apply Cauchy Schwarz to the centered variables \( X - E \left( X \right) \) and \( Y - E \left( Y \right) \).
Standard Deviation
Given a random variable \( X \), we define the standard deviation of that random variable using variance: \[ \sigma \left( X \right) := \sqrt{ \operatorname{ Var } \left( X \right) } \]

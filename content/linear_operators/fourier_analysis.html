Fourier Coefficient
Suppose that \( f \in L ^ 1 \left( [ -\pi,\pi ] \right) \) and \( n \in \mathbb{ Z } \). The \( n \)-th Fourier coefficient of \( f \) is \[ \widehat{ f } \left( n \right) := \frac{1}{2\pi} \int _ { -\pi } ^ \pi f \left( x \right) e ^ { - i n x } \, d x . \]
Fourier Series
Suppose that \( f \in L ^ 1 \left( [ -\pi,\pi ] \right) \). The Fourier series of \( f \) is the formal series \[ \sum _ { n \in \mathbb{ Z } } \widehat{ f } \left( n \right) e ^ { i n x } . \]
Dirichlet Kernel
For \( n \in \mathbb{ Z } ^ + \), the Dirichlet kernel is \[ D _ n \left( x \right) := \sum _ { k=-n } ^ n e ^ { i k x } . \]
Fejer Kernel
For \( n \in \mathbb{ Z } ^ + \), the Fejer kernel is \[ F _ n \left( x \right) := \frac{1}{n} \sum _ { k=0 } ^ { n-1 } D _ k \left( x \right). \]
Riemann-Lebesgue Lemma
Suppose that \( f \in L ^ 1 \left( [ -\pi,\pi ] \right) \). Then \[ \lim _ { \left\lvert n \right\rvert \to \infty } \widehat{ f } \left( n \right) = 0. \]
Fejer's Theorem
Suppose that \( f \) is continuous and \( 2\pi \)-periodic. Then the Cesaro means of the Fourier series of \( f \) converge uniformly to \( f \).
Fourier Series Converges in \( L ^ 2 \)
Suppose that \( f \in L ^ 2 \left( [ -\pi,\pi ] \right) \). Then the partial sums of the Fourier series of \( f \) converge to \( f \) in \( L ^ 2 \).
Plancherel Identity for Fourier Series
Suppose that \( f \in L ^ 2 \left( [ -\pi,\pi ] \right) \). Then \[ \frac{1}{2\pi} \int _ { -\pi } ^ \pi \left\lvert f \left( x \right) \right\rvert ^ 2 \, d x = \sum _ { n \in \mathbb{ Z } } \left\lvert \widehat{ f } \left( n \right) \right\rvert ^ 2 . \]

Principles

Openmath is a human-first mathematics knowledge base: concise, readable, easy to navigate, and open to contribution.

Community

As a catch-all for anyone who doesn't want to use discord, we have GitHub discussions, which you will have access to already if you have a GitHub account.

Contributing

Anyone can contribute to Openmath, to get started make a GitHub account and head to source code. The content folder contains the mathematical source files.
